夢と鎮魂に関する調查報告 ( 夢與鎮魂相關調查報告 ) 第二章

麻生優雨

文身行者和眠之巫女

　反覆夢見死去之人的身影，如此之事例並不稀奇。親近之人的死，在本人沒有意識到的時候，已在心靈上產生某些影響。更何況是很親近之人的話，又更甚之。在夢見他的身影的同時，悼念、理解、或是種安心，逐漸的接受訣別的事實。人們有這樣的情形，並不僅限於現今，而是在各個時代都存在的吧。古代史裡提到，在遙遠的以前，從各地發現悼念死者的遺跡看來，人和野獸之間的分別在於人擁有對死者的悼念之心。
　想忘記對死者的想念，但顯現在意識表層時，也有變成以夢的形式看見過去殘像的情形。
　正因為是與自身親近的問題，所以從以前開始就有許多以「與死者相關的夢」當作題材的各式各樣民間傳說。但在這之中有一個值得注意的傳說，彷彿就是「沉眠之家的夢」。

夢之巫女

　因對與夢有關的傳說很有興趣，而東北地方流傳著「夢之巫女」之傳說。

代替人們承受悲傷的巫女沉睡時
她所背負的業變成夢
她身邊的人們也會夢見相同的夢

　承受與背負悲傷之巫女的故事，但故事本質非常模糊，是寓言程度的故事。而且遺憾的是從這些文字上很難讀取一些具體性的東西。但在一部分的地區，衍生出和這故事很像的「文身行者」民間故事。

文身行者

　過著遁隱的生活，但為了承受來參拜的人們之悲傷，將悲傷以刺青
( = 文身 ) 刻在自己身體上，這樣的故事。

　僧侶為前來參拜的人們，在全身刻入刺青背負他們的悲傷，和那些傾訴悲傷的人們一樣，僧侶在夢裡見那些痛裡所注入的想念，可以說僧侶的夢是與人們共有的。「夢」「刺青」等當作想念的昇華之關鍵字，在稱為「文身娘」的民間故事中也可以看到，這故事是為了不忘記想念之人，每當對方死去時，便將這思念以刺青刻在身體上的悲哀女子物語。已失去「文身行者」的巫覡要素，被俗化淪為二流的故事。

( 便利貼：文身 = 入墨 ( 刺青 ) )
( 相片備註：江戶中期的民間故事研究書「文身的民間故事」裡描繪的「蛇與柊」的紋樣。)

　調查文獻時，發現一個活躍於江戶末期至明治時代，名叫柏木秋人的民俗學者。
已知他當時正在進行當時日本各地的民間故事、童話、上代歌謠等相關研究。殘留記錄顯示他為了一段時期裡內陸的一部分地區所流傳的「眠之巫女」搖籃曲之學術調查而赴東北地方各地實地訪查。

　因為該時代在關於古日本文化及土著的風俗習慣之研究相當盛行，換句話說像他這樣的人物也有很多。
　所謂民俗學，若光是坐在離對象很遠的土地，於書桌上的紙上談兵等是絕對無法拓展視野的。土著的文化根柢到底隱藏著什麼？這些東西必須腳踏上那塊土地，感覺那空氣，以身體去體驗食物和生活習慣，才能真正體會與發現吧。
　為此他計劃前往孕育「眠之巫女」這首歌謠，被山與雪封閉的民俗學未開之地，以數年為單位前往當地訪問。
　對他殘留的調查紀錄非常有興趣，從那神社的儀式和那夢之巫女，都能看出文身行者故事內容發端的濃厚色彩。
　根據柏木秋人殘留的紀錄顯示，他曾於北東北守護古老神社之久世一族的家中逗留，遇見這首歌的來源「眠之巫女」。有趣的是，這巫女所擔任的職責和前述的「文身行者」的故事酷似。

搖籃曲「眠之巫女」為何？

　柏木秋人研究的對象「眠之巫女」，是東北地方的一部分地區所流傳的搖籃曲，如其歌名所述，是以神事的巫女為題材。在其著作「歌謠與傳說」中，有各式各樣地區的搖籃曲，收集其傳承並檢證事實，但其中關於「眠之巫女」似乎特別被注意，關於這首歌的記述也很多。

　「眠之巫女」
　　ねいりなさよ　はたて
　　ねいりなさよ　はたて
　　なくこは　ふねこせ　ついのみち
　　いちわらきせて　おんめかし
　　ねいらせな　きかみはぎ

　　ねいりなさよ　はたて
　　ねいりなさよ　はたて
　　みこさん　あいまに　おきつかば
　　ししにき　うがって　いみいのぎ
　　くもん　ひらいて　やすからず

　「眠之巫女」既被稱為一般的「搖籃曲」，當然必有反覆出現的詞組「ねいりなさよ」即為「快睡吧」。作者自己也認為開頭的呼喚語句 ( はたて ) 是指小孩子，因此當初解釋為，這首歌是徹底「針對小孩子的搖籃曲」。但觀其內容，「哭泣的小孩由舟承載」「終之路」「四肢被木頭穿刺」「禁忌」等，不管哪一個，我想都是會讓幼小的孩子持續感受恐怖的詞語。因為是搖籃曲，裡面的印象等也當然是針對小孩，對不聽話的孩子「壞孩子會坐著船不知道被送到哪去」「做壞事的話會像巫女大人一樣遭受恐怖的遭遇，所以要當個好孩子」等等，屬於說恐嚇的話來達到目的的「恐嚇歌謠」一類。與世俗一般說的「鬼會來抓壞孩子」使其精神上產生恐懼的性質相同。

　但侍奉神明，可以說是神聖之身的巫女，為何會以她「遭受恐怖的事」來作描寫象徵，這點十分有趣。讀起來簡直像是巫女犯了什麼禁忌遭受責罰，或是成為什麼樣的犧牲等微妙的意義。然後引用巫女的事來告誡孩子似的。

( 便利貼：「はたて」=「あだて」不聽話的孩子？)

歌謠和傳說的相似

　但於「歌謠與傳說」第二卷，比之前調查敘述的「眠之巫女」，變成更接近原來歌謠的形式。
　作者本身說到，以下可漸漸看出古時代的「眠之巫女」原樣。

　「眠之巫女原典」
　　ねいりゃさよ　はたて
　　ねいりゃさよ　はたて
　　なくこは　かごぶね　ついのみち
　　いちわらきざんで　おんめかし
　　ねいりゃせな　さかみはぎ

　　ねいりゃさよ　はたて
　　ねいりゃさよ　はたて
　　みこさん　あわいに　おきつけば
　　しせいぎ　うがつて　いみいのぎ
　　くもん　ひらいて　やすからず

　此外，寫這文獻的當時，參考的圖片下有寫著以下的字樣。

　「切斷肌膚，柊，刻入。刺青木，穿刺。
　久遠的，鎮守。於狹間，沉眠」

( 圖片注釋：柏木秋人著「歌謠與傳說二」「巫女與沉眠」裡添上的繪圖。可看出巫女沉眠之異界的象徵。)

　柏木秋人所注意的是，一般唱「穿上華服」，但原本卻是以「刻上」來表現。於此他認為可看出巫女的犧牲之立場，說不定這歌謠的全體是指巫女身體上的苦痛與伴隨的行為。
　於傳承的繪圖裡，值得注意的是於「肌膚」「柊」「刻入」等表現。
　柊有植物的柊的意思，但還有一個是觸碰與刺痛，以及鋸齒狀的葉子有關。痛與疼 ( = 柊 ) 的比喻。即是於肌膚上刻入痛而沉眠。這記述的確和「文身行者」以及據說背負痛苦的久世之巫女的紀錄一致符合。另外為「穿刺」用的稱為「刺青木」的東西，這裡使用「刺青」一詞也應該加以注意。

　因這首歌變成巫女的模式，他前往調查，久世一族的守護神社‧久世之宮，被稱為「納痛的神社」。久世之宮巫女的任務和現在一般的巫女一樣，但不只要舉行侍奉神明的神事，或舉行祭典，還有更傷害性的一面。這並不是俗稱的靈媒師一般的「降靈」，而是擁有另一種稱為「寄宿」的性質。

　以「納痛」為目的的神社，接納「痛」的「寄宿」，就是她們的存在意義嗎？

　翻閱許多古書來看，果然久世的巫女背負的職責，是為那些背負自身無法解決之龐大不幸與悲傷而來神社參拜的人們，承受他們的悲傷。然後再將這些「痛」以刺青刻於自己的身體上，代替人們承受痛苦。

　「文身行者」和「久世的眠之巫女」的共通點
  • 承受人們的悲傷，帶來心的平安
  • 使用「刺青」當作感情的象徵
  • 刻入刺青的巫女 ( 行者 )，刻入的紋樣之痛會以「夢」的形式見到
　從這些來看，久世之宮的巫女能說就是「文身行者」的民間故事吧。
