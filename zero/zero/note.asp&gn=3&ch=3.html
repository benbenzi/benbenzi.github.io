唄ト伝説三 原本 ( 歌謠與傳說三 原本 )

柏木秋人
入手時間：十ノ刻、十二ノ刻
入手場所：眠りの家．二階．着物の間
備考：--
以稱為「久世之宮」的神社為中心，深山封閉的村落裡，仍持續舉行著當地口耳相傳的「儀式」，繼承此儀式的「久世家」存在著。
持續蒐集的期間，我在久世的村落，以客人的身分允許進入久世家。

以下是在其處實際蒐集的事情。

這個社果然執行著和「刺青」有關的神事，「眠之巫女」是該儀式的歌曲。
這個歌曲在此處稱為「鎮魂歌」，舉行刻入刺青的儀式時，為鎮住巫女所受的「柊」而唱的歌。

「柊」，和樹木名稱相同，以「疼」的語源來看應有「痛」的意味。
雖然想觀看實際的儀式，但因舉行儀式的「久世之宮」禁止男性進入，
所以被拒絕了。

這個「鎮魂歌」傳至下界的「眠之巫女」裡，隱藏著一節。
但傳述歌詞意思的人已全部不在，久世家亦沒有清楚的意思的傳承。

因此以我的假設來讀解這篇歌詞。

鎮メ唄　第三節
「眠之巫女 第三節」
　ゆきなさよ　はたて　　　　　　　　( 向涯去吧 )
　ゆきなさよ　はたて　　　　　　　　( 向涯去吧 )
　ゆきぶね　ゆらして　はたて　　　　( 雪舟 搖阿搖 向涯 )
　このきし　ひらいて　はたて　　　　( 此岸 開啟了 向涯 )
　ろうろう　みわたり　かのきしに　　( 隨浪 渡過 彼之岸 )
　しせい　わたして　なくが　てあげ　( 刺青渡過 哭泣著 道別 )

從開頭部份與之前的「ねいりゃさよ ( 快睡吧 ) 」變化為「ゆきなさよ ( 去吧 ) 」來看，便能了解歌謠的內容的展開。

「はたて」，以前的發表內容是以這個地方的方言「不聽話的孩子」「あだて」為解釋，但於屋子中的文獻調查，其解釋是錯的。此世之最後，意即意味著彼世的境界有稱為「涯 ( はたて ) 」的場所。
刻上「いちわら ( 華服 ) 」的巫女，「去吧」和送往「涯」。

在這個地方在下雪時乘坐的架籠，像船一樣稱為「雪舟」，「ゆきぶね ( 雪舟 ) 」也有逝去的意味。

「このきしひらいて ( 此岸 開啟了 ) 」與「彼之岸」，換句話說，相對於彼岸的此岸，有此世的事情。該處的「門」開了，以「身渡」送往彼之岸。

「しせいわたして　なくがてあげ」之意，「しせい」從第一節的「刻」來看，應為「刺青」之意，渡越它，「なく」為哭泣，「てあげ」是這個地方，揮手、有最後的道別之意的話語。

這最後一節，是將刻著「華服」的巫女送往彼之岸的歌曲吧。

是人身供品的意思，或是超渡背負著刺青逝去的巫女的歌，因不能確認實際的儀式，因此也不可考。
