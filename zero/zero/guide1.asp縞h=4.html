麻生優雨

最後的巫女「零華」

　最後的戒之儀被舉行，從久世一族．久世之宮的紀錄完全斷絕的今日約百年前，稱為「久世零華」的刺青巫女應該是最後一次。

　以下，是根據麻生博士殘留的&查紀錄與天倉螢的&查判明「向久世零華襲來的許多悲劇」，依序&溯整理。

久世零華的誕生

　關於零華被&入久世家的原委，並沒有詳細的資料殘留。但以成為巫女的條件之一，「沒有親屬的女性」這點來判斷，或許是因為某些事而失去家族吧。從江戶末期於北東北的各地發生大規模的雪崩的殘留紀錄看來，恐怕是這樣的自然災害而帶來不幸的事故也說不定吧。
　變成孤獨一人。這是久世之宮的人們所看中的，零華於該年冬天，以巫女身分被&入久世之宮，成為「久世零華」。沒有人可以依靠的年輕女性，要在寒冷的村中生存下去是非常困難的。除非發生非常特別的事情，否則像普通人一樣幸福的生活著，已經是無望的事，這也是當時的實情吧。也因此，成為久世的巫女意味著，在人生的最後通牒的同時，對本人而言，成為巫女而結束人生，除了如此放棄的&頭之外應該沒有了。

　入宮之時，和「眠之巫女」的歌相同，遵從自古以來的風&以架籠來運送的形式。根據柏木秋人的紀錄，於久世家，&入巫女的架籠，是在屋子的玄關之天花&上吊著被&管的。於紫魂之儀採取死之血時，將屍體吊著的事情，以及前&的吊牢相同，在久世家的風&裡，可以看出「吊」這樣的行為被賦予意義。
　與宮中的歷代巫女相同，每回參拜者的來訪，於刺魂之儀慢慢刻入刺青，在戒之儀之前的期間都在吊牢中渡過。另外，可確認這時的鎮女為「冰雨」「時雨」「雨音」「水面」四人。

乙月要的歸鄉與死

　在雪代零華成為巫女進入久世家之後，因奇妙的緣分，而發生了至今為止的巫女們所沒有發生過的事件。鎮女之一的雨音，是乙月要的同母異父妹妹。
　他原本不太記得故鄉的具體記憶。因他在年幼不懂事時，就被送出寄養的話，這也是合乎常理的。因此在「沉眠之家」的研究時，他對夢見的曾經出生的家－即是久世之宮－也僅只有說出「故鄉有相似的傳說」這樣程度的認識。似乎在研究之中模糊的省悟村裡傳言的刺青巫女說不定是零華，在証言錄音帶中也殘留這樣內容的話語。因持續夢見沉眠之家，與親&的零華的夢，對這件事感到疑惑，而前往拜訪她所在的久世之宮，因為某些原因，得知鎮女之一是他妹妹的事情吧。恐怕在他身上擁有什麼證明，因而&她知道哥哥的存在吧。

　久世家最後的當主．夜舟的日記裡，有證明他的行動的部份。要似乎是回到故鄉後，藉由雨音之手，踏入禁&的場所．刻宮。刻宮是刺魂之儀被舉行的場所，還隱藏著通往稱為棘獄－巫女沉眠之場所的道路，久世之宮的中樞位置。因此，這被稱為「&子」－被認為是禁&的直系男子進入此，犯了禁&，神官們的動搖不難想像。根據夜舟的記&，要與雨音都被「流放」。自他回鄉里後足跡完全的斷絕也是因為這原因吧。另外關於這件事在似乎是雨音的人物之日記裡也被記&著。雖引導「哥哥」進入巫女的所在，但因擔&而&在後面－這樣的內容。為&上被沉入狹間的零華的要，在久世之宮的最深處，的確見到了將沉眠的她。
　但在那之後，在戀人零華的眼前，要被殺害了。而發生了難以挽回的悲劇。

破戒與瓦解

( &利貼：「はたて」X 小孩　O「涯」吧 )

　因零華而被引起的「破戒」，久世之宮將變成什麼樣的命運呢？
　久世家代代僅由宮大工們之手進行&繕．改建，破戒這時也由代代守護久世家的鳴海一門之宮大工們再度動員，為控制狹間擴張而採行解決手段。即是「自奈落蔓延出的瘴氣無法出去外界」、建築稱為「狹間之宮」的建築物，將久世之宮完全的包覆住。此外再施加二重三重的包圍，宮因此建築而與外界完全的被遮斷，且久世家變成擁有非常複雜構造的巨大屋子。
　因鳴海一門原本就有將從事工程的宮大工們封口的意思，且兼有建立人柱的&慣，似乎稱為「&柱」的人身供品多數的被&用。最後的當主夜舟的前兩代前，也曾舉行大規模改建工程，那時應該也犧牲了相當數量的鳴海一門工匠。
　不得不說是諷刺的一點，番匠們認為自己的工作是「侍奉神尊敬的事務」，所以常常&著白衣裝束。根據夜舟的紀錄，本來可以免於成為人柱的技之繼&者的木工頭竟也將性命了結。我想這恐怕是鳴海家最後的當主，代代的當主之中擁有優秀技術的鳴海天涯，這樣的人物連生命都賭上的封印，如此之事實。破戒的可怕也可從中窺得。
　另外，根據從乙月要開始，於夢裡拜訪沉眠之家的人們的談話，被&廊包圍的中庭，串刺著為數頗多的稻草人形。這本身雖然徹底是夢的話題，但於古時代的民間&仰，屢屢都有這樣的風&&慣也是事實。在神的世界與人的世界之間的境界中豎立稻草人形的，取代活著的人成為守護神，的確屢屢可看到ヒトガ& ( 形代人形 ) 的供物被&用。而且在夢裡所看到的光景，就算實際在久世家被舉行也不是不可思議的事吧。這說不定也是因為零華之詛咒引起的破戒，人形與成為&柱的人們一起負擔守護屋子的責任。

( 照片注釋：成為守護神被祭祀的串人形。即&現在也可看見一部分的地區仍因祭事而樹立串人形的樣子。)

成為破戒誘因之事

　失去要的零華之絕望與悲傷而引起了破戒，換言之，如果零華不知道要的死的話，可以說成為怨&的柊之解放，如此悲慘的事件就不會發生了。
　左右這點的主因是零華親眼「目擊」要的死這件事。從因為連眼睛都刻上刺青&得痛被&回的文身行者故事來比喻與思考的話，巫女的「瞳」之存在極為重要，應該沉入狹間的她的眼，本來應該什麼都無法映入。就像火葬時死者的眼皮閉著一樣。
　本來是不可能背負自己的痛的－因為身邊的人死亡，對自己的人生斷&，已經不存在悲傷等&頭－但要的死並不是別人的事，而是她自身感到「痛」。這想&在她的身體，特別是身為巫女的魂之瞳上，刻入刺青。在瞳上刻入刺青，&如文身行者的傳說般暴走，引起了破戒－這樣的想法會不會看太透呢？

圍繞著乙月要的因果

　「沒有父親的女性」並非只有久世家的人例外。久世家和多數的原始集團相同，徹底的母系相傳的家系。因此由來，在久世家出生的男子都在很小的時候被殺的關&，是 ( 自己變成 ) 沒有男性存在的一族。另外還有一點，和久世家的女性生下孩子的男性，也在沒被世間發現的情況下消失了，因此而生的因&。
　關於要和巫女．零華的事至今為止已經說過很多次了，但在此想再重新記&一次。因他的母親是久世家的女兒久世鏡華，對最後的當主．久世夜舟而言是相當於孫子的人物。而他的父親，根據鏡華的手記之記&，好像是前往久世家&查的民&學者．柏木秋人。
　在沒有男子的久世家，為了血脈的存續而定期的&入客人，和久世一族的女性生下子嗣，柏木秋人似乎自己本身也對這樣的立場了然於&之後結婚。但即&有了血緣關&，在久世家男性沒有存在下去的方法。柏木秋人的消息以久世家的滯留中之手記為最後而斷絕了，認為他成為不吉的因&下的犧牲也是很自然的吧。
　雖然乙月要生於如此非常特殊的家系下，卻在不知道出生之家的情況下長大成人，在男子來說是極為稀少的例子。另外，父親在見到小孩的臉之前就命喪妻之一族之手，母親因為自古的慣例，將小孩託付給不知道在何處，也不知道是誰的人樣育，好不容易才守住了性命。背負如此背景而長大，收養的人也不能對要說出事實吧。
　但幸運地長大成人的他，最後也因為想確認戀人的平安與否而回到故鄉，恐怕與父親相同的，命喪於真正的祖母之手。
　久世家的女性之中－例如要的母親鏡華等－能對這樣的因&提出反對的意見，擁有些啟蒙思想的人物的話，或許就不會引起這樣的悲劇。又或者要不離開村子而留在零華身邊的話，無論是否失去了親屬，她自己連成為巫女的可能性都沒有。
　引起破戒的當然是因為失去青梅竹馬的要之零華的悲傷，但我想說不定裡面也包含了無法切斷久世連鎖悲劇的女性們之絕望。
