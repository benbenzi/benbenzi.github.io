## 有機ELモデル〜見える変化

——
: **さて、「見える変化」と「見えない変化」ですが、<br>まずは有機ELモデルの「見える変化」の方を詳しくご説明いただけないでしょうか。**

山下
: はい。まず新たに採用した有機ELディスプレイですが、<br>色が鮮やかに表現できることに加えて、<br>6.2インチだった画面が7インチまで大きくなりました。<br>画面サイズが大きくなったというのは<br>手に取ってもらうと、すぐに分かっていただけると思います。

![](/others/interviews/new/jp/switch/switch-oled/vol1/img/02-01.png){:.border.border-radius loading="lazy"}

山下
: ですが、実は今回のモデルは、<br>この一番の特長である有機ELディスプレイの採用ありきで<br>進めていた訳ではありませんでした。<br><br>有機ELディスプレイ自体は、<br>「ニンテンドー3DS」<sup>※7</sup>の時代にも存在していて、<br>その技術動向は継続的に調べていましたが、<br>なかなか条件が合わず、<br>任天堂のゲーム機に採用するには至らなかったのです。<br><br>でも、世の中で有機ELディスプレイを<br>搭載した製品は増えてきていますし、<br>技術も進んでいます。<br>今回改めて我々がゲーム機に必要とする条件の中で<br>何ができるかをパートナー企業さんと一緒に確認・検証したところ、<br>今回は採用できるということになりました。

2011年に発売した、裸眼による立体視映像を体験できる携帯型ゲーム機。
{:.note-sm .faded title="※7"}

——
: **画面サイズを大きくするというのは、最初から考えていたことですか？**

山下
: もともと、ディスプレイを大きくしたいという考えはありました。<br>ただ、今回ディスプレイを大きくしている分、<br>画面の縁がスリムになっているのですが、<br>本体サイズや製品の強度を維持したまま、<br>画面サイズを大きくするというのが、<br>今回の新モデルへの改良で、もっとも大きなチャレンジの一つでした。

塩田
: 今回の有機ELディスプレイは、<br>鮮やかな色を表現できる有機材料を採用しています。<br>また、黒をより深く表示できるようになったことで、<br>白と黒のコントラスト差が強く出せるようになり、<br>メリハリのある色表現が可能になりました。<br><br>ですが、この有機ELディスプレイに映像を表示するための<br>ICや配線などの周辺部品を、<br>スリムになった画面の縁に収めなければなりません。<br><br>今までの画面の縁の広さなら収まりますが、<br>今回は同じやり方では収まりませんでしたので、<br>ディスプレイメーカーさんと相談しながら、<br>最終的にICをフィルムに載せて、縁の部分で折り返して、<br>巻き込むような形で実現しています。 

山下
: それから、有機ELディスプレイになって、<br>発色が鮮やかになったのは良いことですが、<br>お客さまによっては、<br>「鮮やかすぎる」と感じられることがあるかもしれません。<br><br>そんな場合を考慮して、<br>画面の鮮やかさを少し抑えることができる<br>「標準」と呼ばれる色モードを選択できるようにしています。<br><br>対して、<br>鮮やかな発色のままでよい方は、<br>購入時から「あざやか」の色モードが選択されていますので、<br>そのままお楽しみいただければと思います。

——
: **なるほど、場合によっては好みで色味が選択できるようになっているんですね。<br>お話を聞いてみると、有機ELディスプレイに変更したから画面が鮮やかになりましたとか、<br>画面を大きくできましたとか、そんな単純な話ではないことがわかりました。**

山下
: 鮮やかさだけでなく、<br>ディスプレイ自体を薄くすることも大切でした。<br>これは「見える変化」のもう一つの部分になりますが、<br>スタンドのサイズを大きくするのに役立っているからです。<br><br>スタンドはテーブルの上に本体を置いて遊ぶという、<br>テーブルモードのためのものですが、<br>画面を見る角度によっては周りの光が反射することもありますので、<br>角度を自由に変えられるようにして、<br>ゲームプレイの際に見やすく、快適に感じていただけるようにしました。<br><br>これを本体のサイズを維持したまま実現するためには、<br>ディスプレイ自体の厚みを減らす必要があったんです。

——
: **これ、確かに大きくなったスタンドを収納する部分は、本体側が薄くなっているんですね。**

山下
: 薄くしましたし、フリーストップを実現するために、<br>大きなヒンジを搭載しているんですが、<br>これがちゃんと本体に収まるように、<br>その分のスペースも計算して空けています。<br><br>本体が横方向にだけ3ミリ伸びているのですが、<br>3ミリ伸ばしたらこのヒンジとスタンドがきれいに収まるということで、<br>決断して少しだけ幅を伸ばしました。<br>これは設計の中でギリギリまで頑張った部分です。

![](/others/interviews/new/jp/switch/switch-oled/vol1/img/02-02.png){:.border.border-radius loading="lazy"}

——
: **確かに、本体の厚みや高さは変わっていませんね。**

山下
: はい。<br>今回、本体幅は3ミリ伸びましたが、<br>厚みや高さは変わっていません。<br><br>それにも関わらず、<br>ディスプレイの表示部分は6.2インチから7インチに大きくなっています。<br><br>そんな中で大きくなったヒンジを本体内に2つも収める、<br>というのですから大変でした。<br><br>この設計を成立させるためには、<br>薄い有機ELディスプレイを採用するだけでなく、<br>本体内部の基板やディスプレイ、充電池などを、<br>ぎゅっと凝縮した形にレイアウトしなおす必要があります。<br><br>なので、どういう形状にすれば、<br>ヒンジとスタンドの両方を収容できるか、<br>ここにもかなりの試行錯誤がありました。

塩田
: Nintendo Switchから変わらない本体の厚さの中で、<br>ヒンジとディスプレイで厚みを取り合い、<br>みたいになっていましたね（笑）。

——
: **逆にそこで本体の厚みを変更しよう、とはならなかったのですか？**

山下
: いや、従来のNintendo Switchのドックにも<br>有機ELモデルの本体が収まるという互換性を考えると、<br>厚みは簡単には変えられません。<br><br>それに、本体を厚くすると、<br>Joy-Conを装着した時に、<br>本体とJoy-Conの間に段差ができてしまいます。<br>それでは収まりが良くありませんから、<br>本体を厚くするという選択はなかったんです。<br><br>ちなみにスタンドの素材を、<br>今回樹脂から金属に変更したんですけど、<br>これは樹脂より金属の方が強度を保ちつつ薄くできる<br>ということで採用しました。<br>これもホームページではご紹介していませんが、<br>触っていただいたら気づいていただける部分です。

塩田
: 金属のスタンドにすることで薄くはなりましたが、<br>面積が広がった分、<br>薄い有機ELディスプレイを採用しても足りない部分はあって、<br>その分はディスプレイのスクリーンカバーの厚みを変更したり、<br>スタンド担当とディスプレイ担当が相談したりしながら、<br>お互いにギリギリまで頑張れるところを<br>何度も何度も調整していましたね。

![](/others/interviews/new/jp/switch/switch-oled/vol1/img/02-03.png){:.border.border-radius loading="lazy"}

——
: **ところで、スタンドが金属になった分、重量は増すのでしょうか。**

塩田
: スタンドを自由な角度で支えるヒンジだけでも<br>重量としては大きいのですが、<br>ほぼ同じ体積のところを、樹脂から金属に置き換えると<br>かなり重くなるんですよ。<br><br>でも、本体全体としては、プラス約20gの重さで収まっています。<br>これは逆に言うと、別の場所で削っているところがあるからなんです。<br><br>これまでのNintendo Switchも<br>当然頑張っていろいろなところを削って、<br>少しでも軽くしようと思って作っていましたから、<br>なかなかこれ以上に削れるようなところが<br>簡単にパッと出てこなかったんですが、<br>今回メカのエンジニアを中心に、<br>強度を保ちつつ削る設計のアイデアをいろいろ出してくれたおかげで、<br>うまく実現できました。

——
: **いろんな苦心の末に行き着いたスタンドですね。**

山下
: ちなみに、スタンドを開くときの感触も、<br>どのくらいの硬さにするか、相当調整をしました。<br><br>スタンドを開いて置いている時の安定性を考えると当然硬めがいいんですが、<br>一方であまり硬くしすぎると、お客さまが開ける時に・・・。

——
: **開けにくいですよね。**

山下
: はい、開けにくかったり、<br>「こんなに硬くて大丈夫？」と不安になられるかもしれません。<br><br>そこで、テーブルモードで遊ぶのに必要な程度の硬さは実現しつつ、<br>「開ける」「閉める」という体験の心地よさを考えて、<br>できる範囲で柔らかくすべきなんじゃないか、と思いました。<br><br>そのちょうどよい硬さを探るため、<br>限られた構造の中でさまざまな硬さのサンプルをいくつも作り、<br>検証して決めたんです。

——
: **そんなところも、細かく検証して設計していくんですね。**

山下
: 設計に関してはスタンド担当だったり、<br>ディスプレイ担当だったり、<br>いろんな担当者がいるんですけど、<br>それぞれが独立して各部品の改良方針を<br>バラバラに考えていたわけではなくて、<br>「本体のサイズや重さを極力キープしつつも、<br>Nintendo Switchの良さを引き出したい」<br>という共通目標に向かってお互い協力して、<br>本体内のレイアウトを試行錯誤していました。<br><br>その結果、「画面のサイズアップ」、<br>「有機ELディスプレイの採用」、<br>そして「スタンドの強化とサイズアップ」という要素が<br>パズルのように上手く組み合わさって、<br>今回の「有機ELモデル」の原型が見えてきたという感じです。

——
: **なるほど。それぞれの部品の組み合わせが、<br>「有機ELモデル」という全体を作っているということがよく分かりました。<br>ちなみに、有機ELディスプレイと、スタンド以外に「見える変化」はありますか。**

山下
: 有機ELディスプレイの搭載とスタンドの改良で、<br>携帯モードとテーブルモードの体験は向上しますが、<br>TVにつないだ時のTVモードの体験も良くできないかと考えまして、<br>今回のドックには有線LAN端子を搭載しました。<br><br>最近、TVやレコーダーなども無線LANが搭載されるようになっていますし、<br>家の中の無線環境が混み合っているケースもあると思います。<br><br>そういった方は、有線LAN端子を使うことで、<br>家庭内の混雑した無線環境の影響を受けずに、<br>快適にオンラインプレイができるようになります。<br><br>細かい改良ではありますが、<br>TVモードの体験向上になっているかなと思います。
