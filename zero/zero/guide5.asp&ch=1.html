遊戲設定之零系列用語錄 - 字首為 あ

紅い蝶 ( あかいちょう )

出自 零～紅い蝶～
零～紅蝶中象徵性的存在。紅贅祭時姐姐絞殺妹妹的脖子的時候，妹妹脖子留下的紅色類似蝴蝶的形狀的記號。皆神村的人，從以前開始就相信紅蝶乃是紅贅祭中做為祭品的雙子巫女 ( 御子 ) 的魂魄經過變化之後的姿態。
( 註：絞殺 = 所謂的掐死 儀式的祭品 = 雙子中的妹妹或弟弟 )

紅贅祭 ( あかにえさい )

出自 零～紅い蝶～
每經過數十年就會在皆神村舉行一次的祕祭。為了抑制從存在於皆神村地底下的虛所吹出的闇而舉行。使用過楔之後再以雙子巫女 ( 御子 ) 為祭品的儀式。祭典之前，由於被黑澤八重逃離村莊的關係導致一個人被留下的紗重被當作祭品，儀式失敗，發生了大償。化作怨靈的紗重將村人全部滅殺，從那時開始，皆神村就成了『從地圖消失上的村落』這個傳說。

麻生邦彥 ( あそうくにひこ )

出自 零～ZERO～及零～紅い蝶～
具有描繪出靈體能力的『射影機』以及放入靈石就可以聽見人的聲音的『靈石收音機』的開發者。由於進行靈界相關的研究，而被學會視為異端者。與民俗學者真壁清次郎是朋友，並借給了他試作型的射影機和靈石收音機。在那之後，麻生為了完成射影機而外出旅行，最後死因成謎。

天倉靜 ( あまくらしず )

出自 零～紅い蝶～
澪與繭的母親。帶著兩人一起造訪即將因為水壩建造而沉沒在水裡的故鄉。靜在當時，就聽說了在接近故鄉的地方有著『從地圖上消失的村落』這件事，對兩姊妹發出了十分嚴格的叮嚀「如果玩的太晚到晚上，就會迷路在村中喔。」

天倉繭 ( あまくらまゆ )

出自 零～紅い蝶～
澪的姐姐。『零～紅い蝶～』的主角之一，被雙子巫女中的紗重附體憑依的關係，而被引導到了皆神村。擔當再一次進行紅贅祭的雙子巫女的角色，最後自己情願地接受了這樣的命運。繭，最後完成了被澪所殺的這個願望。雖說是紗重附體的關係所造成。另一方面在繭的心中是否潛在地有著想被澪所殺的願望？這點希望在遊戲中確認。重要的關鍵在於，Ending 的祭之中，從四面八方聽見的，繭 ( 紗重 ) 的聲音。

天倉澪 ( あまくらみお )

出自 零～紅い蝶～
繭的妹妹。『零～紅い蝶～』這一作的主角之一，為了拯救被雙子巫女的紗重所附體憑依的姐姐繭而在皆神村中奔走。由於小時候讓繭受傷的事情，而過著不斷持續的自責的日子。當繭迷失在皆神村的時候，澪在心中發誓絕對不會再度犯下同樣的錯誤。利用入手的『射影機』來擊退眾多的怨靈。但是自己對於被紗重的姐姐‧八重所附體憑依的事情，沒有什麼自覺。

天倉操 ( あまくらみさお )

出自 零～紅い蝶～
澪與繭的父親。在皆神村附近出生的人。操是天倉家的入贅者，舊姓是『麻生』，與『射影機』的發明者‧麻生邦彥有著血緣關係。

忌人 ( いみびと )

出自 零～紅い蝶～
到虛之洞穴執行紅贅祭儀式的人們。忌人的眼睛是看不見的。在村子中，對於看見虛的裡面這件是非常嚴格的禁止，代代擔任忌人的家族中的人，眼睛被縫上使他什麼都看不見，而可以在最靠近虛的地方執行儀式。另外也會將打破村子禁忌去看虛裡面的人縫上眼睛，令他成為忌人的一份子。

虛 ( うつろ )

出自 零～紅い蝶～
存在於皆神村地下深道最深處的地方的巨大洞穴。做為村子之中最為神聖的場所的關係，只有擔任祭主的黑澤家當主以及神官，還有祭人，跟一部分的村人可以接近。同時對於虛有著各式各樣的限制，虛的事情不可以說，虛不可以看，在所有的書中都是被禁止記載的。
( 註：所以不管是書或口語提到的時候，村人都以 X 來代替。)

大償 ( おおつぐない )

出自 零～紅い蝶～
因為紅贅祭的失敗而無法阻止虛的鳴動，虛中噴出了闇 ( 恐怖 )，使村子被封閉在永遠的黑暗之中的這件事就是所謂的大償。被吞噬入虛之闇中的人們瘋狂而死，無法成佛，只能在這永遠的黑暗中繼續徬徨徘徊。虛每次的鳴動都讓闇持續沸騰，經過數次之後覆蓋了整個皆神村。這股闇徐徐的擴散到四周。

緒方浩二 ( おがたこうじ )

出自 零～ZERO～
高峰執筆的雜誌的責任編輯，與高峰同行前去取材，在冰室宅被霧繪詛咒殺死，成了襲擊深紅的怨靈。

鬼遊び

出自 零～ZERO～
冰室地方傳承的小孩子的遊戲。玩的時候以かごめかごめ來決定誰是鬼，然後讓鬼去追其他的小孩子的遊戲。第一個被捉到的人就是下次的鬼，而到最後留下來的人就成為姬。
( 註：かごめ之曲決定鬼的方法不明，姬，這裡應該指的是繩之巫女 )
