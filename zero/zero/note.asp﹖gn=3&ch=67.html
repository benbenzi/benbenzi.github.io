廃墟に関する調查 ( 關於廢屋的調查 )

入手時間：三ノ刻
入手場所：黒澤邸．家中
備考：--
關於日前取材的廢屋。

以下是從編輯部處聽到有關日前取材的廢屋的資料。
取材之前沒有時間調查，請作為照片的選定資料。

一、屋子的歷史
據說那間屋子的周圍至明治時期左右有幾個村落，但現在只剩下廢屋，當地的人似乎也不知道那間屋子是何時建立，為何建立。

現在入口已經不見，據說本來往裡走的深處有著奇怪形狀的巨大屋子。

古代的地圖記載著似乎有神社的存在，現在也已看不到了，詳細情形也不清楚。

但從以前就有傳說這座山裡與常世 ( 那個世界 ) 之間有聯繫。

二、幽靈之屋的傳言
從以前那棟廢屋就被稱為「幽靈之屋」，也有很多人因為這個傳言而來訪此地。

據傳言說「這棟屋子可以遇見死去的人」，
之後更傳出「會被遇見的死者引誘到那個世界去」。

追朔傳言的開端，也有推測該神社以前可能舉行過降靈儀式的說法。
不過這些傳言全都和山裡所傳的「和那個世界有聯繫」有關。
