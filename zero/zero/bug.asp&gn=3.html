刺青之聲遊戲 BUG 程式漏洞

  1. 十ノ刻在眠りの家一階物置廊下的門使用撫子の鍵後，如果不進入井戸の部屋而是到其他房間裏面，這樣的話井戸の部屋就變得不能進入。而這個 BUG 雖然不會導致無法通關 ( 僅是鏡石及照片取得不能 )。據報告在本作的其他地方也可能出現同樣情形，並導致無法通關。雖然還沒有確認，但是在前作也存在同樣的 BUG 並造成無法通關，這次有這個 BUG 的可能性也很高。為了不讓 BUG 出現，請盡量避免使用鎖匙開門後不立即進門。

  2. 現實中在起居室和深紅對話後，不改變房間就出現「入夜了」這樣的信息出現以後，隨後深紅便突然在眼前消失了。在這種情況下面，明明是大白天，但是也可以「睡覺」，這就與正常狀況不一致了。特別是一ノ刻以外 ( 尤其是七和八ノ刻 ) 發生這種情況而導致不能游戲下去的報告也是有好幾件的。為避免這樣得悲劇，請好好利用五個記錄點。

  3. 九ノ刻開始時主人公變成天倉螢 ( 本來是黑澤怜 )，就這樣一直移動到中庭のある迴廊，就被完全困在那裡了，再現條件不明。預防以及回覆的對策是八ノ刻終了前後，還是推薦存檔為好。

  4. 攝影的時候不能將怨靈打飛 ( 跟 一樣的震退 )。簡單的說就是沒有裝備的狀態下面攝影。發生的原因不清楚，但是報告的例子比較多的是二ノ刻開始後馬上出現的瀧川吉乃以及任務 4-2 的刻ミ女。預防的對策就是保存重來。
