### 3.ジャンケンのようなシステム

桜井
: 『エムブレム』というゲームを実際にやってみると、<br>どっぷりハマるような魅力があると思うんですけど、<br>人がやってるのを後ろから覗いたり、<br>何枚かのゲーム画面を見ただけだと、<br>「難しそうだな」って感じる方が少なくないと思うんです。<br>難しく感じる理由には2つあって、<br>ひとつは、多くのキャラクターが画面にひしめくような<br>戦略シミュレーションとしての見た目の難しさ。<br>もうひとつは、倒されてしまった仲間は二度と復活しないという、<br>シビアなシステム。

成広
: 見た目の難しさで言うと、<br>グラフィックでできるだけわかりやすくするようにしていて、<br>たとえばアーマーナイトは、どう見ても硬そうですし、<br>そこで敵からダメージを受けにくいということが<br>ひと目でわかるようなデザインにしています。<br>防御力が高いので、敵にぶつかりやすい前線に配置して、<br>自軍の盾として使えばいいということも<br>遊んでいるうちにわかってくるようにはしています。<br>でもやっぱり、マップの上に敵と味方がたくさんいて、<br>情報量も多いので、難しいと感じてしまうところはあるんでしょうね。	

![](/others/interviews/jp/nds/yfej/vol1/img_int/image05.jpg){:.border.border-radius loading="lazy"}

桜井
: 確かに情報量が多く見えるんですけど、<br>実際にやってることは、ジャンケンの<br>グーとチョキとパーがたくさんあるようなゲームだと思うんです。<br>遊びの大前提として、チョキはパーにぶつけて、<br>グーにはぶつけちゃいけないという<br>わかりやすい鉄則がありますよね。<br>そこを理解することが『エムブレム』の<br>第一のハードルを超えることになるんでしょうね。

成広
: そう。まさしくジャンケンですね。<br>剣は斧に強く、槍に弱くて、<br>槍は剣に強く、斧に弱くて<br>斧は槍に強く、剣に弱いという<br>3すくみの関係が基本ですけど、<br>これはまさにグー・チョキ・パーなんですね。

桜井
: 剣や斧にはめっぽう強いアーマーナイトが<br>魔法攻撃にはからきし弱いとか、<br>

『エムブレム』特有の関係を知って攻略につなげることが<br>このゲームの楽しさだと思います。<br>弓矢が飛行物に対して強いというのも、少し威力が上がる程度じゃなくて、<br>ダメージにして何倍も効くというようなことがありますね。<br>攻撃するときは気持ちがいいんですけど、<br>やられるときは、すごくおっかないので、<br>そういったところも刺激になって<br>お客さんを惹きつけているのではないかと。

成広
: ただ、ファミコン版のころは<br>そのシステムが歪んでいるところがあって、<br>斧はすごく不利な武器だったりしたんです。

桜井
: 確かに理不尽に感じることもありましたね。

成広
: 最近の『エムブレム』は、ジャンケンの関係が<br>よりわかりやすくなったとは思います。<br>ただ、武器の相性はジャンケンのように単純ではないんです。<br>相性を分かっていれば必ず勝てるというんじゃなくて、<br>何が起こるかわからないというちょっとした緊張感もあったほうがいいので、<br>ランダムな要素を取り入れています。<br>だから、Aという武器は必ずしもBという武器に<br>有効ではないんです。

桜井
: 中には問答無用の組み合わせもあったりしますよね。<br>ついさっきも、防御力がかなり高いドーガ<sup>（※8）</sup>が<br>魔法使いにやられましたし。<br>しょうがないですよね、体ばっかり鍛えてると（笑）。

 ドーガ＝ゲーム序盤から仲間に加わるキャラクター。兵種は「アーマーナイト」で、守備力が高い。
{:.note-sm .faded title="※8"}

成広
: （笑）。いくら鍛えて体を硬くしても<br>魔法でものすごく軟らかくされてしまうんですね。<br>それに、このゲームでは“事故”が多いと言いますか、<br>わかっていても、ぶつけてはいけない敵に動かしてしまったり。<br>しかもボタンを押す、まさにその瞬間に気づいたりして。<br>ああああ・・・置いちゃった・・・って（笑）。

桜井
: あれって不思議ですね。<br>特に今回、敵の移動範囲を表示することが出来るので<br>すごく遊びやすくなっているのに。

![](/others/interviews/jp/nds/yfej/vol1/img_int/image06.jpg){:.border.border-radius loading="lazy"}
	

成広
: そのあと敵の攻撃を受けるときはもう、祈りまくりで。

桜井
: 私も祈っていたら、敵の攻撃がはずれたことがありました（笑）。

成広
: そんなときは、よくぞ生き延びてくれたって<br>ものすごい愛が生まれるんです（笑）。

桜井
: 運よく生き延びてくれればいいんですけど、<br>敵の攻撃が命中して倒れちゃうと・・・。

	

成広
: 二度と戻ってこない。

桜井
: そこが『エムブレム』独特の、<br>すごくシビアなシステムですね。<br>初心者の方にもわかるように簡単に説明していただけますか。

成広
: このゲームではキャラクターのことを<br>「ユニット」と呼んでいます。<br>味方のユニットは、いくら大切に育てていても、<br>敵との戦闘に敗れてしまったら、二度と戻ってきません。<br>まあ、戦場が舞台ですので、当たり前と言えば当たり前ですよね。

桜井
: つまり、失った仲間はエンディングまで行っても<br>復活することはないと。<br>いやー、シビアですね。

成広
: いつ死んでしまうかわからない世界ですので、<br>彼らは一生懸命に生きてるんです。<br>そういった生き様は、短いセリフからも<br>感じていただけると思います。

桜井
: 時にはとんでもない過ちを犯して、<br>大切な仲間を失っちゃったりすることもありますよね。

成広
: はい、1人のユニットも失わずにクリアすることは<br>「パーフェクトプレイ」と呼ばれているのですが、<br>それにこだわるあまり、<br>仲間のユニットが倒されたらリセットボタンを押して、<br>章のはじめからやり直すという遊び方も珍しくありませんでしたよね。

桜井
: それって『エムブレム』を好んでやる人には<br>普通じゃないんですか？（笑）

成広
: 普通じゃありませんよ（笑）。<br>失うことの美学のようなことも<br>このゲームを通じて感じ取ってほしいんです。<br>仲間になるユニットは50人くらいいるんですけど、<br>愛情をもって育てられるのは、<br>せいぜい15人くらいでしょう。<br>もし、不幸にも仲間を失ったとしても、<br>また新しい出会いがあるのが『エムブレム』なんですね。<br>敵だと思っていても、話しかけてみると<br>寝返って仲間になることもありますし。

![](/others/interviews/jp/nds/yfej/vol1/img_int/image07.jpg){:.border.border-radius loading="lazy"}

桜井
: 成広さんとしては、仲間がポコポコ倒されてもいいから、<br>どんどん物語を先に進めてほしいと。

成広
: いや、リセットボタン、僕も、押すときは押します（笑）。<br>この間も、ちょっと油断していたら<br>シーダがポロッとやられちゃったんです。

桜井
: シーダってヒロインですよね。<br>しかも、何人かの敵を説得して仲間にしてくれるので<br>とても重要なユニットなんじゃないですか。

成広
: 今回は、シーダがいなくなっても<br>別のユニットが代わりを務めてくれるので<br>必ずしも絶対に必要なキャラクターではないのですが、<br>やっぱりシーダが倒れたときは、<br>しゃあないなあと思いながら・・・<br>（リセットボタンを押すポーズをして）ポチッと。

桜井
: （笑）。で、今作ではセーブポイントが増えて、<br>戦闘の途中でもセーブできるようになりましたね。

成広
: そこは、初心者の方にも遊びやすくなったポイントです。<br>やっぱりどうしても大切なユニットを失うことがありますので、<br>マップの上にセーブポイントを配置して、<br>もし不幸なことが起こっても、章の初めからではなく、<br>そこから再開できるようにしました。<br>また、セーブポイントには道案内の意味合いもあるんです。<br>とくに初めての人は、セーブポイントのルートを通ると、<br>ちょっと闘いやすくなったり、<br>迷わずにゲームを進めることができます。<br>一方、慣れた人は、セーブポイントを使わずに<br>自分のルールで遊んでもらったらいいと思います。<br>それぞれの遊び方ができるのが『エムブレム』ですから。

桜井
: あと、新しい要素としては、「兵種変更システム」。<br>この大胆な発想にはビックリしました。

成広
: これも、「難しそうだな」と感じる人への<br>回答のひとつです。<br>たとえばナイトを魔道士に変えたりして、<br>自分の部隊のバランスを整えることができるんです。<br>4章以降は、進撃準備として<br>戦闘に入る前に兵種を変更できるようになっています。

桜井
: たとえばアーマーナイトを失うと、<br>盾役がいなくなって苦戦してしまう。<br>そこで、残ったユニットの誰かを<br>盾役に変えることができると。

成広
: だから、大切なユニットを失っても<br>兵種を変えることで、戦闘しやすくなったと思います。<br>それにこのゲームはいろんな遊び方をする人がいて・・・。<br>たとえば女の子だけで部隊を編成する人もいるんです。<br>主人公のマルス以外は、すべて女の子（笑）。<br>そんな部隊でも、兵種が変更できると、<br>戦いやすくなるだろうということなんですね。

桜井
: 逆におじさんのユニットだけを集めて、<br>高齢者部隊で戦う人もいるかも。<br>ちなみにわたしは、さっそく僧侶リフを<br>「剣士」に変えて遊びました（笑）。

成広
: リフを！(笑)

桜井
: 剣士の姿だけどちゃんと頭を丸めているのが、<br>凝ってますねー。

成広
: 一応、イメージが壊れない範囲でしか、<br>兵種変更ができないようにしているんです。<br>でも、僧侶を剣士に変えられるのは、<br>ちょっとイメージが壊れちゃってますかね（笑）。

桜井
: でも剣士にしたのはいいんですけど、<br>力が「3」しかなくって、<br>全然使えないということが発覚したのは・・・<br>敵にぶつけてみたあとでした。

成広
: （笑）
