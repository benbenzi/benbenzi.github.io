英語の勉強真っ最中の中学・高校生のお子さんや、これから英語をやってみようかなと思っている方が身近にいらっしゃるなら、このソフトをプレゼントに選んでみては。

英語のトレーニング、それも普段の英会話に役立つフレーズを中心に学べる実践的なソフトです。ポイントは聞こえてきた例文をタッチペンで書いていく学習方法。聴いて書いて答えを見て覚える、五感を刺激するこの流れがとても効果的なんです。最初は和訳を表示させて（ヒントになります）、慣れてきたら聞こえてくる英文だけを頼りにチャレンジ！　英語を聞き分ける耳がさらに鍛えられます。トレーニングのレベルが上がると英語にちなんだ楽しいミニゲームが追加されていくなど、勉強という固いイメージではなく楽しく学べる作りになっています。

最大４人まで個人データを登録できるので、お子さんが学校へ出かけている昼間はお母さんがトレーニングするような使い方もできます。1回のトレーニングは10分ぐらいで終わるため毎日続けるのも苦になりません。家族みんなで競い合いながらトレーニングすれば、英語がより上達するかもしれませんよ。
