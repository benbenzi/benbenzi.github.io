零 ZERO 角色介紹！本網頁含劇情透露
繩裂儀式失敗後十五年…
宗方良藏為了方便研究冰室家的秘密，於是搬入了冰室邸，結果夫妻二人死在冰室家中。宗方美琴被深紅的曾祖父收養入雛咲家，後來結婚生下女兒雛咲深雪，接著深雪和雛咲真人結為連理，深雪生下兒子雛咲真冬與女兒雛咲深紅。

雛咲深紅

ひなさき みく Miku HinasakiCV: わくさわりか
本遊戲的主角。十七歲，繼承雛咲家族之血緣擁有強烈的靈感力，可以看得到別人所看不到的東西。為人深沉內向，雙親過世後僅剩哥哥相依為命。真冬為了查明失蹤的恩師高峰準星而下落不明，失去依靠的她，孤身進入冰室邸尋找哥哥下落。( 深紅在美版是十九歲 )

雛咲真冬

ひなさき まふゆ Mafuyu HinasakiCV: 金丸淳一
二十一歲，深紅的哥哥。深紅能力的唯一理解者。父母雙亡後，只餘下深紅一個親人。母親深雪把感靈能力遺傳給真冬與深紅兄妹，與深紅一樣擁有強大的靈感。大學畢業後在出版社工作時認識高峰，為了追查作家高峰等人的行蹤而前往冰室邸。

高峰準星

たかみね じゅんせい Junsei Takamine
四十二歲著名推理作家，除推理外也擅長傳記文學作品。與真冬在取材後一直交往，在看到民俗學者宗方良蔵的筆記後對冰室家舉行的儀式很感興趣，為第二回作品的取材，帶同助手等三人探訪冰家邸。

平坂巴

ひらさか ともえ Tomoe Hirasaka
二十八歲，高峰準星的助手兼秘書，自己也有進行寫作。靈感力強，跟隨高峰學習後以成為作家為目標，有預感今次的取材將會不順利。與高峰準星、緒方浩二共同探訪冰家邸取材時行蹤不明。

緒方浩二

おがた こうじ Koji Ogata
三十二歲，在高峰準星執筆後成為編輯。沒有靈感力，兼任恐怖雜誌的取材工作與高峰同行。與高峰準星、平坂巴共同探訪冰家邸取材時行蹤不明。失蹤後緒方給編輯部的報告成為最後的聯絡。

氷室霧絵

ひむろ きりえ Kirie Himuro
為冰室家代代相傳裂繩儀式的最後一位繩之巫女。自幼被挑選後一直在禁止和外界接觸下成長。因愛上來訪的異鄉客而對現世產生依戀導致儀式失敗。被瘴氣沐浴後化成怨靈，把所有進入冰室邸的人殺盡。

宗方良藏

むなかた りょうぞう Ryozo Munakuta
民俗學者。在『紅蝶』中幫助黑澤八重逃出皆神村。後來對冰室家舉行的儀式很感興趣。為方便研究冰室家黃泉之門的秘密，於是一家三口搬入冰室邸，結果妻子八重自殺，而宗方亦在黃泉之門前身亡。

黑澤 ( 宗方 ) 八重

くろさわ ( むなかた ) やえ Yae ( Munakuta ) Kurosawa
幼時體弱多病，靈感力強。逃出皆神村後和宗方結為連理並移居冰室邸養病，藉着女兒拾到的射影機可以看到平常看不到的東西。後來產生陰陽眼而精神錯亂，最後在庭中的櫻花樹上吊自殺。

宗方 ( 雛咲 ) 美琴

むなかた ( ひなさき ) みこと Mikoto ( Hinasaki ) Munakuta
宗方良藏和八重的女兒。七歲時捲入冰室家的咀咒之中，遊玩時收到少女霧繪的射影機後和其他小孩一同神隱，接着父母雙亡。美琴受到射影機的保護下獲救，失去記憶的她被父親的朋友收養。

雛咲真人

ひなさき まさと Masato Hinasaki
傾心研究的考古學者，主角們的父親。在深雪學生時期和她結婚，深雪曾擔任真人的助手，擔任遺跡和土器攝影工作。真人因遺跡調查，一年之中都在全國到處旅行，甚少回家。後來於研究中事故去世。

雛咲深雪

ひなさき みゆき Miyuki Hinasaki
主角們的母親。年幼時靈感力強，不過小學畢業後幾乎全失。對兒女遺傳到靈感力而痛苦不已。得心病後以照相為興趣，後來靈力逐漸復甦而拍到奇妙照片。被自己的感靈能力折磨，最後上吊在家裡樹上。

氷室家当主

ひむろ マスター Master Himuro
冰室家最後的當主。家訓甚嚴的男人，知道霧繪心意而把她想念的人殺死，導致裂繩儀式失敗，間接引起禍刻。冰室家當主是劍術宗師，禍刻後受到瘴氣的影響後發狂把家臣全部慘殺然後在大広間自殺。

氷室一族

ひむろ いちぞく Ichizoku Himuro
傳統的日本神道虔誠信仰者。從古代開始前就統治這個地方，而且定期舉行祭神的神道儀式。對黃泉之門畏懼且崇敬，以身為黃泉之門的守人的身分守護著黃泉之門，背負着封印黃泉之門的責任。

霧絵想念的人

きりえのおもいびと Kirie no Omoihito
冰室家的留宿的客人。對繩之巫女和背負堪坷命運的霧繪感到悲哀。由霧繪的日記中可以看出他和霧繪相戀，被視為儀式的邪魔而被冰室家當主下令殺死，被神官殺死後棄屍於逢魔淵。

麻生邦彥

あそう くにひこ Kunihiko Aso
江戶至明治初期的科學家，發明可以拍攝異界的射影機。由於進行靈界相關的研究，被學會視為異端。以響石耳飾發明靈石收音機，後來把試作型射影機和靈石收音機借給朋友真壁清次郎到皆神村拍攝紅贅祭。曾到朧月館作客，發現面具工匠所用的塗料能進一步改良射影機。接著持續在各地調查，最後在冰室邸研究將靈體完全封印的方法時身亡。
