遊戲前注意事項

故事前傳

為提高趣味性，開始遊戲之前可以先閱讀官網的番外篇「另一個故事」。
雖然沒有明說，但依照劇情和角色，這個基本上是 AR 故事「紫之日記」的序章。

遊戲之前

請注意以下幾點。
  • 請將 AR 筆記本放在明亮且平坦的地方。
  • AR 筆記本如果弄髒或折到的話，會變得難以辨認，請注意。
  • 如果光線直接投射在 AR 筆記本而反射的話，會變得難以辨認。請在光線不會反射的地方使用。
  • 這個軟體需要移動 3DS 本體來遊玩。遊戲之前，請確認周圍是否有足夠的空間，並且用兩手拿著本體來玩。因為有可能造成損傷、故障或是破壞周圍的物品，請不要有太激烈的動作或操作。
  • 這個軟體的音效，只對應立體環繞音效。如果在本體設定中設定成單聲道或立體聲道，在遊玩本遊戲時會自動變成立體環繞音效。

陀螺儀感測器的調整

一開始的時候會調整陀螺儀感測器。請將 3DS 本體放在桌上或平坦的地方，並稍待一段時間。調整結束時請按 A 鈕。
※也可以在オプション ( 選項 ) 裡的「ジャイロセンサーの調整 ( 陀螺儀感測器的調整 )」中作調整。

有關記錄的保存

在遊戲進行當中會自動的保存 ( 自動記錄 )。記錄中，在上畫面的右下角會表示「Now Saving」的文字。
在「心霊写真撮影 ( 心靈照片攝影 )」時 ( → 11 )，拍攝的照片可以保存在 SD 卡中。在拍攝之前必須先將 SD 卡插入 3DS 本體之中。
※SD 卡中未使用的容量不足的話，會無法保存照片。

關於記錄的消去

遊玩『AR ストーリー「零～紫の日記～」( AR 故事「零～紫的日記～」)』時 ( → 7 )，選擇「はじめから ( 從頭開始 )」而開始遊戲的時候，會將「つづきから ( 繼續遊戲 )」中已保存的記錄刪除，這點請注意。另外，保存的照片資料，可以在 HOME 選單中的「ニンテンドー 3DS カメラ ( 任天堂 3DS 照相機 )」中刪除。

有關保存資料的初期化

在 HOME 選單選擇『心霊カメラ～憑いてる手帳～ ( 心靈照相機～附身的筆記本～ )』圖案後，同時按下 A + B + X + Y 鈕一段時間，可以將保存資料初期化。
※消去的記錄無法回復成原本狀態，這點請務必注意。
