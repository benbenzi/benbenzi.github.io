零紅蝶惡搞【 第一刻 】消失在村子中的地圖

KiryuuT. tina8968 ( 巴哈姆特 ) 著
我追著姊姊，馬上就知道她狂奔的原因了。
原來她看上一個掉在路上的黑色小包包。
反正先搶先贏，一個死掰咖跑不了多快，我一下子就跑在姊姊前面撿起包包。
姊姊在我撿起包包時受到了不小的打擊，一直抱著我的腿……口水又快流到我身上了……

我打開包包查看有沒有千元大鈔，但是都沒找到，只發現兩張新聞剪報。
兩張新聞剪報都是用剪刀剪下來的，而且剪剪報的這個人看起來好像很不會用剪刀，邊邊看起來根本就是狗咬的。
而且剪剪報的這個人一定很花痴，剪報之餘還在慎村真澄的名字旁畫了不少大愛心。
我繼續翻著包包，發現了一張沒什麼看頭的情侶照。
姿勢那麼多種不擺，兩人呆呆地站在那裡發呆。
姊姊終於從被我奪去包包的傷痛之中站了起來。
看她口水又流了一身我實在是不想跟她走在一起。
沿著小徑走我隱隱約約地看到一個女的。
就說嘛！這麼大一個村落怎麼可能半個人都沒有？

等我追上去時，那個女的已經不見了。
姊姊把手放在我的肩膀上。
我摸摸她的手，幸好手上沒沾到什麼奇怪的東西。
雖然不知道她把手放在我肩膀上的用意。
可是要是姊姊的手摸過噁心的東西再把手放在我的肩膀上的話，我一定會拉住那隻手使出過肩摔。
手抽離我的肩膀之時，姊姊也同時從我身旁走過去。
我突然領悟到一件事……
原來……

姊姊的動作是如此快啊！果然她運動神經很發達！！
反正世界上不可思議的事情太多了，就算發生在一個智障身上也沒什麼好奇怪。
姊姊推開旁邊屋子的門，她說要借廁所，畢竟我也有點尿意，於是我也一起跟進去了。

進門之後，透過木板可以看到另一個房間。
我看到之前看到的那個女的在隔壁房間看著我跟姊姊傻笑。
之後……她就消失了！
這是多麼不可思議啊！！
原來在電腦、電動遊戲之中才有的技能「瞬間移動」在現實也是看得到的。
等等我一定要把那個女的抓住，叫她敎我怎麼使用「瞬間移動」。

再進一扇門，我看到的是圍爐間。
我看到一本很礙眼的紅色手帳掉在地板上就順便撿起來。
而這時姊姊的表情突然皺在一起，她的手在抓住我的那瞬間我閃開了。
她的手不知道有多噁心啊！誰敢讓她摸……
當我還以為姊姊羊癲瘋要發作之時，她的表情又變回平時的痴呆樣。
姊姊跟我說她剛剛看到一個男的殺了一個女的。
我懶得理她，事到如今我也不想知道她的腦袋裡裝的到底是愛玉還是豆花。
反正撿到的東西不看白不看，於是我翻開手帳。
女性的手帳 ( 1 )
關於從村子中消失的地圖，我也找不到。
從前，我就很了解真澄的個性。
就算是筆心掉在地上斷成好多節，真澄也一定會把所有斷掉的筆心給找出來。
經過村子入口處的鳥居，我就已經開始幫你找地圖了。
我在村子重複持續地找尋。
恐怖女人的笑聲在夜晚的村子裡迴響著。
她一直笑我：白痴！找不到的啦！！

隨後，姊姊又交給我兩本跟我手上那本差不多的筆記，她說是在附近撿到的。
反正她就是喜歡亂撿東西。
女性的手帳 ( 2 )
時常，會感覺到窗戶外有人經過， 手裡拿著火把，邊低聲罵著髒話邊行走。
似乎在尋找著什麼東西，但不關我的事。
遠處，常常傳來很難聽的歌聲。
這村子據說是「祭典之日消失」，我看是「歌唱之日消失」吧。
而地圖說不定也跟著消失了。

女性的手帳 ( 4 )
無論是誰都好，看到這筆記的人，請快來幫我找地圖，我無法找到地圖，拜託。
須堂美也子。

看完這三本手帳，我終於知道是怎麼回事了。
原來是有人在找他們「掉落的地圖」，所以到處留下手帳要別人幫他一起找。
所以說這是一場尋寶遊戲了，到時後幫他們找到地圖不知道有沒有獎金可以拿……
不過他們幹麻為了找一張地圖大費周章？再去買一張不就好了？
我怕姊姊不瞭解手帳的真正涵義，我把我的目的告訴姊姊，要她一起找地圖。
沒想到姊姊從口袋掏出一張世界地圖，她說是剛剛在玄關的門上撕下來的。
她什麼時候撕的我都不知道，擅闖民宅還亂撕別人的東西。
不過這對白痴的情侶把地圖貼在門上還不知道，這就是所謂的習慣成自然嗎？
看久了就把地圖當成門的一部份了。

我在圍爐間旁邊的通道盡頭看到剛剛那個女的。
她一定就是手帳中找地圖的傢伙。
我趕緊追上去，可是那女的又不見了。
我推開眼前的那道門，我想那女的應該在裡面吧……
可是我進去之後沒看到那女的。

果然，那個女的會瞬移 ( 瞬間移動的簡稱 )。

姊姊在進房間之後，馬上又找到一本女性手帳。
我發現，姊姊真的很像一條訓練有素的搜查犬。
女性的手帳 ( 3 )
真澄，我是美也子。
為了幫你找地圖而來的。
一起找吧。
以我們兩人之力也許可以找到。
如果你看到這本筆記，記得大聲呼喊。
人家就在這附近。
加油！

這女的真的很花痴，她又在真澄的名字旁邊畫滿大愛心。
反正快點把地圖交給那個女的吧。
走出去，又看見那女的蹤影，真是神出鬼沒。
唉……我也好想學瞬移喔……

順著那個女的走過的路又發現了一扇門，進去好像是佛間之類的吧！？
我在佛間的桌上發現了一個好玩的東西，就是相機。
而且樣式還很古老，我高興地拿起來藏在懷裡，這相機可以說是古董級的。
在房間搜刮了一陣，沒什麼值錢的東西，我想也該走了。
這時姊姊又找到了一本手帳，我覺得姊姊以後真的可以去當搜查犬之類的。
女性的手帳 ( 5 )
時常，從佛壇附近，會聽見令人作嘔的歌聲。
似乎是從地下傳來的聲音。
反正，還是不關我的事。
小地圖～快出來吧～～～

看到最後一行時，我有想揍那女的衝動，不過還是算了……
當我打開門之時，我看到一張綠色的大臉。
我趕緊「碰」的一聲甩上門。

老實說我被嚇到了，不過我也知道那女的一直瞬移的原因了。
原來她晚上正在敷「冰河泥」時，剛好我跟姊姊來了。
由於她不想在我跟姊姊面前丟臉所以就一直逃。
不過那女的品味還真差，冰河泥塗得整張臉都是濃濃的綠色。
現在不是流行什麼 SK2 嗎？幹麻不用 SK2 呢？
這女的品味真怪。

當我要打開門時我發現門打不開了。
那個死女人，在我面前顏面盡失，為了滅口把我關在裡面，還附帶姊姊當陪葬品。
我很生氣地拿出相機對著門拍照，試圖讓她聽到相機的聲音。
讓她知道，到時候惹我生氣的話，我就去拍她敷冰河泥的臉貼在網路上。
果然，那女的很識相，我再開一次門，門開了。

我生氣地走回圍爐間，姊姊看到我很生氣，也乖乖地跟著我，沒有亂滴口水。
死女人！我不幫妳找地圖了！
我打開玄關的門要走，可是門又被鎖上了。
那女的真的很機車，算了，反正地圖趕快交給她就走了。
突然，那女的也瞬移到圍爐間，我正想叫住她，她居然穿過圍爐間旁的另一扇門。
這女的不但會瞬移，還會穿牆。
說實在，我現在反而有點羨慕她了。
我打開她穿過的那扇門，一打開門就有東西掉在我頭上……好痛……
我撿起那東西，是一隻鑰匙。
鑰匙上面好像有刻花紋，反正不管上面刻的是豬還是狗，那女的一定是故意用鑰匙丟我的。
我走出房間又看到那個女的，她站在樓梯上面，難怪我會被她丟中。
不過那種角度能丟到也滿厲害的。

緊接著，我上了二樓，那女的又在穿牆了。
她一定是在故意刺激我的。
我忍住想揍人的衝動，用鑰匙開了門，命令姊姊把房間重要的東西通通找出來。
兩三下，姊姊就拿著兩本手帳給我。
女性的手帳 ( 6 )
在這搜尋多久了？
地圖永遠找不到。
已經懶得再找……越來越煩了。
我不想找了。
可是真澄會罵人家啦……

女性的手帳 ( 7 )
累了，想放鬆就睡著了，即使找不到真澄的地圖。
在夢裡，村子裡仍是被黑暗包圍著。
參考書、漫畫、倒在地上字典。
站在書堆上的女性、發狂似的笑聲。
拿著地圖罵我白痴的少女……
那個笑聲一直在耳際迴蕩。
再也不想找了。

看到最後，我可憐起那女的來了。
真可憐，要幫男朋友找的東西居然在第三者手上。
而且還被第三者歧視，也難怪她不想找了，真是辛苦她了。

我跟姊姊又在房間裡走了一會兒，發現地上又有個手帳。
我罵了姊姊一聲「笨」，就走過去撿起手帳看。
女性的手帳 ( 8 )
找到真澄的便條了，真澄果然有第三者了。
馬上可以重逢了，但我已經心碎了。
在這邊等他吧，真澄一定會回來告訴我他把第三者甩了。
找到消失的地圖離開村子吧……
到時候我要去夜市掃街，把他的荷包吃光光。
如果他回來了……

緊接著的是空白，但不只是空白，還有浸水的痕跡。
我翻了兩頁……終於知道為什麼會浸到水了……

手帳上最後一行寫著：討厭～又流口水流得整本書都是了～～
我意識到那花痴根本是姊姊的同類，口水老是會失禁，難怪她男朋友會有第三者。

死花痴……花痴！？
我突然想到，其實我到現在還沒有看過真正的花痴。
老實說，我對花痴的定義也不太清楚。
我又想到……
有一次我問老媽說：媽媽，花癡是什麼意思？
老媽說：這個嘛……等等妳上網去「 Yahoo 奇摩」在它的搜尋列打上「花痴」兩個字就可以查到了。
雖然我有照著老媽的話去做，不過我還是不了解花癡的定義。
突然我感覺到我的右肩溼溼的，回頭一看……
我又嚇了一跳，那個死女人臉上的冰河泥換成藍色的了，而且還流口水在我肩上。
我的左肩剛剛才沾過姊姊的口水，現在那女人居然連我的右肩也不放過。
我馬上閃開，結果又踩到一沱溼溼的。
原來姊姊一直找不到廁所尿失禁了，不過我終於想起來我跟姊姊來這最原本的目的。
我們是來借廁所了。

算了，大人不計小人過。
我把地圖丟給那女的趕快要賞金，免得到時候我也要尿出來了……
正當我考慮好要多少數字時，那女的化做一陣光消失了。

「……」
「澪，妳看她成佛了。」

真搞不懂姊姊在說什麼，她明明就是瞬移逃走了……
我把怒氣發洩在姊姊掰咖的腳上，踹了又踹。
最後姊姊受不了我的摧殘逃走了。
我很喪氣地跌坐在板上，我這麼努力地幫那女的找地圖到底是為了什麼？
我思索著這個問題。
