友達が集まってパーティをする時、はたまた忘年会で盛り上げたいイベントが欲しい時、あるいは雑学の知識が欲しい時。
シンプルだからこそ、遊び方を工夫すればどんな演出も可能。
子供さんから大人まで、もしなにを贈ればいいか迷ったり、持って行くものに不安があったら、絶対に外さないこの１本がオススメ。

【食べ物】・【国語】・【スポーツ】・【歴史】・【乗り物】・【音楽】・【生き物】・【地理】・【雑学】・【マンガアニメ】・【ゲーム】11ジャンルから出題される、8000問以上のクイズ。
ゲーム内ルールだけにとどまらず、独自ルールで演出すれば、どこまでも遊び方が広がります。
マイク入力に対応しているので、声を出しているうちに自然と興奮度はアップ、白熱した時間が過ごせます。
人数が増えたらチーム戦に挑めば友情も深まるし、もちろん１人でも充分楽しめる『ひとりでクイズ』もあり、クイズ番組に出演している気分が味わえます。
まずは１人モードで流れを練習して、本番でみんなをビックリさせたりなんてのもOK？

人が集まるパーティにオススメなのは『クイズDEビンゴ』。
賞品やごちそうを用意して、早くビンゴした人からもらえる…なんてやると、みんな目の色が変わって真剣になっちゃうかも。
『伝説のクイズ王』を目指せば、いっそう楽しい気分が味わえます！
