今回のシリーズにはどんな新展開があるのでしょうか。
サトシは、初心に帰り一から出直すために、タケシ、カスミとも別れ、最初のポケモン・ピカチュウだけを連れて、新たなる舞台、ホウエン地方におもむきます。今まで見たことのないポケモンたちがサトシの前に現われます。そして、暗躍する謎の軍団、アクア団とマグマ団…古代ポケモン…ダブルバトル…と様々な新設定も登場します。

今回はどんなポケモンと一緒に旅をするのでしょう。
サトシは、ピカチユウだけを連れて旅立ったので、新しいポケモンをゲットしながら旅を続けていきます。今のところ、サトシはキモリ、ハルカはアチャモをそれぞれ手に入れます。そのあとは見てのお楽しみということで……。

どんなメンバーで旅をしていくのでしょうか。
一人で旅立ったサトシに、まずハルカ、そしてハルカの弟のマサトと、順番に合流していきます。そして、あの男も……といわけで、４名のパーティーで旅することになります。
