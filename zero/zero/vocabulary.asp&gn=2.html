紅蝶關鍵詞解說

紅い蝶

零～紅蝶中象徵性的存在。紅贅祭時姐姐絞殺妹妹的脖子的時候，妹妹脖子留下的紅色類似蝴蝶的形狀的記號。皆神村的人，從以前開始就相信紅蝶乃是紅贅祭中做為祭品的雙子巫女 ( 御子 ) 的魂魄經過變化之後的姿態。
( 註：絞殺 = 所謂的掐死 儀式的祭品 = 雙子中的妹妹或弟弟 )

紅贅祭

每經過數十年就會在皆神村舉行一次的祕祭。為了抑制從存在於皆神村地底下的虛所吹出的闇而舉行。使用過楔之後再以雙子巫女 ( 御子 ) 為祭品的儀式。祭典之前，由於被黑澤八重逃離村莊的關係導致一個人被留下的紗重被當作祭品，儀式失敗，發生了大償。化作怨靈的紗重將村人全部滅殺，從那時開始，皆神村就成了『從地圖消失上的村落』這個傳說。

麻生邦彥

具有描繪出靈體能力的『射影機』以及放入靈石就可以聽見人的聲音的『靈石收音機』的開發者。由於進行靈界相關的研究，而被學會視為異端者。與民俗學者真壁清次郎是朋友，並借給了他試作型的射影機和靈石收音機。在那之後，麻生為了完成射影機而外出旅行，最後死因成謎。

天倉靜

澪與繭的母親。帶著兩人一起造訪即將因為水壩建造而沉沒在水裡的故鄉。靜在當時，就聽說了在接近故鄉的地方有著『從地圖上消失的村落』這件事，對兩姊妹發出了十分嚴格的叮嚀「如果玩的太晚到晚上，就會迷路在村中喔。」

天倉繭

澪的姐姐。『零～紅い蝶～』的主角之一，被雙子巫女中的紗重附體憑依的關係，而被引導到了皆神村。擔當再一次進行紅贅祭的雙子巫女的角色，最後自己情願地接受了這樣的命運。繭，最後完成了被澪所殺的這個願望。雖說是紗重附體的關係所造成。另一方面在繭的心中是否潛在地有著想被澪所殺的願望？這點希望在遊戲中確認。重要的關鍵在於，Ending 的祭之中，從四面八方聽見的，繭 ( 紗重 ) 的聲音。

天倉澪

繭的妹妹。『零～紅い蝶～』這一作的主角之一，為了拯救被雙子巫女的紗重所附體憑依的姐姐繭而在皆神村中奔走。由於小時候讓繭受傷的事情，而過著不斷持續的自責的日子。當繭迷失在皆神村的時候，澪在心中發誓絕對不會再度犯下同樣的錯誤。利用入手的『射影機』來擊退眾多的怨靈。但是自己對於被紗重的姐姐‧八重所附體憑依的事情，沒有什麼自覺。

天倉操

澪與繭的父親。在皆神村附近出生的人。操是天倉家的入贅者，舊姓是『麻生』，與『射影機』的發明者‧麻生邦彥有著血緣關係。

忌人

到虛之洞穴執行紅贅祭儀式的人們。忌人的眼睛是看不見的。在村子中，對於看見虛的裡面這件是非常嚴格的禁止，代代擔任忌人的家族中的人，眼睛被縫上使他什麼都看不見，而可以在最靠近虛的地方執行儀式。另外也會將打破村子禁忌去看虛裡面的人縫上眼睛，令他成為忌人的一份子。

虛

存在於皆神村地下深道最深處的地方的巨大洞穴。做為村子之中最為神聖的場所的關係，只有擔任祭主的黑澤家當主以及神官，還有祭人，跟一部分的村人可以接近。同時對於虛有著各式各樣的限制，虛的事情不可以說，虛不可以看，在所有的書中都是被禁止記載的。
( 註：所以不管是書或口語提到的時候，村人都以 X 來代替。)

大償

因為紅贅祭的失敗而無法阻止虛的鳴動，虛中噴出了闇 ( 恐怖 )，使村子被封閉在永遠的黑暗之中的這件事就是所謂的大償。被吞噬入虛之闇中的人們瘋狂而死，無法成佛，只能在這永遠的黑暗中繼續徬徨徘徊。虛每次的鳴動都讓闇持續沸騰，經過數次之後覆蓋了整個皆神村。這股闇徐徐的擴散到四周。

陰祭

在紅贅祭開始以前，當虛開始鳴動的時候所進行的儀式，可以暫時性的鎮壓住虛。陰祭需要將人作成人柱而成為『楔』以用來進行儀式的關係，會將拜訪村莊的客人 ( マレびと ) 用來做為祭品。承受的痛苦若是越多，作為人柱的效果也會越強，才能夠止住虛的鳴動。

鬼隻

在皆神村中，紅贅祭裡當雙子巫女 ( 御子 ) 的姐姐 ( 哥哥 ) 將妹妹 ( 弟弟 ) 殺死之後，留下來的姐姐 ( 哥哥 ) 就被稱之為『鬼隻』，為眾人所懼怕且尊敬。成為鬼隻者，精神異常的人非常多，這些人被隔離到村子裡的藏。鬼隻們死後，被埋葬在朽木裡。

桐生茜

在紗重與八重之前的雙子巫女中的姐姐，親手殺了妹妹薊而成為了鬼隻。儀式之後，對於殺死自己妹妹的事情非常的自責而在精神上呈現病態，父親為了使茜逐漸恢復而製作了與薊十分相似的人偶。但是寄宿在人偶的魂不知何時操縱了茜。死後的茜也依然繼續在人偶的支配之下。

桐生薊

做為過去的紅贅祭中的雙子巫女的妹妹，由姐姐茜所殺死。仰慕姐姐，並且相信如此做就可以合為一體的這點想法，與繭有著共通點。死後，由於恐懼父親所做的與自己相似的人偶帶來的災難，逐以靈體的形式出現來警告。

楔

紅贅祭開始進行之前，虛開始鳴動的時候，為能暫時鎮壓住那股鳴動而使用的祭品。能夠成為楔的，必須是來自村外的人。皆神村會將拜訪而來的客人監禁，並為了進行陰祭而切割其身體，進行『削身』。在客人之中，也存在著在成為楔之前無法忍耐住這些痛苦，無法成為楔並且死去的人。

朽木

傳說中是好幾個世代以前的紅贅祭的祭主，認為已死的鬼隻非常的可悲，為了鎮壓她們的靈魂而製作的祠堂。但是實際上，從前曾經有雙子為了逃出村子而使用了在暮羽神社底下的深道，被激怒的當時的祭主建造了這個祠堂，封閉村子裡對外相通的深道。朽木裡面，供奉著非常多的雙子地藏以及紅色的風車。

黑澤紗重

大償發生的當時的雙子巫女中的妹妹。由於姐姐八重成功的從皆神村逃走，一個人進行了紅贅祭失敗而引起了大償。想要拯救村子以及被姐姐背叛的怨念，抱著相反的兩個想法化為了怨靈。成為怨靈的紗重一個一個的將村人殺死，在血流成的海上狀似瘋狂地笑著。

黑澤八重

大償發生的當時的雙子巫女中的姐姐。與妹妹紗重一起試圖從皆神村逃走的，紗重卻在途中被村人們抓到，而只有八重成功的逃走了。途中為了搜尋不知去向的妹妹而重返皆神村，但回到村子所在地，這裡已經沒有皆神村的存在，已經是皆神村被大償消滅之後了。失意的八重後方，為了完成與樹月的約定而再度回來造訪村子的宗方良藏到來。在這之後兩人結婚，並移居冰室宅。良藏被黃泉之門所魅惑，而八重由於『射影機』的緣故精神不正常，最後在庭中的櫻花樹上吊自殺了。
( 註：皆神村在大償之後就被消滅了，這裡應該是指被暗闇所覆蓋，造成鬼打牆的現象，而使八重看不見皆神村了。)

黑澤良寬

大償發生之時的當任祭主，也是當年的雙子巫女‧八重與紗重的父親。良寬並不是沒有父親對於孩子的愛情，但身為祭主，『為了抑制虛而舉行的儀式不可不成功』的責任感更為強烈。另外，良寬自己也是過去的紅贅祭中殺死自己弟弟的鬼隻。

射影機

麻生博士所發明的，可以拍攝出『看不見的東西』的照相機。在主角為澪與繭的『零～紅い蝶～』登場的這台射影機是試作機，並沒有能夠完全將靈體給封印住的性能。民俗學者‧真壁清次郎為了拍攝紅贅祭，而帶到皆神村的射影機，在之後被澪所取得。主角為深紅的『零～ZERO～』中登場的『形見の射影機』是完成品，這一台才擁有能夠完全將靈體給封印住的能力。

須堂美也子

慎村真澄的戀人。慎村為了水壩的工程在皆神村的附近進行調查，行蹤不明。美也子為了找尋慎村，而迷路困在了皆神村中。在村子裡異樣而稀少的謎團氣氛之下，成功的與戀人再相會，並打算兩人一起脫出這個村子，結果最後卻被已經化為怨靈的慎村給絞殺了。

立花樹月

大償發生的前一年進行的紅贅祭的雙子御子中的哥哥。雖然進行了儀式，卻失敗了。第二年很快的決定了以八重和紗重為雙子巫女來進行紅贅祭，樹月覺得都是因為自己的關係才會讓八重她們被當成祭品，十分後悔，協助兩人從皆神村逃走並且寫信給朋友宗方良藏，之後追隨弟弟睦月的腳步，在槌原家的藏中自殺了。結果紗重一個人被捉回來，進行了紅贅祭而招致了大償的悲劇。

立花千歲

立花樹月及睦月的妹妹。紅贅祭之時，陷入認為由於八重與紗重從皆神村逃走的關係樹月才因此而死的想法，而怨恨著八重與紗重，並且將這股感情投影在澪身上襲擊了澪。這也跟澪被八重給附身憑依了有關係。有躲進壁櫥的癖好的千歲，在大償發生的那一天覺得十分恐怖而又躲進了壁櫥中，被闇所吞噬了。

深道

存在於皆神村地下非常寬廣且巨大的洞窟，在最深處的是，進行紅贅祭的場所。已知是聯繫著黑澤家、立花家、桐生家等等的連結的地下道。另外，暮羽神社底下也擁有村子裡唯一的通到外面的深道通路。

雙子地蔵

為了供養在紅贅祭中犧牲的雙子們的目的而製作的地藏。皆神村的各種地方都能看見，也有分辨村子的地界線之用。

雙子巫女 ( 御子 )

在紅贅祭中作為祭品的雙子。皆神村，有許多雙子誕生，那些雙子負擔了擔任雙子巫女 ( 御子 ) 的任務。小孩子是女性的場合被稱為巫女，男性的場合則稱之為御子。

皆神村的習慣上，將先出生者當作妹妹 ( 弟弟 )，後出生者當作姐姐 ( 哥哥 )，這點和現代是完全相反的。沿用皆神村的習慣來重新探討澪與繭的關係，澪是姐姐、繭是妹妹的狀況之下，結局的光景簡直是代表著重現儀式的意思。

真壁清次郎

研究日本各地的風俗習慣的民俗學者。為了解出黃泉之門的傳承的真相不惜賭上性命，因助手宗方良藏帶來的皆神村的情報而決定前往皆神村。

真壁與宗方一起拜訪了皆神村，在村子裡受到非常隆重的歡迎。得到了參觀祕祭的特別准許，而逗留在村中展開了各式各樣的調查。例如村子裡的遺跡和書籍，調查了有關祕祭的事情，逐漸到祕祭的全貌…。在完成那個目標之前，真壁就被監禁到黑澤家的座敷牢中了。自己了解了紅贅祭以及陰祭的事情，而領悟了自己將成為楔而被當作祭品。比起不想死的感情，能以這雙眼睛看見祕祭為名的歡喜更加的強烈，這樣的真壁，一點都不打算要從村子逃走。

慎村真澄

為了建設水壩，為了調查實際地點而被派遣來的調查員。調查中進了森林並迷路在皆神村中，為了找尋脫出的方法而在村子搜索著。與追查自己的戀人須堂美也子再會之後，卻在探索黑澤家的時候遇到楔而被殺。怨靈化的慎村往戀人所在的地點移動，並殺了美也子。

皆神村

以『從地圖上消失的村子』而在附近成為十分有名的傳聞。澪與繭的母親聽說了這個村子的事情，所以經常要兩人注意別玩到晚上才回來。

以雙子為巫女並尊崇之，名為紅贅祭的不可思議的關於祕祭的神秘性，以及拜訪村子的客人行蹤不明之類的事，在當時被近鄰的人們所畏懼著。但是由於這個村子的儀式，也被稱為是背負著把此地從闇之恐怖之下救出來的擁有被詛咒的宿命的村落。這個村子的傳說，是藉著在大償發生時稍前一點，由從村子裡出來的宗方良藏的手記的記載，在後世流傳下來。

宗方良藏

在『零～紅い蝶～』中，與民俗學者‧真壁清次郎一起拜訪了皆神村。很久以前是立花樹月小時候的朋友，之後被委託負責雙子巫女的八重與紗重從村子裡逃跑之後的事情。幫助一個人從村子裡逃走的八重之後，兩人結婚。

無法忘記皆神村的記憶的宗方最後惹上了黃泉之門，在『零～ZERO～』中移居到了冰室家居住。結局，妻子八重自殺，宗方在走到黃泉之門前面也死了。另外，兩人的女兒美琴，幼年時代從冰室家逃走，以失去記憶的狀態在山中被發現。美琴的女兒就是深雪，深雪的孩子，就是『零～ZERO～』中的主角深紅以及真冬。

靈石收音機

與射影機相同是麻生博士所發明的東西，可以聽見從靈界傳來的聲音的收音機。改良在迴路的一部分使用礦石並檢測波長的礦石收音機，可以聽見該人寄宿在靈石上的想法或是思念之類的聲音。
