零 ZERO 關鍵詞解說

麻生邦彥

具有描繪出靈體能力的『射影機』以及放入靈石就可以聽見人的聲音的『靈石收音機』的開發者。由於進行靈界相關的研究，而被學會視為異端者。與民俗學者真壁清次郎是朋友，並借給了他試作型的射影機和靈石收音機。在那之後，麻生為了完成射影機而外出旅行，最後死因成謎。

緒方浩二

高峰執筆的雜誌的責任編輯，與高峰同行前去取材，在冰室宅被霧繪詛咒殺死，成了襲擊深紅的怨靈。

鬼遊び

冰室地方傳承的小孩子的遊戲。玩的時候以かごめかごめ來決定誰是鬼，然後讓鬼去追其他的小孩子的遊戲。第一個被捉到的人就是下次的鬼，而到最後留下來的人就成為姬。
( 註：かごめ之曲決定鬼的方法不明，姬，這裡應該指的是繩之巫女 )

形見の射影機

雛咲深雪 ( 深紅的母親 ) 遺留給深紅的照相機，是一台可以拍攝到人所看不見的東西的照相機，被稱為『射影機』。這個照相機，若是交給擁有強大靈力的人來使用的話，能夠做到封印靈體。但是，隨著使用射影機拍攝靈體的次數的增加，也可能造成最壞的場合──被靈的世界給拉了進去，也存在著精神異常的可能性。深雪也曾經手持這個照相機，而在心中留下了心病。

霧繪

冰室家傳承的繩裂之儀式的最後一任巫女。由於有了思念的人，殘留著對現世的執著而沒有成功製作繩裂，御縛之儀失敗了。再者，由於真冬與她所思念的人十分相似，所以打算將真冬拉入闇之世界的這一邊。

黑澤八重

大償發生的當時的雙子巫女中的姐姐。與妹妹紗重一起試圖從皆神村逃走的，紗重卻在途中被村人們抓到，而只有八重成功的逃走了。途中為了搜尋不知去向的妹妹而重返皆神村，但回到村子所在地，這裡已經沒有皆神村的存在，已經是皆神村被大償消滅之後了。失意的八重後方，為了完成與樹月的約定而再度回來造訪村子的宗方良藏到來。在這之後兩人結婚，並移居冰室宅。良藏被黃泉之門所魅惑，而八重由於『射影機』的緣故精神不正常，最後在庭中的櫻花樹上吊自殺了。
( 註：皆神村在大償之後就被消滅了，這裡應該是指被暗闇所覆蓋，造成鬼打牆的現象，而使八重看不見皆神村了。)

御神鏡

在黃泉之門前面放置的映出門的樣子的鏡子，並從這裡將散發出來的瘴氣彈回去，可以做到使門的力量減弱的特殊的鏡子。對於封印黃泉之門這件事擔任著非常重要的角色，但是，由於禍刻的衝擊而使它碎裂成五片，這些碎片散落在冰室宅各地。如果不收集五片碎片並且將其放回鏡子原本的位置拼起來的話，就無法封印黃泉之門。

裂き縄

能夠封印黃泉之門的特別的繩子。由繩裂之儀式所製作。

裂き縄の儀式

為封印黃泉之門而必須進行的製作裂繩的儀式。使用被稱之為裂繩的特殊的繩子，將從一族中選出來的巫女的脖子以及四肢用裂繩縛上，然後使用特殊的裝置拉扯使巫女的身體裂開。集合這些被製作好的裂繩，才能進行御縛之儀式。封印黃泉之門的方法只有這個，冰室一族為了封印黃泉之門，連綿不斷的犧牲著一族中的人。

高峰準星

有名的神秘學作家。為了下次的著作的取材而造訪冰室宅，卻被霧繪詛咒並殺死。在深紅為了找尋真冬而來到冰室宅時，以怨靈之姿襲擊深紅。

繩の巫女

為了製作裂繩而被當作祭品的巫女。巫女是十數年一次，從冰室家的血脈裡七歲的女孩子之中選出來。然後這名巫女從此之後連一步都不許踏出冰室家，在座敷牢中被養育著。這是為了杜絕她對現世的執著的其中一個做法。隨著巫女的成長，以繩子在兩手兩腳縛上，迎接十七歲的時候再將繩子縛上她的脖子來進行儀式。

雛咲真冬

深紅的兄長。是深紅唯一能敞開心扉的存在。與在雜誌工作的編輯因公事而相識，為了追查作家高峰等人的行蹤而前往了冰室宅。

雛咲深紅

是『零～ZERO～』 中的主角，靈能力強，具有能夠感知不屬於人世間的東西的能力。為了找尋行蹤不明的兄長而造訪了冰室宅。

雛咲深雪

身為深紅的母親，並且是宗方良藏‧八重兩人的孫女。因為深紅的祖母‧美琴所持有的射影機的關係而得了精神病，最後上吊自殺了。

氷室一族

從古久之前就統治這個地方的一族。以身為黃泉之門的守人的身分守護著黃泉之門，背負著封印黃泉之門的責任。這個地方的人們，畏懼並崇敬黃泉之門。

平坂巴

高峰的助手兼秘書。擁有微微的靈感，感覺到瀰漫在冰室宅的異樣氣氛，前往取材之前感覺到了不安。與高峰同樣，被霧繪所殺死。

禍刻

御縛之儀式失敗而導致黃泉之門打開，瘴氣噴出的事情稱之為禍刻。最後的繩之巫女‧霧繪的時代，由於使用霧繪在還殘留著對現世的執著的狀態下製作的裂繩來進行儀式的關係引起了禍刻。沐浴在瘴氣之下的村人們紛紛得了瀕死的重症。而沐浴在瘴氣之下而失去正氣的冰室家當主，對沐浴在瘴氣中的人們擴大不可踏出屋子外面的禁則，將居住在屋子內的人全數殺死了。日後，這一天的慘劇就以當主瘋狂了而將全家都慘殺的事情流傳了下來。

御縛りの儀式

使用冰室一族犧牲了巫女而作成的裂繩，為了讓黃泉之門不被打開而進行封印的儀式。

目隠し鬼の儀式

選出繩之巫女的儀式。在一個屋子中集合冰室一族所有的七歲的少女，由臉被遮住的鬼 ( 戴上面具的關係眼睛被毀壞 ) 的少女來捉捕。最後一個逃過目隱鬼的少女 ( 越能從目隱鬼之下逃脫便擁有著越強的靈力 ) 就會被選為繩之巫女。另外在這個儀式中最初被抓到的少女，在下一次儀式的時候，擔當負責選出下一任巫女的目隱鬼的這個任務。

宗方良藏

在『零～紅い蝶～』中，與民俗學者‧真壁清次郎一起拜訪了皆神村。很久以前是立花樹月小時候的朋友，之後被委託負責雙子巫女的八重與紗重從村子裡逃跑之後的事情。幫助一個人從村子裡逃走的八重之後，兩人結婚。

無法忘記皆神村的記憶的宗方最後惹上了黃泉之門，在『零～ZERO～』中移居到了冰室家居住。結局，妻子八重自殺，宗方在走到黃泉之門前面也死了。另外，兩人的女兒美琴，幼年時代從冰室家逃走，以失去記憶的狀態在山中被發現。美琴的女兒就是深雪，深雪的孩子，就是『零～ZERO～』中的主角深紅以及真冬。

黃泉の門

隔絕現世與異界的門。存在於冰室宅的地下，以巨大的石頭製作而成。同時，什麼時候、誰做的、為了什麼而做，這一切的紀錄都沒有流傳下來。只知道，從古老的時代開始就存在於此地，成了現在這樣。冰室家中傳說著，若門完全打開的時候，生者與死者的世界將會相連，所有的死者都會復甦。
