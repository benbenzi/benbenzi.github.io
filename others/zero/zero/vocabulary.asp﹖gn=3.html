刺青之聲關鍵詞解說

逆剝

又稱逆身剝，正式的日文是逆剥或者逆剥ぎ。
其由來為天津罪中的其中之一。
主要是妨害農業或者祭祀的時候才會施加的刑罰。
典故：古代的時候，普通而言要剝皮的時候會從屁股開始，而逆剝就是反過來，從頭頂開始。因為剝時並不管祭品的死活，而剝出的樣子又有如蛇脫皮一般恐怖，就好像剝到中途應已死去的祭品又活了回來一樣，在後世被認定為天津罪中的一個而被人們所忌諱著。

天津罪

日本古代記載的兩大刑罰 ( 天津罪以及國津罪 ) 之一。
天津罪是畔放・溝埋・樋放・頻蒔・串刺・生剥・逆剥・屎戸。
國津罪則因為罪名眾多這裡不列出來。
古代有分為天津神以及國津神兩類神明，前者為高天原一方，
後者為地上世界一方 ( 八百萬神 )，詳細上還有一些典故這裡就不提及。
天津罪是用在懲罰妨礙農業以及祭祀，
而國津罪則是用在個人之間的不法行為、災害，以及詛咒。
故在刺青之聲中會提到天津罪以及逆剥，主要就是妨礙祭祀這一點。

はたて

這一項在遊戲中並沒有好好的解釋。
並非指不聽話的小孩子，而是指「端」。
再套合到遊戲的劇情，那麼這個詞所指的，就是「涯」這個地點了。

とこよのくに

漢字是「常世之國」，也就是通過黃泉比良坂 ( 泉津平坂 ) 之後會抵達的地點。
其中分為根之國以及黃泉之國，兩者就相當於極樂世界以及地獄的分別。
但是刺青中當地觀念只是單純的用以稱呼在涯彼岸的「異界」，故不能確定是往哪一國。
以刺青之聲的設定來說，以異界或者彼岸來稱呼即可。

奇御様

在文件中有提到，附近的村落裡同樣有齋串樣 ( イグシサマ ) 這種人偶的存在。
這兩者是相當的東西，也就是依代、形代一類的東西。
詳細的作用會在斎串様說明。

斎串様

首先要拆開來看，分為斎串以及斎串樣兩者。
斎串就是刺青之聲中，我們到處去收集的刺青木。
以細長的薄版 ( 寬大概兩公分 )，將兩端削尖，屬於祭具的一種。
是負責貫穿斎串樣的道具。
而斎串樣本身就是我們所看見的樹上的人形。
這類的通稱形代，一般是用於祓厄運、除病氣、解煞的儀式。
在形代上寫上自己或者親人的名字，經過虔誠的祈禱祝頌儀式之後，再將其燃燒放入河川放流或者燒成灰燼埋入土中，是為消災解厄的儀式的一種。

響石

同性質的有ホウム石以及木魂石。
據說能夠紀錄起人的話語重新播放出來。
其中，響石是傳說中可以紀錄人的思念的神奇石頭。
根據遊戲的資料，響石也是能使用靈石錄音機收訊的對象之一。
或許，零華就是藉著要送給她的響石耳飾，來傳遞自己思念的「聲音」吧。

優雨的身影

遊戲設定上是怜自己心中的影子。
這也就是所謂的「後ろめたさ」或者「負い目」，這兩者的主要構成原因，
是因為為了家族或者親族，甚至共同體而犧牲的人，對自己的心境造成的負擔。
