蝶‧源之章【 第伍章 】逢坂的回憶～秋收的紅贄祭

KiryuuT. tina8968 ( 巴哈姆特 ) 著
　　走進掛布廊下，前方是一條十字型的通道，絮惆毫不遲疑地選擇左邊的通道，只因為附近的房間有陣陣靈力傳出。

　　推開房門，是佛間。房中書籍散亂，幾本書零星地擺放在桌上，而幾本書又堆滿了旁邊的佛桌。

　　絮惆走向靈力發源處，推開散在佛桌上的書籍，就在佛桌的一角，是顆澄色的水晶。

　　素色的書本與深褐色的佛桌與澄水晶極不搭調，而水晶又散發出陣陣靈力，絮惆可以感覺到，水晶似乎與婦人之靈的感覺非常相似，但那散發出的靈力究竟是有害還是無害？

　　後顧之憂使絮惆不敢隨意撿起，而她所憂慮的就是那名婦人。

　　「妳又在想什麼？為什麼每次都這麼猶豫不決啊？」

　　「喂！妳等一等……」

　　「咦？」

　　「怎麼了？」

　　來不及阻止悠昌，悠昌就撿起那顆水晶，只見悠昌一臉驚慌的樣子，絮惆急忙伸手握住悠昌，而呈現在她面前的是黑白畫面。
　　「紅贄祭……是不是又要到了？」
　　「紅贄祭……是不是又要到了？」

　　雙子女孩不安地問著母親，而她思考了一下，便走向玄關。

　　「我去問問，況且不一定是輪到赤葉與秋伶，乖乖在家裡等媽媽。」

　　雙子的母親──逢坂亞子走向皆神村唯一一家神社，暮羽神社。

　　往暮羽神社的途中可以看見一片金黃色的稻田以及散落一地的楓紅樹葉，亞子停在稻田前凝望著這景象。

　　看似喜樂的秋收，其實不然，在好幾十年前的今天，也是同樣這樣的景象，親手殺死自己的妹妹，雙子的悲哀全藏在這片金黃色的秋稻中，完完全全地隱藏起來。

　　而如今又產下了一對雙子，悲劇又要在自己的女兒身上重演，叫亞子好不心痛。

　　「妳是……逢坂嗎？」

　　亞子望向身旁的人，那人對亞子微笑。

　　「妳是……暮羽神社的巫女大人？」

　　「嗯，妳是為這次的紅贄祭來的嗎？」

　　「巫女大人，到底是輪到我們還是輪到槌原？」

　　亞子不安地問著，畢竟皆神村只剩下逢坂與槌原兩對雙子，只有二分之一的機率。

　　「妳放心，這次是輪到槌原家的雙子。」

　　「是嗎？」

　　亞子鬆了一口氣，但暮羽神社的巫女又道：

　　「這年是槌原，明年就是逢坂了，請妳要有心理準備。這是遲早的事，妳應該知道吧？」

　　「嗯……」

　　「也對，妳當然知道，逢坂的鬼隻。」

　　巫女朝亞子行禮，便走緩步走回暮羽神社，亞子仍是靜靜地看著稻田，在「 X 」前的舊事全湧上心頭，淚水深埋在泥土中，就只剩下一年了。

　　望著夕陽下的稻田，仍是與往昔相同的稻田，而明年，也是吧？
　　畫面突然轉換到某個樓梯中，亞子臉上的表情充滿恨意，不斷地想衝下樓梯，無奈，礙於身後的兩名宮司拉住她。

　　「為什麼會是這樣！明明就不是輪到我們！為什麼！」

　　接著，畫面又順著樓梯下去，在充滿「 X 」的大洞前，赤葉與秋伶面對面，赤葉緩緩地伸出雙手，秋伶毫不抵抗。

　　「是媽媽的聲音……」

　　「嗯……」

　　雙子停下動作，望向樓梯那端，槌原當主連忙走到雙子身旁。

　　「等妳們進行完儀式，就可以見到媽媽了。」

　　雙子對望一眼，隨即看著槌原當主。

　　「槌原叔叔，你騙人……」
　　「槌原叔叔，你騙人……」

　　赤葉與秋伶異口同聲道，眼睛直直地盯著槌原當主瞧，槌原當主急忙閃避雙子的眼神道：

　　「怎麼會呢？」

　　「槌原叔叔，你騙人……」
　　「槌原叔叔，你騙人……」

　　「趕緊讓儀式開始吧……」

　　還是同一句話，槌原當主心底涼意頓起，急忙走回原位發號施令。

　　其實，這次本該是輪到槌原的雙子，但槌原當主實在是不想與他的女兒分離，他無法忍受這種痛苦，等儀式一結束，槌原當主就打算帶著女兒遠走高飛，一切都在他的意料之中。

　　宮司們開始敲打錫杖，雖然看不見樓梯中的亞子，但亞子嘶聲力竭地喊叫著，叫聲同樣充滿整個洞穴。

　　被叫聲吸引的雙子停下動作，然後又繼續開始，隨著叫聲而緊張地望向樓梯口。

　　「不能再這樣下去，不然儀式會失敗，動手吧……」

　　槌原當主與身旁的宮司竊語，宮司走上樓梯，過不了多久，亞子驚呼一聲，聲音逐漸由大轉小，然後漸漸地消失，那邊結束了。

　　聽到媽媽驚呼的雙子女孩本來是不安的，但聽到驚呼聲消失後卻突然變得鎮定，他們已經知道發生什麼事了。

　　停下所有的動作，兩人面向槌原當主，雙手緊緊牽著，臉上的表情充滿決心。

　　「槌原叔叔……你都在騙人……」
　　「槌原叔叔……你都在騙人……」

　　「等等你們做什麼！！」

　　無視槌原當主大喊，曾經即將化為村中保護力量的兩人面向充滿「 X 」的大洞。決心已定，兩人牽著手跳了下去，瞬間，保護的力量化為毀滅的力量，這邊，也結束了。

　　「該死！儀式失敗了……」

　　槌原當主還來不及抱怨完，虛中的瘴氣便從洞口湧出，瞬間充滿整個洞穴。

　　最後，一切都結束了。
　　亞子的身體被利刃刺穿了一個洞，死後，靈魂從身體脫出，仍是望著女兒慘死的那一幕，為自己無力阻止感到悲哀，為自己的悲傷找藉口。

　　造成這樣的原因，起因全都緣由雙子，要是沒有雙子，自己就不會受到兩次這樣的痛苦了。

　　雖恨雙子，但亞子還是想找回女兒，並且，她要向所有的雙子報復。

　　然後，亞子伸手掐住了眼前的雙子。
