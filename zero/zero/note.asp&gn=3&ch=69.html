螢からの手紙二 ( 螢的來信二 )

天倉螢
入手時間：一ノ刻
入手場所：黒澤邸．二階．優雨の部屋
備考：--
前略

關於「沉眠之家」的都市傳說的事，地圖沒有標示附近有個廢村。

那個照相機是在那個近有個巨大廢屋裡發現的。

由病中的姊姊保管，姪女澪 ( 因為繭失蹤那時給你添麻煩了，還記得吧 )
看到那個照相機顯得很害怕，從言語中好像想起什麼事的樣子。

想暫時交給你保管。的確，你對古道具比較清楚，如果能調查到什麼就太好了。

我想暫時還是無法取得連絡，因只有我單方面連絡你，
若查到什麼的話，再見面時請告訴我。

　　　　　　　　　　　　　　　　　　　　　　　　　　　天倉 螢 草
