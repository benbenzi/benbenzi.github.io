紅蝶技巧心得

各靈體觸發連續攻擊的時機

連續拍攝 就是在相機對着靈體時出現紅圈紅點時進行連續攝影。但是每一個靈體會在有不同的時間出現紅點，如何掌握每一個惡靈的紅點時間就是在任務模式取得高分數或突破惡夢模式的關鍵。
  • 躲在箱中的女性：她會慢慢一步一步向你走，等到她停下來時向後面退一點點，等到她將身子挺起要抓你時就是紅點。
  • 闇黑男：先靠近他，在他向你「鞠躬」後，頭往上移就是紅點。
  • 闇黑女：速度比闇黑男快，正當她要抓你的時候就是紅點。
  • 被沉溺之女：等她潛下水底，從橋底下冒出來抓你的時候就是紅點。
  • 神官：他有兩個時候會出現紅點，第一個就是他靠近你用權杖揮下來，第二個就是他要放靈魂權杖往後面擺放鬼火就是紅點。
  • 被楔殺害之男：先靠近他，在他向你「鞠躬」後，頭往上移就是紅點。
  • 被楔殺害之女：他有兩個時候會出現紅點，第一個就是她靠近你時，第二個就她到達一定距離後就會停下來身子向後擺作勢攻擊你就是紅點。
  • 小孩：他們衝向你的一瞬間就是紅點。呼叫朋友時就可以 。
  • 頭被折斷之女性：她靠近你時會蹲下再準備抓你，那時候就是紅點。
  • 跳下之女性：等她消失之後，舉起相機向後退，掉下來的瞬間就是紅點。
  • 人偶師：通常在人偶攻擊後發動，他衝過來時就是紅點。
  • 桐生雙子：先找出靈體 ( 詳情看下方心得 ) ，當她靠近你並舉起雙手時你先慢慢向後退，等到她露出臉撲向你時就是紅點。
  • 立花千歲：她有兩個時候會出現紅點，第一個就是等到她朝你靠近攻擊時。第二個就是當她坐着哭喊就是紅點。
  • 忌人：他和你靠很近的地方會停下來再突然衝過來，一直等他衝向你的那一瞬間就是紅點。有時候他們會站着轉側面，可以移一步再停下來引他。
  • 黑澤家當主：等他衝過來時就是紅點。( 詳情看下方心得 )
  • 繩の男 ( 楔 )：在他作勢攻擊你的時候別急着攻擊，他會舉起右手一段時間後突然揮下，在他揮下的瞬間快速把相機往左移就是紅點。如果他揮到一半時還沒有紅點請馬上用右搖杆後退，加減拍一下或等待第二次機會。
  • 黑澤紗重：當她三次抓不到你後，解除無敵模式並自言自語就有紅點。

蓄靈力的竅門

當你在攝影後靈力燈發亮時迅速放下相機再舉起，你蓄的靈力會比正常多。

「避」裝備機能的深入探討

當被靈體抓住時按下圓形或 R1 按鈕就有很大的機會能發動，發動後你完全不會掉血。通常會發動失敗大多是因為太早按下去的關係，應該等到畫面放大要反白之前按下去。並不是對所有攻擊也有效，有些攻擊怨靈會先把你揪住 0.5 至 1 秒才會產生傷害，此時才會發生效果。當怨靈直接向你攻擊馬上產生傷害時 ( 通常是衝刺 )便會無效。

「封」強化鏡頭的使用時機

先儲滿靈力，在接 COMBO 時到第三段時使用，走向靈體再進行 、 再用鏡頭完成 、 即可達到分數之最大值 13461 ( 使用底片為一四式 )。如果只是純粹三段接的話，只有 10448 pts 或 11968 pts。

「連」強化鏡頭的使用時機

在黃色靈力圈蓄力後但不達成紅色靈力圈時殺傷力最大，若是在紅色靈力圈 ( 即達成 條件 ) 時使用威力反而變少。在全蓄力黃圈加上甚至可以打出超過紅圈紅點的傷害值。這個鏡頭不適用於拼高分的時候用，只適用爭取時間的任務使用。

「誘われる繭」的拍攝方法其一

收集所有靈的照片中，遇到最困難就是第二刻的「被引誘的繭」。這照片亦是整個系列作中數一數二的難。必須使用被廣泛討論的「三走九跑法」才可以照到。首先貼牆跑繞到左方的記錄點，這樣可以避免觸發繭的情節。當你跑到桐生家那個轉角位走三步到達臨界點 ( 即是走多一步繭就會出現 )，然後用相機模式對準繭消失的角落。放下相機按 X 跑九步，立刻拿起相機拍下去。

「誘われる繭」的拍攝方法其二

其實這照片有沒有擴沒影響，擴只是讓鏡頭縮放讓取景畫面得到遠近變化。但其實這張照片主要是看主角跟繭的實際距離，如果距離不夠近將鏡頭拉近也是沒用的。這張照片是比較特殊的照片。要訣是要繭出現後操控主角跑一段路到適當時機就舉機，也就是跑到繭要消失時才舉機才能讓跑的距離夠長，才能讓捕捉光圈的小圓圈跑到繭身上成為準心，也就是偵測到繭。一定要距離夠近才會偵測到繭，偵測到繭拍照下去才算成功。綜合起來就是，繭出現前先選擇一條直線路徑，對準繭消失的地方往前跑 ( 依照選擇路徑不同要小心別被建築物卡住 )，跑到你覺得繭馬上就要消失時就舉機，然後相機會自動偵測到繭。這時請不要移動取景器對準準心，最好是直接拍攝，以免花時間對準後反而遲了一步。如果失敗就再讀檔重新試一次，慢慢抓舉機的時機，直到有讓捕捉光圈偵測到繭為止。只要能抓到時機，大概馬上就能成功了。時機大概就是操控主角一直跑到繭消失前的那一秒鐘馬上舉機吧。

二ノ刻 双子巫女 血付指輪隱藏事件

在二ノ刻逢坂家奧の間裡取得村調查記錄一、四後，再調查下面四個地方：
  1. 三方路の双子地蔵
  2. 蔵近くの井戸
  3. 暮羽神社入り口
  4. 遊戲一開始的御園の鳥居
在這四個地方會得到完整的調查紀錄，每次調查後都會出現民俗學家的靈，記得要把他攝影下來。拍完第四個後，二ノ刻在御園の鳥居調查時就會得到村調查紀錄五。之後在四ノ刻在黑澤家大広間的納戶才可以照到槙村真澄的自縛靈，得到血のついた指輪。有了指輪後在五ノ刻時前往逢坂家奧の間便會發生事件。拾起血塗れのメモ後槙村真澄和須堂美也子便會聯手出擊。

三ノ刻 大償 大広間的怨靈體為何拍不死

在繭動畫身處的房間內的怨靈是無敵的，稱為「絕對靈」，而且被摸到就馬上死。你只要繞過大広間到另一邊的門便可以去到坪庭階段繼續遊戲。

六ノ刻 鬼隻 戰鬥時如何判斷那個是人偶

有沒有發現當你和桐生茜和桐生薊戰鬥，有時候會攝影無效？因為你照的並不是桐生茜的靈體，而是人偶，人偶是沒有血量的。至於如何判斷那一個是人偶，看得到嘴巴的才是靈體看不到的就是人偶，還有人偶在攻擊時頭會往左右擺和聽到像機械的怪聲。

六ノ刻 鬼隻 如何進入桐生家仏間旁的房間

得到仏間的提示相片後，走進仏間會聽到有人在講話，對那牆壁洞裡的人偶師攝影，聽他講完話後離開你的視線並有開門聲後，才可以離開仏間。開門後便可到隔壁あかずの間。

七ノ刻 紗重 立花千歲躲藏的位置

五個場所分別是：
  1. 立花家．二階．階段廊下 / 下方衣櫃門，會先聽到動靜的
  2. 立花家．一階．玄関 / 出座敷部屋後右手邊的櫃子
  3. 立花家．一階．上座敷 / 上來先調查一下衣櫃，準備離開的時候自動出現
  4. 立花家．二階．並んだ座敷－西 / 調查剛剛劇情時你藏身的衣櫃
  5. 立花家．二階．障子の間 / 開「押入」的門
很多攻略說要打五次，其實只要打頭三處就可以了，不用理會打的先後次序，其他二次可以不用打。最後一次打敗立花千歲便會得到鈴の鍵。

八ノ刻 片割レ月 四個家紋風車的位置

先在七ノ刻時拿到立花家風車。八ノ刻到逢坂家着物の間裏面的納戶，調查紅箱子戰勝後得到逢坂家風車。再由側門進桐生家 ( 建議不要從正門進入 )，上二階對人形攝影，然後在障子の間內調查牆壁可進入密室，戰鬥後得到桐生家風車 ( 記緊在動畫前推門出密室，不然會很難纏 )。最後到皆神墓地對着祠堂拍照後，一個墓碑會發生反應，去拍攝有反應的墓碑之後會引出怨靈，戰勝後調查剛拍攝過的祠堂得到槌原家風車 ( 次序不分先後 )。

八ノ刻 片割レ月 如何拍攝三個小孩

這裡三個小孩會往不同的方向跑。比較保守的方式是先抓兩個來拍，接着在通關後二周目時拍剩下的那一個。由於九O式底片沒有裝填時間，請盡量使用，不然當你裝填底片時小孩早就跑光了。有玩家說可以用先拍他們，我沒有試過你們可以一試，時機抓的好就可以一次全部照到了。

終ノ刻 紅い蝶 惡夢模式千萬不可以去的地方

玩惡夢模式時以下幾個地方千萬別去，否則你會後悔的：皆神村皆神墓地會引發三個村民之靈，而且一旦引發根本不可能甩掉，他們甚至會跟着你在全村跑上一圈、黑澤家明暗の間會在坪庭階段被二個小孩纏住，連地形都註定你在這個難度下跟他們基本沒的打、黑澤家大広間會被 N 多個攻防超強的楔殺者一起追殺，在惡夢模式下你只要進去就別想活着出來。

終ノ刻 紅い蝶 深道的八個忌人

忌人是本遊戲數一數二難纏的傢伙，其實深道的忌人是不用殺的。每遇到忌人就給他一下 然後馬上往前衝，因為他將會向前亂抓。至於最後下樓梯那三個忌人，先在左方拿到九O式底片，接着馬上跑向樓梯並進入相機模式，這時你會自動轉向背面，別管前方的兩個忌人，馬上 360 度轉方向對準樓梯下的忌人給他一下 然後馬上往下衝。如果你技術精湛，則可以不停用追加機能閃過忌人戰隊。

終ノ刻 紅い蝶 擊退黑澤家當主的方法

黑澤家當主有三種攻擊模式。發放鬼火、繞圈準備近身打你以及召喚兩個官司。如果他發放鬼火，你可以對準他到一定時間會有 可以拍。如果他繞圈準備衝過來，你可以一邊後退慢慢後退準備抓三段 COMBO。當集滿靈力便使用強化鏡頭抓 COMBO。

零ノ刻 虚 拍攝天倉繭時注意事項

進入零ノ刻後，往下走後跟姐姐對峙時先不要走太近，不然跟繭距離夠近的地方時，繭就會往前走。千萬別被抱到，這時候的繭是一擊必殺的，被摸一下就要重玩。首先進入相機模式然後用 R3 慢慢靠近繭， 在繭開始往主角移動時用 R3 開始後退，靠很近時繭就會直接撲上來，只有一瞬間很短的時間等到紅圈紅點馬上拍下去，如果你在她跑來時才進入相機模式多半會趕不及。有玩家說可以用先拍她，我沒有試過你們可以一試。

零ノ刻 虚 擊退黑澤紗重的方法

首先跑到虛的左邊等待紗重對你進行攻擊 ( 右邊角度比較難看 )，先等她站着狂笑然後走上去引誘她提早抓你，當她三次抓不到你後就會解除無敵模式並自言自語，這時抓三段 COMBO 靈力就會滿 4 點，然侯紗重又進入狂笑模式。當跑向狂笑的紗重時她會消失然後出現在附近地方繼續狂笑，繼續向她跑她便會由狂笑模式轉為抓人模式，重複剛才的步驟抓 COMBO 第一段就用鏡頭然後把 COMBO 接完，再次轉移陣地重複剛才步驟。也可以一開始先往紗重拼命照，靈力滿 4 點後開始引誘紗重抓你，當她解除無敵模式時換上零式 +把 COMBO 接完。
