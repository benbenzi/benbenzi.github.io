零紅蝶惡搞【 第六刻 】噗吱

KiryuuT. tina8968 ( 巴哈姆特 ) 著
進了這棟建築物，裡面陰森的感覺不比冰河女家和養豬廠。
我四處晃晃，這裡是玄關啊！門呢？門在哪？
當我搜尋著門，窗戶的破洞出現了兩顆大頭。
是兩個小女孩，他們喃喃低語不知在說什麼，我仔細地聽。

「嘻嘻嘻，她好醜喔……」
「對啊，跟剛剛那個一樣醜……」

怎麼可以拿我跟姊姊相比呢？
我生氣地衝到窗戶的破洞前，那兩個死小鬼也消失了。
不過正因為如此，我從窗戶的內部撇見門才知道門在哪裡。
不然我可能會直接把窗戶砸了衝進去。
進去之後我撿起一本掉在地上的紫色日記。
紫封皮日記 ( 1 )
為什麼要殺呢為什麼要唰呢
會什麼要唰呢貴蛇咪告差呢
阿啥裡跳擦的襪俗仳藥渣了
嗚苦靡耗紮捏……

我想應該是剛剛那兩個死小鬼寫的。
該說我老了還是怎樣？日記的內容我完全看不懂。
小孩子寫的日記就是那麼天馬行空，童年真好。
不過雖然說看不懂，裡面還是有些字我看得懂。
我繼續盯著日記瞧，我得到了一個總結……

這死小鬼是個關西腔。

難怪我剛剛看時都看不懂，原來她是把她的關西腔原封不動地寫下來。
真是本口語化的日記。

我開門繼續走，走廊又看見兩個死小鬼在亂跑，我緊追著他們。
跑到某個房間後，那兩個死小鬼又消失了，沒辦法，繼續走吧。
我正要打開某個門時，兩個死小鬼又出現在我身後的坐墊上。
我直接把包包拿起來丟他們，丟到其中一個死小鬼了。
那死小鬼摸摸頭，把她的頭扶正之後又消失了。
我撿起我的包包，底下又壓了一本日記。
紫封皮日記 ( 2 )
偶親所，掐死嚕薊。
好卵、好湳。
暗暗滴動瑟。
但速，只靠偶低力量係乎只會給薊帶來透骨。
最後，勾司大人原本想要巴麻。
薊搜，幾想讓偶藍。
我和薊合而為七。
但速，薊寒四屬哩。
再诶不能恨他搓掛。
物七大狼為哩偶把薊帶奎來了。
偶批僻滴鬥洩，薊诶延亮偶了，因廢朴摁再唰哩，因廢再诶每右郎需要唰郎了。
物七大狼唆薊漢成了蛆而要唰了啪。
打算唰了薊。
偶不以洗，偶不嘿以洗那總速奇花生。

老實說，我看得很痛苦。
原本就不了解關西腔，加上裡面的內容亂七八糟的，實在是看不下去了。
而且偶也懶的老解角含子在梭什麼。
唉……看了這本日記連我都快要變成關西腔了……
收起日記我打開門往外走，是一條走廊。

當我走到走廊中間，兩個死小鬼出現了。
他們一個在我前面、一個在我後面，慢慢地走過來。
我拿了你們的日記就想圍毆我嗎？要打架來啊！
我直直地看著某個死小鬼的眼睛，她的眼睛被頭髮遮住了看不清楚。
幹麻用頭髮把眼睛遮起來？
在我好奇之下，我衝到某個小鬼前面，移開她眼前的頭髮。

「看到了！」
「呀！倫家不要啦～～」

果然是關西腔……
那小鬼頭往後一彎，整個頭髮甩在我臉上，痛死了。
怒火中來，靠得那麼近，剛好可以揍她一頓。
但是……人消失了？
好不容易才找到一個發洩的對象……
既然消失了我也沒輒，我繼續走著。

走到一個房間裡，裡面擺滿了一堆人形，其中以房間中間那兩個最大。
仔細看，長得跟那兩個小鬼還滿像的。
其中一個好好的，但另一個斷手斷腳還斷頭。
我走近那個完整的等身大小人形仔細看。
做的還不錯嘛！拿去賣至少也有個幾萬吧。
我打算把人形塞進我的包包裡，但人形太大了，先去搜刮一下屋子再來拿好了。
總不能叫我拖著一個人形到處跑吧？

這屋子很大，害我一直迷路，而且有夠破的。
走到一間佛堂，牆上還破了一個大洞。
這洞怎麼破的那麼大？
我走近看看，沒想到洞前出現了一顆人頭，我嚇得倒退三步。
原來是一個老不死正在對我比鬼臉吐舌頭。
死渾蛋！我的拳頭直接從破洞穿出去 K 他。

他飛出去了，而且還撞到牆壁。

知道我的厲害了吧！
我走出去要收屍，可是人不見了，跑得還真快。
我走回佛堂，跟剛剛的景色不一樣，我好像走錯間了。
不，根本就是走錯間了，沒差，先搜刮一下。
我從桌上撿起一本破日記。

人形師的手記
嘿嘿嘿……軀的頭跟手都被我藏起來了。
哈哈哈……有種來找啊！

這是挑戰書吧？
不過他說的頭跟手應該是我剛剛看到的斷手斷腳爛人形的身體部份。
要是我找到那爛人形的身體部份的話……
再把兩個人形拿去賣……然後……錢乘與二……
好吧，別人的挑戰哪有不接受之理，我馬上就把人形的頭跟手找出來！

我走出房間，老不休從我眼前快速飄過去，我連忙追上去。
他飄進了一個房間，我打開門。
老不休正站在房間中央對我比鬼臉。

「醜八怪。」
「……」

對我比鬼臉還罵我醜八怪，死白目。
我不但用想的而且還說出來了。

「死白目！」
「妳說白目是吧？」

我還以為那老不休又要說什麼了，沒想到他從他的衣袖裡掏出了一個東西丟出來。

「來，白目。」

那是人形的眼珠，這老不休真好玩……
再多罵他一點看看……

「死腦袋一個……」
「死腦袋是吧？來，腦袋給妳。」

那老頭又丟出一顆頭，是人形的頭。
再繼續把手臂要到手吧……
我這麼想時，沒想到那老頭正在對我比中指。

「手賤啊！」
「對，手賤！」

那老頭又把某個東西丟出來了。
是人形的雙臂，我真搞不懂這老頭到底在幹麻……
反正東西撿一撿也該走了……我不想跟神經病一起玩……到時候也變成神經病……
我轉身要走出去，那死老頭居然飄到我前面挖鼻孔。

「廢渣，滾！」
「嗯……廢渣……」

那老頭又丟出一本紫色的日記跟一本爛書，我懶得陪他玩了。
我直接對那老頭揮出右拳，那老頭被我打得哀哀叫，隨即消失。
要是他還要再繼續玩下去，說實在，我很傷腦筋的。
我翻開他丟給我的紫色日記。
紫封皮日記 ( 3 )
偶才不要什麼替身。
偶速一直和茜姊姊一起的。
和茜姊姊合一。
刷，唰哩那個人形。

原來那兩個死小鬼的其中一個叫做「茜」啊，那個關西腔很重的傢伙。
至於那個關西腔不怎麼嚴重的應該是茜的妹妹，叫做「薊」吧。
我正打算要翻開另一本老不休給我的爛書，標題是「人形師的設計圖」。
這本爛書應該是在說組裝人形吧？等等我要裝人形時再看。
我走回擺滿人形的房間拿出人形師的設計圖翻開。
人形師的設計圖
p1 ( 第一頁 )
某天，我在房間裡組裝新人形，這時小茜跟小薊走過來吵著肚子餓了。
這時我應該……

1. 快點作飯給小茜跟小薊吃。( 請翻到第 3 頁 )
2. 不理他們，繼續作人形。( 請翻到第 2 頁 )

誰理什麼小茜還小薊的，當然是裝人形啊！
我翻開第 2 頁。

p2 ( 第二頁 ) 你這個死沒天良的！
要是我的小茜和小薊餓死怎麼辦？
你要怎麼負責？

Game Over

這是什麼爛書啊？別人要組裝人形還玩這種把戲。
我直接翻到最後一頁，我想組裝的方法可能在最後一頁。
p996 ( 第九百九十六頁 )
哈哈哈！你以為翻到最後一頁就可以看到組裝人形的方法嗎？
你想得美！！

爛書！
我把那本書摔在地上踩了又踩，最後還是很無奈地撿起來。
九百九十六頁……要怎麼找啊……
沒辦法……只好慢慢玩了……
我翻到 p3
p3
把小茜跟小薊餵飽後，我回到房間繼續做人形。
做著做著天色已晚，小茜跟小薊說想要到外面玩。

1. 帶他們出去玩。( 請翻到第 49 頁 )
2. 叫他們乖乖去睡覺。( 請翻到第 67 頁 )

要順著那兩個小鬼是吧？我翻到第 49 頁
p49
你白痴啊！
天色那麼晚了，應該要叫他們去睡覺啊！
小孩子怎麼可以熬夜呢？
而且那麼晚了，要是被壞人抓走就糟糕了。

Game Over

這老頭做的書真的很欠扁……
於是我只好翻回前頁慢慢地玩這本書上的遊戲。
好不容易被我玩到最後了……
p747
恭喜你到達本書的最後一頁了！
看來我該敎你怎麼組裝人形了。
請先拿出塞在人形內側的紙條，上面有說明。

我再度把這本書丟在地上踩。
書裡根本就沒有說怎麼組裝人形，純粹是那死老頭在耍我。
不理這本書，果然，我從人形理面找出一張紙條。
上面有說怎麼組裝人形。
不過人形很老舊了，到處都是灰塵，害我背上的青春痘一直在癢。
為什麼別人的青春痘都是長在臉上而我的是長在背上？
這真是不公平！要嘛就不要長，要嘛就長在臉上嘛！
我邊裝邊摳背上的痘痘。
過不多時，人形已經被我裝好了。
裝好之後，我開始般人形，但是怎麼般都搬不下來。
這時我把我抱的那個人形往右移動，另一個人形居然往左移動了。
這又是怎麼回事？我開始亂轉人形，人形也跟著我轉動的方向亂移。
當我把人形轉成面對面時，我前方的衣櫃居然倒下來了。
而且還冒出一堆灰塵……害我背上的痘痘又在癢了……
我右手伸到背後抓痘痘，而且還摳下一層皮，這時我感覺到我後面有一陣風，我轉頭看。
我背後的人形手呈推人狀，我跟人形面對面呆在那邊。
我順手把摳下來的那層皮彈到人形身上……

「噗吱！倫家蒲要啦！！」

她退後三步，沒想到另一個人形也退後三步。
原來是那個叫做茜跟薊的死小鬼裝的。
剛剛那個說話關西腔超重的應該就是茜了。
我繼續把背上的痘痘皮摳下來，彈在她身上。
茜向後轉奪門而出，剛好順手把門關上了。

真是個隨手關門的好小孩。

薊也跟著茜要跑出去，可是茜把門關起來薊反應不及，一頭撞在門上。
薊的頭跟右手臂掉下來了……也對……她剛剛是我裝的人形……
接下來，薊做出讓我傻眼的事……
她把頭安裝在右臂上，把右手臂裝在頭上，然後左手開門逃走了。

她可能覺得不管看到的東西是歪的還是正的都沒差吧……

不理那兩個小鬼了，看看那個倒下來的衣櫃。
原來那衣櫃後有一條地道啊。
我走進地道裡……
