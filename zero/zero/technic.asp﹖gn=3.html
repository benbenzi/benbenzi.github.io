刺青之聲技巧心得

睡醒時對道具數量的影響

每次當你回到夢境時，萬葉丸、O七式、一四式底片的數量會變回預設值；六一式或以上的底片、御神水和鏡石則會繼續累積下來 ( 只有雛咲深紅和天倉螢擁有六一式底片預設值 )。所以若你在普通模式中回到現實時身上餘下萬葉丸一瓶、御神水兩瓶，回到夢境後萬葉丸數量便會自動變回三瓶，至於御神水由於沒有預設值，所以保持兩瓶。三人之間不會互相共有道具。

O七式底片99 張50 張30 張20 張
一四式底片60 張30 張25 張20 張
六一式底片20 張10 張10 張10 張

在現實黑澤家卡關

在家裡卡關有幾個可能，通常跟深紅講話、去暗房沖照片、聽錄音帶、去玄關鞋櫃或廚房桌子上拿信、接電話以及幾個房間調查一下。調查床時出現選項便會可以選擇睡覺。

提高黑澤怜的除靈能力 ( 嚴重劇情透露 )

刺青之聲中是由三位角色交替使用，所以道具都出現在相同位置。但是只有萬葉丸、一四式可以重覆取得，而御神水、鏡石、六一式、九O式、零式。當角色 A 拿了，角色 B 便拿不到。所以在刺青之聲中道具不可以亂撿，如果撿了不應該撿的東西有可能導至遊戲後半時其他角色戰力不足的情況。由於最終頭目是由黑澤怜出戰，所以最好把零式或九O式底片留給她，補品多留給雛咲深紅。注意雛咲深紅在十一ノ刻而天倉螢在十二ノ刻後便不會再出場，所以深紅篇和螢篇最後一刻時底片和補品並不用吝惜盡量用吧。

祓瘴燈火 ( 蠟燭 ) 的作用

十ノ刻後場景會彌漫着瘴氣 ( 黑白畫面 )，身上持有祓瘴燈火才能吹散瘴氣。蠟燭是有時效的，一支全新蠟燭大概只能維持約 5 分 35 秒，熄滅後開始被刺青巫女追殺，追殺時重燃燈火無效，一定要等到劇情動作才會解除。零華在終ノ刻之前也是絕對靈，不要妄想去打她，被摸到就馬上死。在十ノ刻後必須清楚知道屋子內各祓瘴燈火的位置以及那些地方即使在燈火熄滅的情況下零華也不會出現。祓瘴燈火在簡易至困難模式的時效是一樣。

【避】裝備機能的深入探討

當被靈體抓住時按下圓形或 R1 按鈕就有很大的機會能發動，發動後你完全不會掉血。通常會發動失敗大多是因為太早按下去的關係，應該等到畫面放大要反白之前按下去。並不是對所有攻擊也有效，有些攻擊怨靈會先把你揪住 0.5 至 1 秒才會產生傷害，此時才會發生效果。當怨靈直接向你攻擊馬上產生傷害時 ( 通常是衝刺 )便會無效。

【測】裝備機能的取得方法

六ノ刻在眠りの家一階仕切られた部屋拍攝瀧川吉乃，接着在現實世界沖洗照片生き残った女後得到うずくまった女。然後在九ノ刻到眠りの家一階玄関，戰勝乳母車の老婆後取得違い矢筈の鍵。用它打開左方旁邊的門，通過隱藏通道到仕切られた部屋跟瀧川吉乃戰鬥，戰勝後取得裝備機能。

【連】強化鏡頭的使用時機

在黃色靈力圈蓄力後但不達成紅色靈力圈時殺傷力最大，若是在紅色靈力圈 ( 即達成 條件 ) 時使用威力反而變少。在全蓄力黃圈加上甚至可以打出超過紅圈紅點的傷害值。這個鏡頭不適用於拼高分的時候用，只適用爭取時間的任務使用。

PS 版第二結局取得方法 ( 柊の鍵 )

十ノ刻或十二ノ刻的天倉螢跟久世鏡華在眠りの家二階着物の間交戰後，在二階階段廊下推開櫃子從屋簷走到二階閉ざされた座敷。沿梯子爬下到一階囲炉裏の間後會出現髪を梳かす女。戰勝後取得強化鏡頭、二周目才會拿到柊の鍵。回二階着物の間開啟梳妝台的抽屜取得鏡華の耳飾り、唄ト伝説原本和螢に似た男性。有了鏡華の耳飾り做護身符，天倉螢在十二ノ刻便只會躺在怜客廳的沙發上沉睡而沒有消失，通關後才有三個人都活下來的結局。這個結局是二周目限定，並與難度無關。

十二ノ刻．戒ノ儀．擊退久世家當主的方法

久世家當主其實很好打，不要以為她有很多手便會很強。她一共有三招，一招會放千手抓人，其餘兩招也會放出雙手，其中一招有紅點，另一招只有紅圈。當她彎下身子時便是準備使用千手抓手；第一招是原地站着放出雙手，當手飛到一定距離後便會出現紅點；第二招便是彎腰類似深呼吸的動作，這時會出現紅圈。被她的手印打中便放下射影機隨機應變吧。擊退她的方法很簡單，其實只要她在左邊，你便跑到右邊，她到右邊你便跑到左邊，她所走的路線是固定的，永遠和她保持一段距離。當她放出雙手時不要緊張也不用放下射影機，直接用 R3 左右平移便可以輕鬆避過。和她一直繞圈跑，一有機會便拍下去。

終ノ刻．涯ノ淵．擊退久世零華的方法

久世零華有三招，第一招是帶你進異空間，第二招由天上高速俯衝下來，第三招便是慢慢向你逼近或放出鬼火。如果她使用第一招，畫面會變為黑白，速度也會變慢。先別亂動，在你未發現她由那一個方向來時千萬不要動，逃跑時多留意腳旁有什麼東西，因為當你被河卡着時便會被抓到，在異空間中零華是一擊必殺的。如果你想輕鬆離開異空間切記以下兩點：在進入異空間前先留意現在的地形，進入後不要往河的方向跑；其次是逃跑時不要一直跑直線。另外第二招當她由天上衝下來時便是紅點時間，當她升到天上時馬上把射影機往高空對焦，然後等待她俯衝下來。注意她的飛行角度是拋物線而不是直線，所以射影機最好偏地面對焦。她的速度極快，紅點亦很短，若你不是有十足信心最好先逃跑，被她打中付出的代價會相當大。最後便是在地上慢慢向你逼近，這時會出現紅圈亦是攻擊她的最佳時機。基本上只要你繞圈跑俯衝那招是打不中你的，異空間中只要小心一點便行，一直等她使用第三招便對她攻擊便可以輕鬆把她擊退。
