日本民俗學選集．七　魂鎮 タマシズメ 第六章

真璧清次郎, 宗方良藏編

陰祭

　終於，代替「紅贅祭」的「陰祭」即將開始。黑澤家、桐生家、立花家、逢坂家、槌原家以外的家屋，全部門戶緊閉，御園則被火炬照亮。據說本來祭典舉行的時候，村人們會手持火炬集合到御園、祈禱著、唱著贊歌為生贅之魂送行。御園是在「虛」的正上方，模擬「虛」開的一個洞穴。以被稱為贅岩的岩石堵塞於該穴上，代替生贅的意思。同一時間，地下真正的「虛」，則以「雙子巫女」為生贅，圍繞其之神官們唱著贊歌。因為地下的祭典是絕對不能看的，所以神官們皆以布遮住臉孔，幫忙祭典被稱為忌人 ( イミビト ) 的人們則是雙目被縫上。因此，通常村人們只能在御園祈禱。我則是稱為「楔 ( クサビ ) 」之一時頂替的生贅。根據書物，客人成為生贅的情況，給予客人的苦痛越強烈，成為「楔」對「虛」的抑制效果越佳。苦痛在隱藏文書裡有寫到，即為具體的以繩子綁住且吊起後，身體被細切。…那樣的苦痛我是否能忍耐得了呢？為什麼那時候沒有和宗方一起逃走而留在這裡呢？即使說不定能看一眼「黃泉之門」，那時候我也已經不是此世之人了吧。至少，麻生的研究成果－射影機的事想在世間公開。若有誰撿到這個記事本的話，希望能代替我將這皆神村中所發生的事傳達出去。
　宗方，我因想知道「黃泉之門」而研究的期間，自己的全部、連魂魄都被「黃泉之門」所奪走。但願，祈禱你不要和我選擇相同的道路。拜託

( 未完 )










真璧清次郎
民俗學者　一八八五～一九四六年　東京都


於日本全國旅行時，一邊調查研究於土地生根的種種信仰、傳承等的民俗學者。其著書超過 80 冊。成為絕筆的本作品，是整理本人的取材記事本，以其為基礎，由其同往皆神村的弟子˙宗方良藏氏加筆潤飾而後發行。
