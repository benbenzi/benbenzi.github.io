夢と鎮魂に関する調查報告 ( 夢與鎮魂相關調查報告 ) 第三章

麻生優雨

背負柊之巫女

　久世之宮被稱為「納痛之宮」。但這心之痛，意即悲傷，是被限定的，限指與人死別之痛。為了與死去之人肉體上．精神上的訣別之儀式，而前往參拜久世之巫女。家族內有死者的話，將該遺體裝入麻袋中，有殘存紀錄寫著看過這些為舉行最後的訣別而來參拜的人們。

　冬天下雪時，參拜者都隱其面目，
　用板車拉著很大的袋子。

　那個樣子看起來就像是哪裡的葬列。

　( 擷取自應該是柏木秋人氏殘留的手記 )

　久世的巫女，恐怕也被稱為搖籃曲中出現的「眠之巫女」，但因她以刺青來背負，代替那些與親近之人死別的人們之悲痛，也被稱為「刺青之巫女」。

　詳細解讀巫女所承擔的職責，無疑的看的出與該搖籃曲「眠之巫女」一致。為表示該歌謠與文身行者間的一致，關於久世之宮所舉行的三個儀式「紫魂之儀」「刺魂之儀」「戒之儀」，整理於下。

紫魂之儀

　製作刻巫女的刺青之墨的儀式。

　來訪久世之宮的參拜者，來「納痛」不僅只是說出悲痛、祈求平安，還有「將痛具象化」的讓巫女背負。即從死去之人的身體裡採取血液 ( 死之血 ) 與殘存之人的血液 ( 生之血 ) 混合製作「紫之墨」，以此墨在巫女身上刻入刺青，可看出巫女的痛 = 與死者訣別之悲痛被代替承受的形式。

　另外，這儀式是以稱為「刻女」的女性為中心來進行的。她們是從鄰近的村中被召入，一旦入宮，通常都是兩人一起生活，將此生奉獻給久世之宮。

　向巫女納痛之人，
　將生之血和死之血的紅與藍混合在一起，
　以紫魂之墨刻劃柊。

　( 引自「紫魂之儀文書」)

　在久世之宮被保管之「戒之儀文書」中，可由記述中推測關於這之後持續的儀式「戒之儀」，這裡也看見參拜者以自己的血寫下自己的名字之殘留書頁。表現出在久世之宮特別認為血是傷的象徵。

刺魂之儀

　以「紫魂之墨」，對刺青之巫女施行刺青的儀式。

　參拜者們的悲傷與痛苦以「柊」之紋樣，伴隨著包圍柊的蛇之圖案，深深刻在巫女的全身。久世的巫女僅有巫覡性象徵的儀式，接受刺青的巫女自身當然受重視，而我想施術的刻女們應該也同樣的受到重視。
　於刺魂儀式刻下的刺青，納入的痛=柊一點一點的刻入巫女的身體。換句話說，在巫女的全身被刺青埋沒前，儀式本身會反覆的被舉行。

　於刺魂儀式中承受許多人們的「痛」的巫女，在她的職責結束之前 ( = 被刺青埋沒進入沉眠 ) 這段期間，究竟是怎樣的處境呢？關於這個問題，是根據我天倉螢不知道是夢還是現實的記憶裡，記得有見過在高高的天花板吊著的牢中沉睡著巫女姿態之女性的印象。那空中垂吊的牢，若是古書中稱之為「浮獄」的東西的話，巫女應該就在那等待著儀式的最終之日的到來。

　如一般所知的，刺青是伴隨著強大的肉體之苦痛的行為。現在也被稱為 Tattoo，流行的感覺，除了是自我表現的手段，另外也有格調變高的感覺，主要將其當成裝飾品的感覺而刺青的年輕人也很多。但因為以色素貫入直至皮膚的真皮，當然也伴隨著相當的痛楚。在以前，刺入大的紋樣伴隨著會讓成熟的成人也失神的痛。
　何況久世之巫女的情況，是為了將人們的痛轉移並代替承受的神聖儀式。我想並不會有減輕她負擔的措施。
　讓年輕女性身受這些痛苦，想必是個辛苦的儀式吧。

刻女

　二人一組的女性。
　製作紫魂之墨，刻劃刺青。

　刻女擔任的任務是製作紫魂之墨並為巫女刻上刺青，但為求她們徹底的完成這任務，而被強迫作肉體的犧牲。
　其一是為了了解刻於身體的痛，而在全身刺入無數的針。用來比擬以針刺入皮下，注入色素的過程等刻入刺青的行為。
　還有一個是為了讓巫女不被現世的煩惱所拘束，以身代之，挖去自己的眼睛。奪去光明的行為，是為了讓巫女喪失對俗世的想念，並也有為了在面臨刻印刺青，不會被看見的東西所迷惑，可以用更上一層的靈力刻劃紋樣。而因外力失去雙眼的她們，已經不可能再回去了。我想應該也有一旦進入久世之門的身體，就不可能再度回到俗世的戒訓意味。
　此外，刻女們被摘出眼球的眼窩，似乎於兩眼間交叉通過兩條麻繩，但關於此的正確理由現在已找不到。因為不單只是將眼睛堵塞，左右兩邊以繩子聯繫捆上的形貌，關閉感，換句話說，可以想像成是將這個與俗世連繫的肉體，變成只連繫巫女一個人的世界的意義。
　這裡特別注意的是，她們這些刻女們，由同世代的女性兩人構成，睡覺吃飯都在一起，到死都於宮內工作的事實。簡直就如同雙子一般，被當作成對的存在對待。
　原本將雙子特別視為神般的存在的案例很多。認為本來為一個的肉體，在母親的胎內分裂成兩個的狀況很多，無論這是吉兆，或是禍事，都會給予其一點意義。例如在中部地方的山間一代現在還存留著「雙子祭祀」，據說兩人到了一定的年齡時，將被舉行神降的儀式。穿著白色裝束的雙子衣帶結著紅結，有「還原成一人時將宿有神之力」的意義。
　想來，無論久世之宮的刻女們是否是真的雙子，以「兩人一組」的形式，說不定也暗示著因這理由所以原本就是侍奉神的身體。

鎮女

　巫女的照顧者之少女。

( 照片注釋：由柏木秋人遺族所藏的映寫機中發現的膠捲的一部分。映出應該是鎮女的少女們之身影。)
　刻上刺青的巫女，從身受無數的針刺而疼痛的肌膚，不能再回到世俗的身體之痛苦，想必會心情紛亂吧。負責支持她身心的，是被稱為「鎮女」的少女巫女。
　鎮女通常由 4 人所構成，與巫女和刻女一樣，是自鄰近的村莊選出的少女。找出像她們這樣 5 歲至 9 歲左右的童女，且自迎巫女入宮那天起，需在她身邊照顧她，不允許再次出去外面的世界，可看出是擔任巫女的談話對象。另外儀式面的任務，是舉行稱為「咎打」的儀式，將代替承受痛的人形釘打於牆上，鎮住受到刺青之痛折磨的巫女。
　而這些少女們最終的任務，便註定要「穿刺巫女的四肢」。關於這點還有進一步調查的必要，但若用文字來解釋的話，就像在歷史上時常被提到的磔刑一樣。
　幼小的鎮女們對「穿刺」這樣殘酷的責任，沒有一點疑問反而極肯定的接受這件事情，讓我感到非常有興趣。在受久世家影響的土地上成長，接受身為鎮女的徹底教育之成果，可以說在此展現出來。
　此外，任期結束的鎮女讓其再度歸家，和其他的村女們一樣，為久世家庇護的一員，什麼事都沒有的生活下去。刺青之巫女的儀式裡，雖然她們揹負極重要的責任，但卻沒有將其封口之類的而讓她們回到俗世的理由是因為，鎮女們還很幼小，且因為人們自私的想法，所以才有人身供品眠之巫女存在的必要。恐怕久世之宮附近村莊的女性裡面，有很多幼小時有擔任過鎮女的經驗，正因如此而對其儀式默然的了解，在必要時也會讓巫女的儀式持續被舉行。

戒之儀

　讓巫女沉眠的儀式。

　結束刺青之儀，全身刻入刺青的巫女，來到刻宮的地下建造的，稱為奈落的深穴，降到它的最底部。奈落的確有通往黃泉之國的入口，不是活著的人可以踏足的場所。巫女在該處與世俗的連繫完全的被切斷。不久，來到久世之宮的最深處，稱為「狹間」的巨大地下洞穴，藉由鎮女之手被釘打封印於祠中，然後迎向永遠的沉眠。
　不用說，這便是給神的生贅，巫女成為人身供品的意思。在地下洞窟的祠中，身體被釘打的狀況下被關起來的巫女，已經沒有逃回地上的辦法，與歷代巫女們的亡骸一起，背負著刺青，結束這一生。
　四肢以杭木穿刺，是自古代於世界各地有殘留痕跡的風俗習慣之ㄧ。較為人所知的是耶穌基督的磔刑之圖，總之，比起說是儀式的要素，還更強烈感到刑罰的微妙意味。於久世之儀式裡的這個行為，是鎮服刺青巫女的心，確實的將之送往黃泉，換句話說是要讓她無法移動，不能回到現實而舉行的儀式，如此強烈的暗示。在那裡，也包含為了讓從人們那承受的刻劃之痛不會返回，並且讓巫女不會醒來，而以肉體的拘束將她留在那個地方的微妙意義。

( 便利貼：「夢」和刺青的關係 )

逆身剝

　對成為巫女卻無法切斷留戀的巫女之懲罰。
　刺青的剝奪。

　關於這件被附上意義的神事，去知道當事者對這件事的想法，以大觀的視點來看，應該是沒有什麼意義。確實是因為知道自己的命運，對成為生贅感到恐懼的巫女們很多吧。
　但是，知道自己所承擔的任務，但儀式進行到一半便拒絕繼續執行，一旦接受刺青，卻無法切斷對世俗的想念時，她們將會被剝下巫女存在的意義－刺青的真皮，被「流放」，即是被殺的樣子。
　「逆身剝文書」裡亦提到關於這個逆身剝的事情。我想恐怕是在久世家最後的當主．夜舟那代還要更遠前的時代的事情，於這文書中，舉行對成為巫女卻無法切斷留戀的巫女之懲罰，稱為「逆身剝」的行為。被剝下來的皮膚，被收納於祠，為巫女鎮魂以及戒示下一位巫女。

( 照片注釋：青森縣民俗學博物館所藏的刺青。可確認蛇與柊的紋樣。)

　以優雨的研究為基礎來看，這個地方可看到以「流放」的表現，用來暗喻「殺死」。但是以關於水的動詞來看，要說起來被認為是海邊的地區可看到的用詞。這樣在山岳區域被使用的例子非常稀少吧。

刺青的紋樣的意義

( 照片注釋：北条家的家紋．三片鱗。巫女離去之後留下三枚的鱗片為其由來。巫女和蛇的密切關係被圖案化的例子。)

　象徵巫女的蛇。
　表示痛的柊。

　久世之巫女刻於身體上的紋樣，是蛇和柊的組合。
　自古以來，巫女和蛇的關係非常得深。蛇被認為是侍奉神的龍之化身，降到地上的龍，是畏懼恐怖的對象。例如巫女卑彌呼，也有將蛇的鱗片似的紋樣描在肌膚上的傳說，這是最典型的吧。將蛇的一部份納入自己的身體，似乎就能寄宿神之力。這不僅是神話的時代如此，因為人被自然所支配，這樣的想法也刻入人們心中。北条時政因能遣使龍神的巫女所引導而得到繁榮，家紋便用表示蛇之鱗片的三鱗形之家紋。且從古文獻中，超常的力量之化身，常常有許多「被神之蛇所襲擊的人們」之類的繪圖殘留著。超自然的災害與超越人類智慧之力量的象徵，大多使用蛇。久世之宮的巫女以蛇作為纏繞的紋樣，也可以看出是忠實的取材於此類事例。
　另外關於柊有許多說法。之前曾說過，因碰到鋸齒狀的葉子刺刺的疼，自古以來便被稱為「疼木」，以這個痛來比喻描寫巫女背負的疼的說法，是最一般的說法。另外也稱為「鬼之刺」，因為已知在節分 ( 編按：立春前一日 ) 時，柊枝與沙丁魚的頭一起被用做消除災厄的象徵，擁有除魔的力量。為常綠樹，說不定在雪中真摯的連接生命的形貌，給人有神靈附身的印象吧。從這樣的特徵看來，大概是因此成為巫女之紋樣的動機吧。

破戒

　巫女自夢中醒來。

　 意指以刺青刻上的痛所見到的夢=寢目，返回納痛的人們的狀況。
　 　痛返回時，刺青擁有蛇所擁有的神之力，會向痛的本來持有者襲去，被認為是絕對的禁忌，為了防止這情形發生而用盡所有手段。
　 　破戒在傳承中只有比喻性的被提到，找不到關於實際情形的清楚敘述文獻。最初實際引起破戒的紀錄，只在久世家最後的當主．久世夜舟時看到。換句話說，本來應該不會引起的變數發生了，夜舟時的破戒，的確是儀式徹底的失敗了吧。

　「文身行者」的傳說裡，有這樣的事件－僧人全身接受的痛，但因為某契機而返回原本痛的持有者，從刺青的紋樣裡實體化的蛇，將他們吃掉。以此做比喻的話，若說背負蛇的巫女所引起的破戒是超常現象，那「因巫女的詛咒引起的巨大破滅」是可以想像的。

　成為破戒之誘因的事件為何，這裡僅作推測，
  • 對承擔柊這件事 ( 巫女的責任 ) 的拒絕
  • 往眼球的刺青 ( ∵根據「文身行者」的記述 )
　「於眼中刻入刺青」這句話的意思 ( 即使從是否可能實行的側面來看 ) 應該不是如字面意義「實際於眼睛刻上刺青」，恐怕是巫女的眼睛被刻上「傷」，換句話說，說不定是於與現世聯繫的眼睛上被刻上傷 = 悲與痛，如同成為遺憾的事物被烙在眼上這樣的解釋吧。

　這樣解釋的話，與後面所敘述實際引起破戒的原因一致。

狹間之宮

　為防止破戒擴張的建築物。

　萬一引起破戒時，為了防止它向外界擴張，採行將社全體以巨大的建築物覆蓋的方法，狹間之宮的名字意即將久世之宮非置於此世，而是留在那邊的世界。
　並不只有用物理性的與外界隔離，還使用人柱的呪術之手段，與招致破戒的巫女一起沉入黃泉。
　以狹間之宮完全覆蓋，此外再施以二重三重的包圍，與外界完全的被遮斷的久世家，變成擁有非常複雜構造的巨大屋子。關於屋子的構造，從乙月要及當時「沉眠之家」的患者們的證言中可以確認，我想大致上不會錯的。
