第二夜 鬼遊び 攻略

深紅做了一個惡夢，夢見自己變成繩之巫女，四肢被繫上繩子肢解。噩夢醒來發現自己竟然沒死，難道是霧繪藉著惡夢將自己的痛苦傳遞給深紅知道嗎？醒來眼前竟然是個小孩，手裡就拿著前一晚辛苦才到手的御神鏡碎片。當你要喝令他時他一溜煙就跑掉了。醒來時就在和人形部屋裡，發現角落的牆上有靈動，拍下出現一幅掛軸。撿完東西開門看到那小孩飛奔似的從你眼前晃過，走回弔間打開荒廢部屋存檔。在弔間又會得到卷軸，出現目隱鬼儀式。在荒廢部屋前方的大門有靈動，拍下出現古井。井旁有一刻，石往有古井的井戶裡庭去。經過櫻之中庭強制進入劇情，出現了目隱之靈。經過鏡子前面，那個小孩又忽然出現。走道轉角處出現了第一夜也看過的長手男之靈，拍死他以後也不會再出現了。往前走看到前方的門出現靈動，拍下出現樓梯下的畫面。還好往回走幾步路就可以在樓梯下拍到封印靈避中的怨念，拍完後門就可以開了。在水道部屋那小孩又衝出來了，這回比較好拍，之前兩次錯過的務必把握。出去的門密碼是「 1126 美版是 2611 」。存檔完畢後前去把那古井的蓋子打開，果然冒出一個水井少女之靈，擊敗她後得到蒼色刻石。

  • ( 此處可自由選擇進入不影響劇情 ) 到鳴神神社，在鳴神神社裡唯一的鏡子下方出現靈動，拍下後出現痛苦的人們。

回程時在階段廊下那個目隱之靈第二度出現。也可以往二樓上去，還會拍到一個地縛靈漂浮在黑暗中的僧人。在荒廢部屋把蒼色刻石崁入門上，開啟大門進入寬敞的渡廊下，前方的轉角看到哥哥的背影。走了一段路後眼前的大門又需要刻石了，拍下靈動出現逢魔淵的畫面。前往逢魔淵，前一夜的斷橋現在竟然通了。走過後到前方小島，得到紫陽刻石後再走回去吧。第一夜那棵櫻樹下上吊的女人又出現了，她是誰呢？走幾步路，看到前方二樓月見台有個男人。回來的階梯上，地板角落出現了少女，很快就消失。到渡廊下拿出紫陽刻石開門吧。

開門後進入鬼口，一個很大的穿堂，走樓梯往下到達地下一階，地上除了鳥居外四邊還點了火把。往前調查唯一的大門得到一盒火柴。同時大門出現靈動，拍下出現後一個隱目面具。同時出現那個月見台出現過的男人。回頭時那個目隱之靈再次出現。原路走回荒廢部屋，打開往弔間的門眼前出現那個第一夜幫你的白色和服少女指著燭臺的方向。調查燭臺玩蠟燭遊戲，把所有蠟燭點亮即可，解開後就可以拿到鬼遊掛軸。目隱之靈第四次出現，弔間走道較窄要小心對付。拍死目隱之靈後往和人形部屋去。把掛軸吊在原來的牆壁吧，忽然一陣反應，這時第一夜那長的像一沱抹布的匍伏少女之靈再度出現。牆壁變成一扇秘門，推開後進入著物之間，那個跑的很快的小孩從衣服縫裡掠過。走回圍爐裡之間，看著牆壁旁邊時鐘顯示 11 時 45 分，這時候那個小孩忽然衝了出來。拍死他後，終於把第一夜的御神鏡碎片拿回來了。第一夜書房裡壞掉的樓梯竟然通了。往上走看到屏風旁那扇門有靈動，拍下出現繩之廊下的天花板照片。回玄關路上的繩之廊下發生靈動，往上一拍出現了俯視的怨念。拍了後解開二階的封印門。在玄關儲存點對面有靈動，拍下它出現怒的面具。回到圍爐裡之間上了二階，隔著窗戶裡面有一個人在彈琴，開啟門後進入上敷座。眼前又出現那個男人的影子，跟著進房間發現寶箱有靈動，拍下出現一把鑰匙。打開二階的門進入迴廊，目隱之靈再次出現，建議把它引進上敷座拍死她。迴廊的另一端門口出現靈動，拍下出現一樓迴廊的夾縫。從另一端下樓去，在一階迴廊牆壁旁的縫隙有靈動，拍下出現少女的封印靈，解開二階封印門。回到二階開門進入琴之部屋。

  • ( 此處可自由選擇進入不影響劇情 ) 到大廣間，如果第一夜來過這裡也撿了一堆東西卻相安無事，第二夜會有惡靈出現。浮遊女之靈或者是長髮女之靈，是我到第二夜以來認為最難對付的亂數靈，行蹤飄忽不定還會躲起來。到倉庫，本夜此處有靈石可撿，密碼是「 7925 」。

前往琴之部屋看到地上一堆散落照片，有位女子的日記散落桌上，她發現了相機的神奇功能。調查了一下房間得到青銅之鑰。留聲機忽然發出那位女子的聲音。離開回去上敷座用鑰匙把寶箱打開後得到一本書，是宗方良藏寫的，探討目隱儀式。此時宗方良藏再度出現，說完話後出來又看見他往月見台方向消失。打開大門道月見台看個究竟。看到前方櫻樹上弔死的那名女子越看越像媽媽。忽然感覺到像媽媽的女生，拿著相機到處拍照而且愛拍漂浮的小幽靈。回神一瞧樹下有一張照片，趕快去櫻之中庭撿。剛出二階房門又遇到哥哥那晚遇到的被束縛的男人。看到民俗學者看到像媽媽的女人的死狀很是難過，他們應該是夫妻吧。深紅叫了那名女子一聲媽媽，恍然驚醒變成厲鬼，拍死上吊女之靈後，出現五個娃娃的畫面。得到宗方八重遺書與兩張照片，一張是宗方八重與年紀小小的獨生女宗方美琴合照，另一張則是美琴與那個小霧繪合照。在和人形部屋右邊的供桌上有靈動，四個娃娃變五個。調查左手邊的娃娃出現五選一的畫面拿起娃娃，放到對面的供桌上掉出怒的面具。

帶著面具回玄關開門，把怒的面具掛在牆上推門進入。( 進入此處在沒拿到似姿面具前，必須都用對應的面具方能開門，所以在開門後記得把面具拿下帶走 ) 進入隱通路，在轉角處的屋頂有一靈動，很難拍的地縛靈。轉角另一端的門有靈動，拍下後出現了掛滿面具的柱子，前方另一扇門則可自由通到燈火廊下，把該扇門樂的面具取下。往通路裡端走，宗方良藏又出現了，往一扇門裡消失。開啟門後進入面之部屋，並把門上的哀之面具取下。在面之部屋照片裡的那根柱子出現了，不過並沒有掛滿面具。調查柱子上唯一的那扇面具 ( 如果你要把還沒湊齊的兩扇面具掛上也行 ) 感應到冰室家幾近恐怖的目隱儀式，少女就被目隱面具活活刺瞎。劇情完畢後可憐的目隱之靈第六次出現，這邊走道非常狹小要小心奮戰。拍死目隱之靈後，把牆上那面喜的面具取下，拿去開另一端出口。進入箱庭迴廊，迴廊中間的中庭終於有儲存點了。迴廊中有兩扇門，一扇可以回到燈火廊下，另一扇則有靈動，拍下後出現石柱。石像就在中庭裡，在石像上得到山吹刻石，帶著刻石去開門吧。進入佛堂非常的寬敞，四周有各缺四支的大型佛像。中間的供桌可得到似姿面具，接著民俗學者之靈第一次與你戰鬥。拍完後再回到面具部屋，現在你一共有喜怒哀樂再加似姿五扇面具了。把喜怒哀樂這四扇面具掛在柱子上便會聽到開門的聲音。帶著唯一的似姿面具離開面具部屋，宗方良藏再度出現指引你進入目隱部屋。現在目隱部屋門可以開了，進入舖著地毯又豪華的目隱部屋，祭壇上得到目隱面具，帶著面具去鬼口吧。在圍爐裡之間這邊會巧遇第一夜那個折首女之靈或是其他的亂數靈。在鬼口由於要往地下道走去，當下階梯到一半時民俗學者之靈再次出現。辛苦的奮鬥完畢後再往下走到最底層，調查底層大門旁的靈動掛鉤，目隱之靈七度出現，拍死她讓她永遠安息吧。把目隱面具掛上掛鉤，深紅會發現宗方良藏就是開啟這道門後被萬鬼齊發吸進去。這時大門打開，深紅冒著膽子繼續走進去禁忌的地下道。在地下道，宗方良藏的屍體上得到第二塊御神鏡碎片。跟第一夜一樣，霧繪姐再度引領禍刻靈們找上無助的深紅，又是一陣暈眩。
