主人公であるあなたがポケモンの世界へ迷い込み、突然ポケモンになってしまったところから冒険が始まります。

ゲームの目的は、パートナーのポケモンといっしょに「救助隊」を結成して、災害に苦しむポケモン達を助けること。ポケモン達は「不思議のダンジョン」の奥であなたの助けを待っています。「不思議のダンジョン」では、入るたびに地形が変わるためハラハラドキドキの冒険が展開されます。頼りになるのは、どんな場面でも切り抜けられる「あなたの知恵」です。また「青の救助隊」同士のプレイヤー同士が街ですれちがった際に行われる「すれちがい通信」で、冒険に役立つどうぐを入手することができます。「青の救助隊」プレイヤー同士の輪もきっと広がるはずです。

「不思議のダンジョン」が持つ奥の深い楽しさと、ポケモンになれちゃう（なっちゃった？）楽しさを味わえる夢のゲーム。ポケモンたちとのアツい感動の冒険物語で寒い冬もへっちゃらです。

※ご注意
「青の救助隊」と「赤の救助隊」のゲーム内容は、ほぼ同じです。
