一ノ刻 徴 攻略

展現在怜眼前的，是一個古老的宅子，她沒有選擇，只能向前走。來到囲爐裏の間二樓的時候，怜發現地上好似什麼東西。不知道是誰的手，正把一部相機推向怜的身邊。拿到相機之後，怜來到了二樓的回廊，在牆上的一個洞裏，她看到了一個正在拿著木槌敲打木釘的小女孩，這究竟有什麼意義呢？正當怜疑惑不解的時候，突然聽到了女子的哭聲，尋聲而去，看到一個紅衣的女子正坐在地上哭泣，怜用剛剛拿到的相機對著她拍照，卻得到了一張看起來很奇怪的照片…正當怜看著照片的時候，紅衣的女人仿佛被驚嚇到了一般，嘴裏喊著：「不要走」沖出了怜的視線。無可奈何的怜只好返回，結果發現來時的門，不知道被什麼力量給封住了，更糟糕的是，她的面前出現了一個來意不善的靈．彷徨う母娘。用手中的相機，怜還算輕鬆的解決掉了眼前的母女靈，可是她在這裏一刻也不想待了，就在走到玄關的時候，她看到了那個紅衣女子，正坐在地上，嘴裏不停的說著什麼。怜關切的想去安慰她，「沒關係…」，可是怜的手還沒有碰到她，她就像一隻受了驚嚇的鳥慌忙的爬到的另一端。或許是看到怜的臉上沒有惡意吧，紅衣的女子才稍稍的放鬆了一點，可是，在她的眼中又突然的出現了驚恐的神情，順著她的目光看去，怜也被眼前的情況驚呆了，一個滿身刺青的女子正緩緩的向她走來。紅衣的女子已經嚇得魂飛魄散，而怜也對此束手無策，就在刺青女子步步逼近的時候，怜感到自己幾乎不能呼吸了…不知道是哪裡來的力氣，她撥開了刺青女的手，拼命的向門口跑去，而刺青女正在後面不停的追逐著怜，還差一點點了，怜用盡全力的向前跑著，就在她即將沖出古宅的時候，刺青女的手在怜的右肩上劃了一下…「啊～」怜從睡夢中驚醒，看著眼前熟悉的擺設，這是自己的房間，她稍稍的舒緩了一口氣…還好只是個夢，她想著，但是，對於自己右肩突如其來的劇痛，怜忍不住呻吟了起來，而在她右肩，出現了難以置信的和刺青女一樣的青色紋身…它清楚的告訴怜：這一切不是夢…

怜走出了自己的房間，「怜，早上好」，客廳裏，怜的助手深紅在跟她打招呼。深紅遞給了怜一封來自螢的信。來到戀人優雨生前的房間，這裏的一切都還是維持著優雨生前的樣子。怜出神的看著電腦，這時優雨正坐在床邊對怜說話：「怜，怎麼啦？」「啊，對不起…」正當怜習慣性的回答的時候，突然感到有一絲不對勁，當她轉過頭看的時候，床上空無一人…可能是太緊張了吧…怜拉開窗簾，出神的看著外面，外面正下著不小的雨，就好似優雨出事的那天…那是她永遠不能忘記的畫面啊…在優雨的房間，怜找到了一些資料還有螢信中提到的照相機。照相機裏面有一些特別的照片，所以怜到自己的暗室把它們沖洗了出來。那是夢中的那個紅衣女人哭泣的樣子…怜覺得很是奇怪，於是把照片交給了深紅，希望她能幫忙調查一下。回到房間，怜感到自己有些累了，於是她躺下很快睡著了…可是，她又做了那個夢，於是她再度來到了這座古宅…

「又是這個夢…」怜喃喃自語到。沒有辦法，只能前進，玄關的走廊裏，她看到了那個紅衣女子跑了過去…怜也沒想太多，只能跟著追過去了，這裏到底是怎麼回事？或許那個紅衣女子會知道答案。穿過仕切られた部屋，追隨著紅衣的女人來到御簾のある座敷，拿到了這個女人的證件，可是因為不知道是不是火的原因，證件的一部分模糊不清了，大致能看出來這個女人叫瀧川吉…正當怜准備離開的時候，這個女人突然開口說話了：「如果你醒來了…拜托…請你喚醒我吧，我…」，怜在這個時候醒了過來。

身上的刺青又開始作痛了，不過每次都是顯現那麼一下就消失了。怜下樓接到了深紅的電話，說上次調查的照片有眉目了。這個女人名叫瀧川吉乃，是墜機事件唯一的倖存者，但是因為昏迷而一直在醫院裏。怜來到了醫院，她不敢相信眼前的一切，這個躺在病床上的女人，真是她夜夜夢中所見的人。更可怕的是，這個女人的身上乃至臉上，都出現了和她肩上一樣可怕的刺青紋身…怜被眼前的景像嚇得轉身就跑，誰知道那個女人的床邊傳來了很低沉的聲音，而當怜轉過身去的時候，竟然…床上除了一些褐色的灰燼，什麼都沒有…她失神的回到家中，在浴室裏，怜想到最近發生的事情，不禁難過起來…

在夢中，怜果然又回到了古宅。來到御簾のある座敷，怜發現榻榻米內個櫃子有些異樣，調查以後發現只是個普通的娃娃，怜稍稍安心了，誰知道突然發覺身邊有一個人，仔細一看，正是在醫院消失的瀧川吉乃，而更可怕的是，她身邊出現了三個黑影。怜毫不猶豫的拿起相機戰勝了三個靈，但是誰知道走向瀧川的時候，她竟然也化作怨靈攻擊怜。經過苦戰，怜終於打敗了眼前已經化作怨靈的她，而正準備離開的怜卻再度遭到瀧川的襲擊…正在這時，怜從夢中驚醒，身上的刺青也越來越深，越來越疼。來到客廳，從深紅處得到了瀧川的日記，從日記中，怜瞭解到，其實這個女人自從住進醫院也非常痛苦，還有，她也夢見了怜所夢見的一切…那麼說，她們兩個現實中的人在夢中相見了？

攻略流程：
  1. 直走後右拐 ( 轉角處的右邊有萬藥丸，初期還是帶上的好 )，進入囲爐裏の間。( 門口和進門都有浮游靈，以後除非需要觸發劇情的，一般不作提示了，反正一周目也全靈不能 )
  2. 來到二樓取得相機 ( 這裏有 SAVE 點 )
  3. 進入二樓的回廊，在牆上有洞的地方拍照，可以拍到兩個浮游靈
  4. 對著格子那邊的紅衣女人拍照 ( 找不到的看感靈燈的提示 )
  5. 原路返回，在出門時觸發劇情，強制戰鬥
  6. 在玄關的走廊遇到紅衣的女人
  7. 醒來回到現實
  8. 和深紅對話
  9. 調查優雨的房間
  10. 到暗室沖洗照片交給深紅
  11. 強制回到房間，可以通過相機 SAVE
  12. 調查床，可以選擇睡覺
  13. 經過仕切られた部屋，追隨紅衣女人來到二樓階段廊下
  14. 拍攝回廊的門得到封印照片
  15. 來到1樓階段廊下，拍得解封印的照片，同時取得裝備機能【報】
  16. 回到2樓回廊，進入御簾のある座敷，調查女人後離開
  17. 回到現實
  18. 下樓接到深紅的電話，觸發劇情
  19. 休息，進入夢中的古宅
  20. 回到御簾のある座敷，觸發劇情
  21. 強制戰，之後回到現實
  22. 從深紅處得到日記，第一刻結束

CHECK 小要訣：
首先是 SAVE 點有兩種，一種是夢境中的青綠色的燈，另一種是現實中的相機。建議初次玩的玩家多多記錄，反正也不會影響評價，養成記錄的好習慣吧。不過有兩種情況下是不能記錄的，一個遇到戰鬥的時候，另一個是有浮游靈出現的時候。所以有時候看到記錄點的燈不亮，就得找找原因哦。遇到箱子櫃子都多檢查一下，說不定會有意外的收穫哦～看到閃光點不要忘記調查，因為很多重要道具就是用閃光點提示你的。每次回到現實總會強制怜做一些走動，但是只要發現怜換上了睡衣，就可以選擇休息了，如果不能的話，遊戲會提示你還有什麼事情沒有做。多和深紅，螢對話，可以取得很多的資訊。遊戲中的怨靈分為一般戰和強制戰，對於自己的技術信心不足或者想要儘快通關的話，就可以避開一般的戰鬥，可以通關開門關門來躲避，除了個別的靈，一般都不會追趕的。

BOSS 戰：
第一個迷失的母女比較好對付，只要針對其母親下手就可以了。因為攻擊方式的單一，所以幾乎就是送上門的 FATAL FRAME，要好好的把握，相信能輕鬆戰勝。第二個紅衣女人比較難於對付，通常的方法是先不要靠近她，解決掉3個在天上漂浮的靈之後，再和她戰。因為她也是直線型的，幾乎不會瞬移，所以正好可以拿來練習 FATAL FRAME。以後的 BOSS 戰如果沒有特殊的說明，就是大同小異了。
