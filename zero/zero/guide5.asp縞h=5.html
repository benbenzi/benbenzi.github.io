出自 零～紅い蝶～
研究日本各地的風&&慣的民&學者。為了解出黃泉之門的傳&的真相不惜賭上性命，因助手宗方良藏帶來的皆神村的情報而決定前往皆神村。

真壁與宗方一起拜訪了皆神村，在村子裡受到非常隆重的歡&。得到了參觀祕祭的特別准許，而逗留在村中展開了各式各樣的&查。例如村子裡的遺跡和書籍，&查了有關祕祭的事情，逐漸到祕祭的全貌…。在完成那個目標之前，真壁就被監禁到黑澤家的座敷牢中了。自己了解了紅贅祭以及陰祭的事情，而領悟了自己將成為楔而被當作祭品。比起不想死的感情，能以這雙眼睛看見祕祭為名的歡喜更加的強烈，這樣的真壁，一點都不打算要從村子逃走。

禍刻 ( まがとき )

出自 零～ZERO～
御縛之儀式失敗而導致黃泉之門打開，瘴氣噴出的事情稱之為禍刻。最後的繩之巫女‧霧繪的時代，由於&用霧繪在還殘留著對現世的執著的狀態下製作的裂繩來進行儀式的關&引起了禍刻。沐浴在瘴氣之下的村人們紛紛得了瀕死的重症。而沐浴在瘴氣之下而失去正氣的冰室家當主，對沐浴在瘴氣中的人們擴大不可踏出屋子外面的禁則，將居住在屋子內的人全數殺死了。日後，這一天的慘劇就以當主瘋狂了而將全家都慘殺的事情流傳了下來。

慎村真澄 ( まきむらます& )

出自 零～紅い蝶～
為了建設水壩，為了&查實際地點而被派遣來的&查員。&查中進了森林並&路在皆神村中，為了找尋脫出的方法而在村子搜索著。與&查自己的戀人須堂美也子再會之後，卻在探索黑澤家的時候遇到楔而被殺。怨靈化的慎村往戀人所在的地點移動，並殺了美也子。

皆神村 ( &なか&むら )

出自 零～紅い蝶～
以『從地圖上消失的村子』而在附&成為十分有名的傳聞。澪與繭的母親聽說了這個村子的事情，所以經常要兩人注意別玩到晚上才回來。

以雙子為巫女並尊崇之，名為紅贅祭的不可思議的關於祕祭的神秘性，以及拜訪村子的客人行蹤不明之類的事，在當時被&鄰的人們所畏懼著。但是由於這個村子的儀式，也被稱為是背負著把此地從闇之恐怖之下救出來的擁有被詛咒的&命的村落。這個村子的傳說，是藉著在大償發生時稍前一點，由從村子裡出來的宗方良藏的手記的記載，在後世流傳下來。

目隠し鬼の儀式 ( めかくしおにのぎしき )

出自 零～ZERO～
選出繩之巫女的儀式。在一個屋子中集合冰室一族所有的七歲的少女，由臉被遮住的鬼 ( 戴上面具的關&眼睛被毀壞 ) 的少女來捉捕。最後一個逃過目隱鬼的少女 ( 越能從目隱鬼之下逃脫&擁有著越強的靈力 ) 就會被選為繩之巫女。另外在這個儀式中最初被抓到的少女，在下一次儀式的時候，擔當負責選出下一任巫女的目隱鬼的這個任務。

宗方良藏 ( むなかたりょうぞう )

出自 零～ZERO～及零～紅い蝶～
在『零～紅い蝶～』中，與民&學者‧真壁清次郎一起拜訪了皆神村。很久以前是立花樹月小時候的朋友，之後被委託負責雙子巫女的八重與紗重從村子裡逃跑之後的事情。幫助一個人從村子裡逃走的八重之後，兩人結婚。

無法&記皆神村的記憶的宗方最後惹上了黃泉之門，在『零～ZERO～』中移居到了冰室家居住。結局，妻子八重自殺，宗方在走到黃泉之門前面也死了。另外，兩人的女兒美琴，幼年時代從冰室家逃走，以失去記憶的狀態在山中被發現。美琴的女兒就是深雪，深雪的孩子，就是『零～ZERO～』中的主角深紅以及真冬。

黃泉の門 ( よ&のもん )

出自 零～ZERO～
隔絕現世與異界的門。存在於冰室宅的地下，以巨大的石頭製作而成。同時，什麼時候、誰做的、為了什麼而做，這一切的紀錄都沒有流傳下來。只知道，從古老的時代開始就存在於此地，成了現在這樣。冰室家中傳說著，若門完全打開的時候，生者與死者的世界將會相連，所有的死者都會復甦。

靈石收音機 ( 霊石ラジオ = れいいしらじお )
