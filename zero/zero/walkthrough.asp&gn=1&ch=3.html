第三夜 禍刻 攻略

深紅這一夜夢見了冰室家如何因為裂繩儀式失敗發生禍刻。醒來時手腳的繫繩跟第一夜一樣消失了。醒來時的地方是前一夜沒走完的禁忌的地下道。決定再進去一次，開門進入禁忌的地下道，紅色的牌樓柱子還是掩飾不了陰森的氣氛，在前一夜民俗學者陳屍之處發現靈動，拍下民俗學者宗方良藏之靈，他已經化身為地縛靈了。再往前走會出現死於瘴氣的人們惡靈。他們都是一次三個成群，一次拍多個可以提高分數。再往前走發生了地震，再往前走隔著地下道的縫隙看到哥哥就在另一端，你呼喊著他但是他卻沒發現你。回頭時那個小霧繪又出現了，她指者右邊的地上就消失。撿起地上的文件，文中提到月祠堂，到月祠堂去吧。出鬼口爬上階梯走回去，在渡廊下出現了若隱若現的臉之靈，遠看只是一坨霧氣，拍照後就化為一張臉隨即散掉。散掉時無法拍照，但聚集成一團後又可以拍了。出口的門被封印了，封印靈就在後方的瓦礫堆中。這裡又得到禍刻的文件。前方出現一個帶著面具的男人，跟他走，開門到櫻之中庭，一到中庭就雷電交加，中庭裡的櫻樹上拍到地縛靈學者之妻宗方八重。前方月祠堂大門出現靈動，拍下出現佛堂裡祭壇的畫面。又看到那個面具男往階段廊下走去，跟他走看見那面具男往鏡子旁那扇前兩夜都不能開的門進入，跟著他之後開門。

進入佛堂，前方神桌上撿起式繩，拿到兩邊的四尊大佛像玩式繩遊戲方法很簡單，試錯了再重試即可。成功後祭壇的門會打開拿到碧色刻石，這時候深紅看到一個神官喘噓噓逃到佛堂，被那個戴面具的男人冰室家當主追殺到此，揮劍一斬神官的身首分離。劇情結束後冰室家當主現身，由於對方帶著長劍所以千萬不能被他砍到，就算他被拍到，已經出手的攻擊還是會繼續。不過可以讓他的攻擊停頓一下，拍完後要快跑，否則還是會被他腰斬。同時他會遁地，如果相機沒反應可能代表已瞬移到後方，這時要胡亂跑。同時沒拍死他前門也打不開，所以別想要逃命。

拍死冰室當主後離開佛堂，回月祠堂 ( 此時無法原路回去，建議開啟大門後從燈火廊下回去 ) 在燈火廊下可能會遇到浮游僧人之靈亂數出現，會發射靈氣彈。這裡也有可能遇到亂數靈，並從這裡打開往弔間的門。

  • ( 此處可自由選擇進入不影響劇情 ) 可開門可進入倉庫，密碼是「 1213 」。在倉中有幾顆靈石可撿，也可以到樓上夾層拿東西，不過會出現折首女之靈。

使用碧色刻石開本遊戲最後一扇門，開門進入月祠堂。祠堂中間的柱子四面都有靈動，隨便先拍哪一面都行，我的方法如下：

  1. 第一面，拍下後出現箱庭迴廊的石碑照片，往那裡去。開門進入佛堂，開門進入箱庭迴廊，往前方的石碑走去，第一個神官出現。神官如果是無頭狀態，相機沒有靈動，神官的頭冒出來時，先拍照讓他停住，否則會飄忽不定。同時神官會遁地從腳底冒出來作近距離攻擊。如果他在遠處則會發動麻痺攻擊，被攻擊到一動都不能動，只能祈禱他不會趁機攻擊了，所以最好不要讓他有出手機會。拍死神官後，他的臉上會冒出梵文符號同時也拿到神官日記，記下來臉上符號原路回月祠堂。
  2. 到原來拍照處調查，按下剛才神官臉上出現的符號，再去拍柱子另一面的靈動，出現逢魔淵島上的鳥居照片。前往逢魔淵，經過水車時出現靈動，原來是助手平阪巴已化身為地縛靈，拍照後願她安息。往橋上的小島走去，調查鳥居會出現神官，神官如果是無頭狀態，相機沒有靈動，神官的頭冒出來時先拍照讓他停住，否則會飄忽不定。同時這裡的神官會發射靈氣彈，一樣拍照可化解攻擊，如果拍死他前沒化解掉靈氣彈，那挨一砲的代價可能不小。拍死神官，一樣得到神官日記，記下臉上符號回月祠堂去吧。
  3. 按下對應按鈕後，再拍另一面靈動，出現井戶裡庭的墓碑，前往井戶裡庭開啟往水道部屋的門，密碼是「 1347 」。如果前兩夜都沒拜訪過大廣間，在這裡會遇到漂浮的女人或是長髮女之靈。調查井戶裡庭的墓碑，出現跟逢魔淵一樣攻擊模式的神官。擊敗他後，記下臉上符號後可原路回月祠堂或繼續前往神社 ( 自由選擇不影響劇情 ) 在深森參道會遇到三個成群出現的死於瘴氣的人們亂數出現，到鳴神神社，屋頂會拍到地縛靈高峰準星，神社門口會得到靈石與九零式底片。到井戶裡庭這裡可能會遇到女人頭之靈或是折首女之靈。如果在深森林道沒遇到死於瘴氣的人們，那在水道部屋可能會遇到。
  4. 在月祠堂按下對應符號後，拍下最後一個靈動，出現櫻之中庭的石柱。前往櫻之中庭調查石柱後，最後一個神官出現，他的攻擊法與箱庭迴廊的那一個一樣。拍死後拿到神官日記與記下符號回月祠堂。

按下最後一格符號，柱子又發生靈動，拍照後出現當主之證。接著柱子底下冒出大片血跡一直通道門外，沿著血跡走一路走到了大廣間，一進大廣間，馬上就被大片血跡震撼。接著冰室家當主隨即二度出現。拍死當主後取得當主之證，同時大廣間的供桌上可拍到冰室家當主的地縛靈，所以他應該在這邊自殺吧。在這裡可得到御神水別忘了仔細檢查，此時可原路回月祠堂或去其他地方逛逛。在著物之間 ( 自由選擇不影響劇情 ) 打開櫃子拍下編輯者緒方浩二的地縛靈，同時此屋也有鏡石可撿。在目隱之間 ( 自由選擇不影響劇情 ) 這裡會拍到已化身為地縛靈的目隱之靈。把當主之證崁進月祠堂的柱子上，柱子整個移動，原先的柱子下方出現了大洞月之井戶。走下去月之井戶，地洞裡空間不大，前方的供桌有一顆骷髏頭 ( 應該是初代繩之巫女吧 )，調查得到第三塊御神鏡碎片，此時霧繪從一攤水裡冒了出來，深紅拔腿就逃。往原路想順著爬梯往上走，可是似乎走不完一樣，此時小霧繪出現伸出手來。
