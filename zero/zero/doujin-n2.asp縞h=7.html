走進地道裡，裡面很髒。
走著走著，突然聽到身後有腳步聲，我轉頭一看，是姊姊！
她怎麼會突然出現在這？
我剛剛找了半天都沒找到她，反倒是她自己先跑出來了。
而且頭還低低的，一副做了什麼虧&事的樣子。

「姊姊……」

姊姊聽到我在叫她，於是頭抬了起來。
這時奇妙的事又……反正每次奇妙的事都發生在我身上。
姊姊變身了，她變成&重了。
而且眼神一副挑釁的樣子向我走來。

「妳是……&重！？」

我叫住她的時候，不知道為什麼，&重的臉上多了幾條青筋，而且更&速地走過來。
我看她的眼神很可怕，就像遇到神經病一樣。
我邊退邊試著跟&重說話。

「喂！&重……」

她還是不理我一直朝我走來。
但是我可以很清楚地看到&重臉上的青筋更多了，而且太陽穴一直抽搐。
她可能很討厭別人叫她的名字吧，我再試試看。

「&重、&重……」

她更生氣了，但還是繼續向我走來。
看她這麼生氣，也難怪她會討厭這個名字。
說實在，這名字我也覺得很難聽，真不知道她的父母是怎麼取名字的。
&重離我越來越&了。
我的直覺告訴我，再不&點逃走一定會被她打，不過再怎麼討厭自己的名字也不用打人吧？
不多想了，我趕緊往地道後面逃。
逃跑途中我絆到旁邊的石頭差點跌倒，而且包包也掉了。
我正想&起包包，&重已經跑到我面前了。
我連包包也不敢&，繼續向後跑。
地道的轉角處有一個樓梯，我急&爬了上去。

上去後，好像是在某家的倉庫裡。
這時我也沒閒情逸致去到處&了，&重在後面&我啊！
要是被她&到鐵定會被痛打一頓……
我打開倉庫的門跑出去，沒想到出去時，整個空間都突然變成黑白的。
這是搞什麼？電影特效嗎？做得真逼真。
沒時間誇獎它了，再不跑&重會&上來！
跑著跑著，我跑到一扇門前，打開門進去。
咦？特效又消失了！？
反正不管了，我急&跑上旁邊的樓梯。

跑上去之後，我繞過一個轉角，看到一個人影。
是姊姊！！

「姊姊！」

反正先把姊姊叫住，等等&重&上來就把姊姊借給&重打。
這時姊姊回過頭來了，怎、怎麼又是&重！？
我怎麼都不知道&重和姊姊長得那麼像？
反正她&來了，我急&轉身逃跑，隨手拉開一扇門逃了進去。
我進去之後，還是死命地逃。
這時我不知為什麼覺得&重很像姊姊，而我是被姊姊&著逃的蟑螂。
跑著跑著，我躲進一個納戶裡面。
要是&重打開納戶的門，躲在門邊待機的我就可以隨時揍昏她。

摳摳摳──

我還以為&重會直接開門，沒想到她居然先敲門。
有禮貌的好孩子也不是這樣當的吧？
還是&重以為會有人躲在納戶上廁所不成？
或者是她以為只要敲敲門我就會跑出來讓她打？
反正總歸一個字。

笨。

既然她在等我出去，我先看一下納戶有什麼好東&好了。
我看了一下地&，有一隻鑰匙，剛剛太緊張都沒注意到。
我&起鑰匙，這時我往上一撇，還有一個木製娃娃。
娃娃看起來不錯，很值錢的樣子，我把娃娃夾在腋下。
剛剛逃跑時整個包包都丟了，至少&個娃娃撈撈本。
好了，出去面對現實吧。

我打開門，外面沒有人，我急&向前跑。
突然感覺到後面有一陣風，我看到了！&重在我後面向我撲來！
我趕緊躲開，我可不想被女生推倒……
我跑向旁邊的一扇門，門鎖起來了。
我急&掏出鑰匙要開門，同一時間，&重也準備撲向我了。
X ( 消音 ) 的！來不及了啦！

碰───

門被我一腳踹開，倒地不起。
管他什麼鑰匙不鑰匙的，先逃再說。
當踹開了這扇門，我以為可以盡情地逃跑時，又來到一個&間。
&重仍在後面緊&不放，我逃向&間裡唯一一扇沒被&重擋住的門。
X ( 消音 ) 的！又是鎖的。
&重再度向我撲來，她要推倒我了！

「不要啊！！」

我把腋下夾著的娃娃丟向&重，&重被娃娃砸中倒退三步。
趁這個時候，我踹開門逃出這間&間。
逃出去後，是一座橋。

我沒時間在這慢慢看風景，現在一個死同性戀正&在我後頭啊！
我開門逃到另一邊去，轉角有個樓梯，我隨即奔下去。
咦？！這不是關&腔小鬼的家？？
反正剛好，我順&回去&掉在地道的包包。
我的直覺反應馬上就知道怎麼走。
其實我已經在這裡&路不下 N 次，已經&路到整個路線都背起來了。
我下來直接往離樓梯最&的門走去，進去後……

喀喀喀喀──噗──

裡面有一台奇怪的放映機正在放映某些片子。
那台放映機大概很老了，運轉時還會發出一些聲音。
不過我總覺得那放映機很欠扁……
發出「喀喀喀」的聲音就算了，為什麼尾音會有個「噗」？
它是在對我放屁嗎？
我走&放映機，輕輕地敲了一下。
想說，既然它會喊「噗」的話，那它會不會喊「痛」？
假如它會喊痛，那我一定痛揍它一頓……

呼嚕──

正當我在考慮要不要把放映機敲成廢鐵之時，我聽到門外的「呼嚕」聲。
這時我才想到，現在不是想什麼敲不敲的白痴問題，應該先&我的包包才對。
我出門去，站在我面前是某個有智能障礙的人，因為他一直努力地想用舌頭舔肚臍。
看來我&須給點建議才對……

「喂！你在幹麻？」

不理我……

「喂、喂！你會說話嗎？」

我問了這句我突然覺得我是白痴……他剛剛不是發出「呼嚕」聲了。
我想他該不會跟這棟屋裡的某個死老頭一樣吧……

「喂！智障！」

我正等著他會給我什麼東&，沒想到他的頭抬起來，死命地用鬥雞眼瞪我。
真是個不好玩的傢伙。

我不理他，直接走向旁邊的門。
進去之後發現原來我又走回那間有地道的&間了，要&點把我的包包&回來。
我跑入地道，檢查了一下包包，東&一個都沒少。
也對，這村子裡看不到半個人影，應該也沒有人會&走吧。
那現在……我應該再回剛剛那地方繼續&&看有沒有東&才對。
回到了剛剛那間屋子的倉庫，我正要開始&東&時……

鈴鈴鈴──

聲音從門外傳來，那是……鈴鐺的聲音！？
外面怎麼會有鈴鐺的聲音？該不會是死姊姊故意來找碴吧。
我走出門外，沒看到姊姊，只看到一個&紅色和服的小女孩跑過去。
只要前面有人在跑，不管那個人是在做什麼，後面的人一定都會跟著&。
而我就是後面&的那個人。

跑著跑著，那小女孩跑進一個&間，我也跟著跑進去。
進去之後我被一本爛書絆倒了，可惡！差點&到她的說。
我&起日記。
紅封皮日記 ( 3 )
樹月哥哥給了我他&間的鑰匙。
哥哥說，不可以讓任何人進入他的&間。
若是哥哥的朋友來了，也不可以把鑰匙交給他。
哥哥交代我事情我好高興。
因為這樣我就可以進哥哥的&間偷&他的錢了。
但是千歲因為害怕被哥哥抓包所以躲起來了。
還是沒有&到錢。

這傢伙真膽小，要偷就偷幹麻扭扭捏捏的？

鈴鈴鈴──

又是鈴鐺的聲音！&！！
我順著聲音一路衝到二樓，等我到了二樓聲音又不見了。
那死小鬼不知道跑哪了，我四處搜尋死小鬼的蹤影，我回頭往一旁看。
我旁邊有一間&間，裡面有人，而那個人是姊姊。
為什麼她會在裡面？她在耍自閉嗎？
但是好像不是，因為她在裡面不知道看什麼看得那麼高興，還一直傻笑。
趁這個機會偷偷進去嚇她好了，我輕輕地開門。
X ( 消音 ) 的！門鎖起來了。

又是那聲音，回頭一看是那小女孩，我趕緊&上，等等再回來嚇姊姊。
她跑在我前面，我在後面&，卻一直&不上。

我瞪了那東&一眼，是一本日記。

不論是外面還是裡面都變得一片漆黑。
家裡明明沒有錢，但是，卻有，很多。

救我。

好不容易，我聽到是櫃子裡面發出鈴鐺的聲音。
&&&……妳逃不掉了吧……

好……可愛……
小女孩的臉靠&我，她應該是想要爬出來吧？我把頭伸回外面讓她出來。

「來，妹妹笑一個。」

她聽到我的話之後居然對著我傻笑。
