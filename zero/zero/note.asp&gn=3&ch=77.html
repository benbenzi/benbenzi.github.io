「眠り巫女」について ( 有關「眠之巫女」錄音帶 )

麻生優雨
入手時間：十一ノ刻
入手場所：黒澤邸．二階．優雨の部屋
備考：--
這首歌似乎是東北流傳的「眠之巫女」的搖籃曲原曲，從該地的大學殘留的蓄音機裡找到的，透過編輯部拿到。

聽過之後，第一節和第二節都和現在流傳的「眠之巫女」有一點點不同，但我想可能是流傳時地方上的方言的不同。

除此之外還有「第三節」的非常大的不同。

雖無法詳細調查與解釋，第二節之後的第三節，和這首歌的印象有點不一樣。以下，寫出聽到的歌詞。

「眠之巫女 第三節」
　ゆきなさよ　はたて　　　　　　　　( 向涯去吧 )
　ゆきなさよ　はたて　　　　　　　　( 向涯去吧 )
　ゆきぶね　ゆらして　はたて　　　　( 雪舟 搖阿搖 向涯 )
　このきし　ひらいて　はたて　　　　( 此岸 開啟了 向涯 )
　ろうろう　みわたり　かのきしに　　( 隨浪 渡過 彼之岸 )
　しせい　わたして　なくが　てあげ　( 刺青渡過 哭泣著 道別 )

第一節和第二節有點像「威脅的曲調」的恐怖歌詞，
第三節有點寂寞、悲傷的曲調的歌。

特別是歌曲的結束部份，與其說是搖籃曲不如說是祭弔的曲的感覺。
