遊戲設定之零系列用語錄 - 字首為 か

陰祭 ( かげまつり )

出自 零～紅い蝶～
在紅贅祭開始以前，當虛開始鳴動的時候所進行的儀式，可以暫時性的鎮壓住虛。陰祭需要將人作成人柱而成為『楔』以用來進行儀式的關係，會將拜訪村莊的客人 ( マレびと ) 用來做為祭品。承受的痛苦若是越多，作為人柱的效果也會越強，才能夠止住虛的鳴動。

形見の射影機 ( かたみのしゃえいき )

出自 零～ZERO～
雛咲深雪 ( 深紅的母親 ) 遺留給深紅的照相機，是一台可以拍攝到人所看不見的東西的照相機，被稱為『射影機』。這個照相機，若是交給擁有強大靈力的人來使用的話，能夠做到封印靈體。但是，隨著使用射影機拍攝靈體的次數的增加，也可能造成最壞的場合──被靈的世界給拉了進去，也存在著精神異常的可能性。深雪也曾經手持這個照相機，而在心中留下了心病。

鬼隻 ( きせき )

出自 零～紅い蝶～
在皆神村中，紅贅祭裡當雙子巫女 ( 御子 ) 的姐姐 ( 哥哥 ) 將妹妹 ( 弟弟 ) 殺死之後，留下來的姐姐 ( 哥哥 ) 就被稱之為『鬼隻』，為眾人所懼怕且尊敬。成為鬼隻者，精神異常的人非常多，這些人被隔離到村子裡的藏。鬼隻們死後，被埋葬在朽木裡。

霧繪 ( きりえ )

出自 零～ZERO～
冰室家傳承的繩裂之儀式的最後一任巫女。由於有了思念的人，殘留著對現世的執著而沒有成功製作繩裂，御縛之儀失敗了。再者，由於真冬與她所思念的人十分相似，所以打算將真冬拉入闇之世界的這一邊。

桐生茜 ( きりゅうあかね )

出自 零～紅い蝶～
在紗重與八重之前的雙子巫女中的姐姐，親手殺了妹妹薊而成為了鬼隻。儀式之後，對於殺死自己妹妹的事情非常的自責而在精神上呈現病態，父親為了使茜逐漸恢復而製作了與薊十分相似的人偶。但是寄宿在人偶的魂不知何時操縱了茜。死後的茜也依然繼續在人偶的支配之下。

桐生薊 ( きりゅうあざみ )

出自 零～紅い蝶～
做為過去的紅贅祭中的雙子巫女的妹妹，由姐姐茜所殺死。仰慕姐姐，並且相信如此做就可以合為一體的這點想法，與繭有著共通點。死後，由於恐懼父親所做的與自己相似的人偶帶來的災難，逐以靈體的形式出現來警告。

楔 ( くさび )

出自 零～紅い蝶～
紅贅祭開始進行之前，虛開始鳴動的時候，為能暫時鎮壓住那股鳴動而使用的祭品。能夠成為楔的，必須是來自村外的人。皆神村會將拜訪而來的客人監禁，並為了進行陰祭而切割其身體，進行『削身』。在客人之中，也存在著在成為楔之前無法忍耐住這些痛苦，無法成為楔並且死去的人。

朽木 ( くちき )

出自 零～紅い蝶～
傳說中是好幾個世代以前的紅贅祭的祭主，認為已死的鬼隻非常的可悲，為了鎮壓她們的靈魂而製作的祠堂。但是實際上，從前曾經有雙子為了逃出村子而使用了在暮羽神社底下的深道，被激怒的當時的祭主建造了這個祠堂，封閉村子裡對外相通的深道。朽木裡面，供奉著非常多的雙子地藏以及紅色的風車。

黑澤紗重 ( くろさわさえ )

出自 零～紅い蝶～
大償發生的當時的雙子巫女中的妹妹。由於姐姐八重成功的從皆神村逃走，一個人進行了紅贅祭失敗而引起了大償。想要拯救村子以及被姐姐背叛的怨念，抱著相反的兩個想法化為了怨靈。成為怨靈的紗重一個一個的將村人殺死，在血流成的海上狀似瘋狂地笑著。

黑澤八重 ( くろさわやえ )

出自 零～ZERO～及零～紅い蝶～
大償發生的當時的雙子巫女中的姐姐。與妹妹紗重一起試圖從皆神村逃走的，紗重卻在途中被村人們抓到，而只有八重成功的逃走了。途中為了搜尋不知去向的妹妹而重返皆神村，但回到村子所在地，這裡已經沒有皆神村的存在，已經是皆神村被大償消滅之後了。失意的八重後方，為了完成與樹月的約定而再度回來造訪村子的宗方良藏到來。在這之後兩人結婚，並移居冰室宅。良藏被黃泉之門所魅惑，而八重由於『射影機』的緣故精神不正常，最後在庭中的櫻花樹上吊自殺了。
( 註：皆神村在大償之後就被消滅了，這裡應該是指被暗闇所覆蓋，造成鬼打牆的現象，而使八重看不見皆神村了。)

黑澤良寬 ( くろさわりょうかん )

出自 零～紅い蝶～
大償發生之時的當任祭主，也是當年的雙子巫女‧八重與紗重的父親。良寬並不是沒有父親對於孩子的愛情，但身為祭主，『為了抑制虛而舉行的儀式不可不成功』的責任感更為強烈。另外，良寬自己也是過去的紅贅祭中殺死自己弟弟的鬼隻。

御神鏡 ( ごしんきょう )

出自 零～ZERO～
在黃泉之門前面放置的映出門的樣子的鏡子，並從這裡將散發出來的瘴氣彈回去，可以做到使門的力量減弱的特殊的鏡子。對於封印黃泉之門這件事擔任著非常重要的角色，但是，由於禍刻的衝擊而使它碎裂成五片，這些碎片散落在冰室宅各地。如果不收集五片碎片並且將其放回鏡子原本的位置拼起來的話，就無法封印黃泉之門。
