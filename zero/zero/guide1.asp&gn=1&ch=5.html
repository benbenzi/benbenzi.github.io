夢と鎮魂に関する調查報告 ( 夢與鎮魂相關調查報告 ) 第五章

麻生優雨

尋求「沉眠之家」

　總括至今為止之麻生優雨氏的調查，以及整理柏木秋人氏的殘留手記與文獻、乙月要的聽取紀錄、麻生邦彥博士的研究，另外還有從古至今的怪奇誌與東北各地殘存的各式文獻．資料等，我認為在東北被傳說的幽靈屋敷「沉眠之家」，是久世之宮的一部份之可能性相當高。

　姑且不論像麻生博士這樣，原本就在收集超常現象問題的人。以柏木秋人為首的當時民俗學者之研究，或是在現代民俗學．文化人類學的領域裡，完全沒有說到關於「沉眠之家」的夢。儘管至今所見都與現實中相當特異的風習間有共通點，但只被認為是妄言之類，而被人們忽視。的確，因為科學觀點無法解釋原因的病理領域，而將它與特定的地區可見的因習聯想在一起，會被認為是極奇怪的事吧。

　但這許多人們所夢見的「夢之病」、「沉眠之家」的夢是實際存在的地方。這是因為我天倉螢本身也實際體驗這個夢，且這些體驗．資料及其他証言等，都和以該屋子為舞台所引起的悲劇有非常多酷似的要點。

麻生博士的研究

　手邊有兩個麻生博士留下的研究成果。那是稱為「射影機」的「可照出異界的相機」，以及稱為「靈石收音機」的「可捕捉到異界的音波之礦石收音機」。

　這些是麻生博士的子孫，已故之麻生優雨氏所收藏之物品，對這次的研究及我自身體驗的「夢之病」．「沉眠之家」的研究擁有重大的意義。
　雖然現在「射影機」已經壞了不能使用，但實際在其中遺留的底片，與夢裡見到的「沉眠之家」的影像連結，也藉此知道其他被「夢之病」與「沉眠之家」囚禁的人物存在。
　此外，「靈石收音機」是以「聽見異界的聲音」為概念而被開發出來，特殊的礦石收音機。有關於這收音機，同樣有麻生博士的研究資料被留下，從應該是前述之乙月要留下的耳飾檢視出不可思議的聲音。給實際在夢中徬徨的我們不小的影響。

　根據麻生博士留下的乙月要之聽取紀錄，他夢見身為巫女的他所想念之人久世零華的夢與聲音。那夢，說不定是因為這耳飾而讓他看見的吧。

持續沉眠的人們

　從「沉眠之家」的夢裡醒來之後，我再次重新調查於各地被稱為「夢之病」「沉眠之家」的奇病紀錄。
　這裡所記述之患者們的談話，果然與我自身的記憶、夢一致。
　夢見沉眠之家的起因，親近之人的死，迷失於夢中的日式房屋中，這樣的証言反覆著，逐漸落入沉眠，最後失蹤。
　包含我與數名人物自這夢中逃出，是因為手邊有前述之麻生博士遺留的兩樣「異界」研究成果之道具，另外，也是偶然的和擁有同樣症狀的人物一起行動所致。
　在我們的夢中，久世家之悲劇迎向終點，零華應已自想念之中解放。但今後是否不會再發生同樣的病症，沒有人可以保障。

「眠之巫女」的意思

　現在我手邊有從某大學的研究室裡留下的蓄音機裡被發現的，應是錄著當時「眠之巫女」歌謠的錄音帶。關於第三節，有以下的整理連結。

　眠之巫女　第三節？
　　ゆきなさよ　はたて
　　ゆきなさよ　はたて
　　ゆきぶね　ゆらして　はたて
　　このきし　ひらいて　はたて
　　ろうろう　みわたり　かのきしに
　　しせい　わたして　なくが　てあげ

　於柏木秋人未正式公開發表的文書中，對這最終節作出以下的解讀。

　　刻上「いちわら ( 華服 ) 」的巫女，「去吧」和送往「涯」。

　　　～中略～

　　「しせいわたして　なくがてあげ」之意，「しせい」從第一節的「刻」來看，應為「刺青」之意，渡越它，「なく」為哭泣，「てあげ」是這個地方，揮手、有最後的道別之意的話語。

　　這最後一節，說不定是將刻著「華服」的巫女送往「彼之岸」時的所唱的吧。

　　( 擷取自柏木秋人氏手記，應是「歌謠與傳說」之原本 )

　這一節最初被認為是前二說的延展「指儀式的最後之部分」。但實際的儀式裡，巫女既已被咎穿，「送往彼之岸」便不太可能是儀式的一部份。

　於此，我對這節歌謠做出應為引起「破戒」後的對應處理之解讀。
　換句話說，我解釋為這應是為刻著刺青而自夢中醒來之巫女「鎮魂」的一節歌謠。

　以我為首的數名人物之夢的結尾，的確如這歌謠所述一般。

關於調查之繼續

　這個報告，說不定會被當成荒誕不稽玄妙類的怪奇文書吧。但即使被當成這樣，也可以以某種形式殘留下去的話，若再有被引入這「夢」的人們出現時，說不定可以成為拯救他們的助力。

　我今後也會繼續追查這事件及症例，打算報告更正確的結果。

　最後，感謝蒐集這許多貴重文獻並遺留下來之我的摯友，麻生優雨。

　託他之庇祐，這報告至此暫且結束。
