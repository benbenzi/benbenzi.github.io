遊戲設定之零系列用語錄 - 字首為 ひ、ふ

雛咲真冬 ( ひなさきまふゆ )

出自 零～ZERO～
深紅的兄長。是深紅唯一能敞開心扉的存在。與在雜誌工作的編輯因公事而相識，為了追查作家高峰等人的行蹤而前往了冰室宅。

雛咲深紅 ( ひなさきみく )

出自 零～ZERO～
是『零～ZERO～』 中的主角，靈能力強，具有能夠感知不屬於人世間的東西的能力。為了找尋行蹤不明的兄長而造訪了冰室宅。

雛咲深雪 ( ひなさきみゆき )

出自 零～ZERO～
身為深紅的母親，並且是宗方良藏‧八重兩人的孫女。因為深紅的祖母‧美琴所持有的射影機的關係而得了精神病，最後上吊自殺了。

氷室一族 ( ひむろいちぞく )

出自 零～ZERO～
從古久之前就統治這個地方的一族。以身為黃泉之門的守人的身分守護著黃泉之門，背負著封印黃泉之門的責任。這個地方的人們，畏懼並崇敬黃泉之門。

平坂巴 ( ひらさかともえ )

出自 零～ZERO～
高峰的助手兼秘書。擁有微微的靈感，感覺到瀰漫在冰室宅的異樣氣氛，前往取材之前感覺到了不安。與高峰同樣，被霧繪所殺死。

深道 ( ふかみち )

出自 零～紅い蝶～
存在於皆神村地下非常寬廣且巨大的洞窟，在最深處的是，進行紅贅祭的場所。已知是聯繫著黑澤家、立花家、桐生家等等的連結的地下道。另外，暮羽神社底下也擁有村子裡唯一的通到外面的深道通路。

雙子地蔵 ( ふたごじぞう )

出自 零～紅い蝶～
為了供養在紅贅祭中犧牲的雙子們的目的而製作的地藏。皆神村的各種地方都能看見，也有分辨村子的地界線之用。

雙子巫女 ( 御子 ) ( ふたごみこ )

出自 零～紅い蝶～
在紅贅祭中作為祭品的雙子。皆神村，有許多雙子誕生，那些雙子負擔了擔任雙子巫女 ( 御子 ) 的任務。小孩子是女性的場合被稱為巫女，男性的場合則稱之為御子。

皆神村的習慣上，將先出生者當作妹妹 ( 弟弟 )，後出生者當作姐姐 ( 哥哥 )，這點和現代是完全相反的。沿用皆神村的習慣來重新探討澪與繭的關係，澪是姐姐、繭是妹妹的狀況之下，結局的光景簡直是代表著重現儀式的意思。
